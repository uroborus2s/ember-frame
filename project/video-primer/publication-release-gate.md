# 商业出版准入门禁

这份教程不能只按“章节已写完”判断是否可发行。商业出版必须按证据放行：读者付费后，应该能看到真实跑通过的结果、失败样例、参数记录、项目交付包和跨平台边界。

## 放行结论

当前状态：不允许标为商业发行版。

可用定位：

- 内测版
- 公开修订稿
- 待实测教材

阻断原因：

- 本仓库尚未包含真实生成视频、关键帧截图、耗时、显存峰值和评分记录。
- 本机当前未发现 Wan2.2 `.safetensors` 权重，无法补第 5 章最小生成闭环。
- 第 26-29 章仍缺填好的示范项目包和最终交付证据。
- Windows / Linux 尚缺实测路径、截图和版本记录。

## 真实证据包目录建议

不要把学习者作业写进课程源码目录。课程作者用于出版验证的证据包可以放在单独目录：

```text
evidence/
  chapter-05-first-run/
    inputs/
    outputs/
    screenshots/
    run-log.md
    failure-log.md
  chapter-07-13-core-workflows/
    t2v/
    i2v/
    ti2v/
    flf2v/
  project-26-product-ad/
    client-brief.md
    asset-rights.md
    shot-list.md
    run-log.md
    score-sheet.md
    revision-log.md
    delivery-checklist.md
    qc-report.md
```

`evidence/` 中的轻量文本记录可以进入 Git。视频、截图、客户素材、音乐和模型权重必须先确认版权、文件体积和隐私边界；进入 Git 的正文只能引用已经确认可公开的截图、表格和压缩样片。
