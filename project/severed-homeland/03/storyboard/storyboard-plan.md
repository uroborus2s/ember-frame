# 第03集 Storyboard Plan：半枚驿令

## 分镜总锁

- 画幅：9:16 竖屏短剧；人物尽量贴中轴，路线信息放在下三分、墙脚线或右上/左下方向。
- 方向：北线固定偏画面右侧或上沿；南线固定偏画面左侧或下沿。反打前必须用刻痕、手指、箭尖、脚步或视线校准。
- 视觉编码：金赤只属于旧驿路线、半枚日月驿令、灵翼残纹；灰白只属于清明香；虫蜡白只属于血牒白蜡虫纹；黑绿只属于追兵局部甲壳、刃肢和火星。
- 连续性：沈维桑左臂全程不可用；所有动作由右手、右肩、右脚或牙齿完成。晏南枝血牒始终隐蔽，不正面展开成圣物。
- 标记说明：[KEY-PANEL] 重点板；[ART-REF] 需要 art-room 资产参考；[LOCK-FRAME] 需要锁定透视/连续性帧。
