进入废驿暗道后，路线冲突明确分裂：沈维桑依据沈照眠烧焦发绳、木车车轴印、北线旧印追北；晏南枝依据血牒、隐歌谷认印石和清明香记味风险要求南下。两人的争执不是单纯南北口角，而是两种救法冲突：沈维桑要先追活人车辙，晏南枝要保住能把旧名叫回来的证据。

清明香逼出油纸血牒边缘白蜡状虫纹，说明追兵能够记味追踪血牒。晏南枝只透露血牒与旧名证据、隐歌谷认印有关，没有解释血牒来源、自己身世或复国真相。

旧驿测绘廊中，沈维桑先依靠猎户观察水痕、灰尘、风、石栓磨痕判断危险，再出现眼角金赤灵翼残纹短闪。残纹只回应旧驿坏掉、断开的残响，不负责找路，不提示敌意，不替代观察。

塌门断后靠空间与物理因果完成：晏南枝先过窄口撬下栓，沈维桑用半成品明劲只松动下栓半寸，旧石栓结构失衡使半塌石门向追兵廊斜落。追兵被门隔在外侧，沈维桑代价为鼻血、左肩出血扩大、咳血跪倒。

结尾，旧驿图室中南线先亮到隐歌后暗下，北线穿过金河磨盘形小印并继续指向北境边墙。晏南枝受到冲击，因为她所受教育认为“月下在南，不在墙后”。沈维桑决定“先金河”，目标从北坡暗道转入第04集金河粮磨区。
