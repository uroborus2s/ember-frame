# 第03集 Evolution Notes

> 本文只是 learning-evolution-agent 基于第03集产物整理的演进提案。未经用户明确批准，不得修改 writer-room skill 本体、角色卡、评分 rubric、连续性门禁或模板文件。

## 本集可复用经验

- 前5秒奇观+危险同框最有效：半枚驿令点亮石门不是单独展示设定，而是与清明香、螳刃逼近、晏南枝催促放在同一画面，观众同时理解“有奇观”和“不开门会死”。
- 抽象目标需要可见物证绑定亲人风险：晏南枝保血牒若只停留在隐歌谷、盟誓、旧印会显得抽象；加入白蜡虫纹、白册记味、旧名证据，并明确可能影响沈照眠，才与救妹妹线同等有重量。
- 路线争执应写成两种救法冲突：沈维桑追车辙救活人，晏南枝保血牒保旧名证据。冲突不是南北口角，而是两种风险管理方式互相抢时间。
- 能力边界先观察再共鸣：沈维桑先看水、灰、风、磨痕，再出现灵翼残纹回应旧驿断裂残响。能力只补一个窄信息，不找路、不预警敌意、不替代判断。
- 低魔动作靠机械因果成立：塌门主因是旧石栓结构、门倾斜方向和晏南枝短刀撬动；沈维桑半成品明劲只松下栓半寸，并付出鼻血、肩血、咳血代价。
- 复杂空间动作先建门线/窄口/敌我位置：检修缝、断栓岔路、塌门石栓之所以可拍，是因为先写清谁先过、追兵卡在哪里、门往哪边落、主角站在门线哪侧。
- 下一集接口必须包含行动地点/身体限制/追踪物证/未解释悬念：第03集结尾给出金河粮磨区、左臂失力和止血需求、清明香/车辙追踪、月下在南却指向墙后的悬念，避免第04集重新发明动机。

## 建议加入未来角色卡的检查项

- 主角亲人线：每集至少保留一个可拍、可触摸、可被夺走或污染的亲人风险物证，例如发绳、车辙、旧名证据、药牌或识别者线索。
- 秘密盟友线：角色若坚持抽象目标，必须用一个可见物证说明“不这么做会伤到谁”，避免被写成只护纸、不顾人的冷血立场。
- 能力使用线：拥有特殊感知或残纹能力的角色卡应写明使用顺序为“观察现实痕迹 -> 触发有限共鸣 -> 付出身体代价”，禁止直接把能力写成万能导航或危险雷达。
- 伤势继承线：角色卡每集启动时应列出上一集遗留的身体限制，并要求至少影响一次行动、伪装、战斗或逃脱。
- 保密角色线：晏南枝这类角色只能释放推动当集行动的半截信息，不提前解释完整身世、血牒来源、复国真相或制度大谜团。

## 建议加入评分/连续性门禁的检查项

- Hook 门禁：前5秒必须同时出现“新奇观/新事件”和“即时危险/倒计时”，二者要在同一画面或同一连续动作里成立。
- Motivation 门禁：任何抽象目标都必须绑定可见物证，并说明该物证如何影响亲人风险或当集生死结果。
- Ability Boundary 门禁：特殊能力必须先观察再共鸣；若能力直接找路、预警敌意、解决机关或替代角色判断，应判为连续性风险。
- Spatial Filmability 门禁：复杂空间动作必须先建立门线、窄口、敌我位置和力的方向，再写动作结果；否则即使情节正确也应扣可拍性分。
- Low Magic 门禁：超常力量只能提供小幅物理增量，主要结果必须来自结构、工具、地形或协作，并保留明确代价。
- Exposition 门禁：大谜团只能释放能推动当集行动的危险线索，不用主题性解释提前回答血牒、身世、盟书或复国真相。
- Next Episode Interface 门禁：每集结尾交给下一集时，必须同时包含行动地点、身体限制、追踪物证、未解释悬念；缺一项应作为 continuity warning。

## 不要改动的规则

- 不要自动把本文件内容写入 writer-room skill 本体。这些只是提案，未经用户批准不得修改 writer-room skill 本体、agent task cards、rubric、模板或 schema。
- 不要取消“低魔边界”：半枚驿令不是万能钥匙，灵翼残纹不找路、不预警敌意、不替代观察。
- 不要把晏南枝写成单纯护纸的人。她保血牒的价值必须与旧名证据、沈照眠风险或制度追踪风险绑定。
- 不要让沈维桑伤势在下一集无成本恢复。左臂失力、肩伤、不能满弓、咳血必须继续影响动作。
- 不要在第04集一次性解释血牒来源、晏南枝身世、月下盟书和复国真相。
- 不要重复第03集石门开门钩子。第04集应从金河粮磨区的新危险、新需求和新物证启动。

## 下一次 writer-room 启动提示

- 启动第04集前，先读取第03集结尾接口：行动地点是金河粮磨区；身体限制是沈维桑左臂失力、肩伤未止、不能满弓；追踪物证包括车辙、清明香、血牒白蜡虫纹风险，以及可升级为清明香沾上焦痕衣角；未解释悬念是“月下在南”与“旧驿指向北境边墙”的矛盾。
- 第04集前5秒不要复用石门开门，应重新设计“金河粮磨区奇观+即时危险同框”，例如磨坊蒸汽、粮磨机关、清明香白线或追踪物证直接威胁沈维桑伤口/伪装。
- 所有抽象目标都要落到镜头可见物证：药布、止血草、粮牌、干粮、苦役伪装、车辙粉末、清明香白线、焦痕衣角或旧名证据。
- 写任何灵翼残纹或驿令作用前，先让角色观察环境痕迹；共鸣只能回应旧驿断裂残响或局部路线，不能直接替代推理。
- 设计追逐、潜入、磨坊机关或逃脱段落时，先建立门线/窄口/敌我位置/力的方向，再写动作结果。
- 第04集结束时继续输出下一集接口，且必须包含行动地点、身体限制、追踪物证、未解释悬念四项。
