# 第03集 Outline：半枚驿令

1. 钩子：半枚驿令点亮山腹石门。目的：证明旧驿系统存在。情绪：惊异。视觉：金赤线沿石缝亮起。
2. 路线争执：晏南枝要去南方隐歌谷，沈维桑要回头找妹妹。目的：建立价值冲突。情绪：互相不信。视觉：血牒与发绳并置。
3. 追兵入道：螳刃虫族循香进入，刃肢刮石。目的：制造压迫。情绪：紧张。视觉：暗道窄光、刃影。
4. 灵翼残纹：沈维桑左肩旧伤发热，眼角金赤纹路闪现，看见废驿活路。目的：展示能力边界。情绪：痛苦和困惑。视觉：短促光纹。
5. 晏南枝吐露：她说月下盟书能证明旧帝国曾有一条没有被清明籍抹掉的共同誓约。目的：把护送目标具体化。情绪：诱导与真诚混杂。
6. 塌门断后：沈维桑用陷阱和旧石栓封住追兵，自己伤肺咳血。目的：强调武脉代价。视觉：石门塌落、血沫。
7. 结尾转折：旧驿图亮点指向北境边墙，晏南枝脸色变了。目的：季路线转向。视觉：地图上雪线光点。
