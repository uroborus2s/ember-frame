# 第03集《半枚驿令》Script v01

时长：约2分45秒  
形式：竖屏短剧

## 1. 外景｜北坡山腹黑石缝｜夜未尽｜0:00-0:18

【0:00-0:05｜前5秒钩子】

极近景：沈维桑右手发抖，半枚日月驿令贴上黑石凹槽。断月纹对不上。

他用拇指抹过掌心血灰，把血灰填进缺口，狠狠按入。

咔。

石门缝里金赤光线一跳，沿残缺日月线疾走。山腹暗道一截一截亮起，像沉睡的路被点醒。

【0:05-0:18】

沈维桑被亮光照出惨白脸色。左肩布条已被血浸透，左臂垂着，鼻血干在唇边。

晏南枝立刻按住怀里的油纸血牒，后退半步。

晏南枝：你做了什么？

沈维桑喘着气，看石门向内错开一掌宽。

沈维桑：我按的。它自己醒的。

门内冷风涌出，吹散他掌心灰血，也吹来一缕淡到发苦的清明香。

晏南枝抬眼。

晏南枝：追兵也醒了。

## 2. 内景｜废驿暗道入口｜夜｜0:18-0:43

两人挤入白石拱道。竖屏上方是剥落日月双印，下方是沈维桑拖着伤腿的脚。

墙上亮起残缺路线：一条细线向南，刻着模糊的隐歌二字；一条断线向北，断处有车轴形旧印。

沈维桑一眼盯住北线，把沈照眠烧焦发绳攥紧。

沈维桑：木车绕北坡走。旧驿线也指北。

晏南枝：南下。隐歌谷有人认血牒上的月下誓印。

沈维桑：认完呢？

晏南枝：把没被白册抹掉的一句盟誓找回来。

沈维桑冷笑，左肩一抽，笑声变成低咳。

沈维桑：我妹不是一句盟誓。

晏南枝把油纸压得更紧。

晏南枝：血牒若进清明院，月下就断了。

沈维桑：车辙若断了，照眠也断了。

石门外，刃肢刮过黑石。声音细长刺耳。

## 3. 内景｜白石窄廊｜夜｜0:43-1:05

清明香残味贴着地缝爬进来。火把光被墙面切碎。

窄廊后方，只有影子先出现：两道螳刃形黑影交叉，刃尖在石壁上划出白痕。

沈维桑反手摸弓，左臂刚抬一寸就失力，箭掉在地上。

晏南枝看见了。

晏南枝：你拉不了弓。

沈维桑捡起箭，改用右手攥成短刺。

沈维桑：能看路。

一截黑绿甲壳从门缝后掠过，又缩回暗里。刮声更近。

晏南枝：看路不能挡刀。

沈维桑蹲下，右指摸过地上湿痕。

沈维桑：挡刀的人，先别挡我眼。

## 4. 内景｜旧驿测绘廊｜夜｜1:05-1:33

测绘廊有三道矮门。墙上旧刻痕被灰盖住，只露出三短一长的测线。

沈维桑没有看亮纹，先看地。

他用右指沾水：左门前湿，水却倒着往门外渗；中门下方灰尘被吸成细线；右门石栓磨痕向外，像常年有人推过。

晏南枝压低声音。

晏南枝：哪边？

沈维桑吹一口灰。灰进中门，立刻直坠不见；到右门，却被一股细风卷回。

沈维桑：中门下面空了。右门有风。

他伸手摸右门石栓。左肩忽然像被火针扎穿，眼角金赤残纹短闪。

半息间，暗道深处传来旧驿铜针般的轻响，方向偏右；随即右门内又回了一声更低的刮石声。

沈维桑猛地缩手，鼻血重新渗出。

晏南枝：你看见了什么？

沈维桑：先看见水。后来，疼了一下。

他指向右门旁一条不起眼的低缝。

沈维桑：门后有东西。走栓下的检修缝。

晏南枝看他眼角残纹已经熄灭。

晏南枝：它能找路？

沈维桑：不能。只能提醒我哪里会死。

## 5. 内景｜断栓岔路｜夜｜1:33-1:58

两人贴地钻过检修缝。竖屏里只见石缝、袖口、油纸血牒和沈维桑拖垂的左臂。

前方出现断栓岔路。南线刻痕完整，北线被石灰封过，封灰上却有新鲜车轮震落的粉末。

沈维桑摸到粉末，眼神一紧。

沈维桑：他们走过北线。

晏南枝：也可能是引你去死。

沈维桑：我看得出新灰旧灰。

晏南枝忽然拽住他。

晏南枝：你听清楚。血牒不是我的护身符。它能对月下盟书旧印。隐歌谷还有认印石。

沈维桑：说人话。

晏南枝：清明籍抹掉名字前，有人立过共同的誓。不是只给皇室的。

沈维桑盯她。

沈维桑：那就更该往北。车、旧驿、我妹，都在北边。

晏南枝：南边能让这张纸活下来。

沈维桑举起发绳。

沈维桑：北边有人还活着。

后方刃肢探入检修缝口，刮掉一片石皮。清明香猛地浓了。

晏南枝松手，眼神仍硬。

晏南枝：先过这一门。出了门，再吵。

## 6. 内景｜塌门石栓｜夜｜1:58-2:27

断栓尽头是一扇半塌石门。门边有两枚旧石栓，上栓裂，下栓完整。

沈维桑靠近，没有碰亮着的日月纹，只摸石栓周围的灰。

上栓灰厚，下栓边缘有新刮痕。

沈维桑：下栓一松，门会落。

晏南枝：你要把路砸断？

沈维桑：砸后面。

刃影压到墙上。追兵不露全身，只见一对刃肢从黑暗里交错探出，甲壳摩擦出硬响。

晏南枝短刀插进下栓缝隙。

晏南枝：你只有一只手。

沈维桑用右肩抵住石门，左臂软垂，额头冷汗滚落。

沈维桑：你还有刀。

两人同时发力。石栓不动。

沈维桑咬牙，掌根顶住栓尾，短促吐劲。闷响很小，只震落一圈灰。鼻血滴在石上，左肩血线立刻扩大。

石栓松半寸。

晏南枝反手一撬。

轰。

石门斜塌，贴着两人后背坠下。刃肢被隔在门外，疯狂刮石，火星从缝里溅进来。

沈维桑跪倒，咳出一口血沫。

晏南枝扶他，他却先推开她的手，捡回发绳。

沈维桑：别碰左肩。

晏南枝：你这样追不到木车。

沈维桑抬头，声音哑。

沈维桑：所以要走旧驿。

## 7. 内景｜旧驿图室｜夜转灰晨｜2:27-2:45

塌门后的风把灰吹开，露出一间窄小图室。墙上是旧驿图，不是画，是密密麻麻的刻线和点位。

半枚驿令在沈维桑掌心发热。他没有按墙，只靠近。

图上南线先亮，又迅速暗下。北线一截一截亮起，穿过一个磨盘形小印。小印旁残字：金河。

晏南枝脸色变了。

沈维桑：金河粮磨区？

晏南枝没有答。她看见更北处，一点光越过所有岔线，亮得刺眼。

那点旁边，是雪线刻痕和黑墙纹。

沈维桑：北境边墙。

晏南枝的手指收紧红线，血牒油纸被捏出细响。

晏南枝：月下盟书的旧驿，不该指那里。

沈维桑看她。

沈维桑：可它指了。

塌门外，刃肢还在刮石。清明香残味从裂缝里渗进来。

沈维桑把烧焦发绳系在右腕，艰难站起。

沈维桑：先金河。再北墙。

晏南枝看着最亮的北境光点，半晌才低声。

晏南枝：进金河，别让他们闻到血牒。

沈维桑右手握紧半枚驿令。

沈维桑：别让他们抹掉车辙。

黑场。

音效：石壁刮声不断，旧驿图上的北境光点仍在轻响。
