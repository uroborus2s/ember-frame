# 第03集：半枚驿令

时长目标：4分10秒
格式：AI竖屏短剧详细剧本
本集功能：用火后残阳坳、诱线奴兵和旧驿暗道证明沈家并非普通猎户，北上目的从“逃离”变成“去月下”。

## 场景表

1. 火后残阳坳：驿令亮路。
2. 山道难民沟：诱线奴兵接近。
3. 旧井废口：沈维桑识破暗语。
4. 旧驿暗道：残缺路线指向北境。

## 成片剧本

### SC01 火后残阳坳 / 清晨 / 5秒钩子

灰烬下，半枚日月驿令贴上焦黑石缝。地面亮起半截金赤残纹，像被烧剩的血线。

沈维桑蹲在废井边，左手按住驿令，右手拨开草药灰。

晏南枝：这是旧帝国驿路。

沈维桑：这是我家井底。

晏南枝看向他。沈维桑没有看她，只把一截烧焦发绳塞进怀里。

沈维桑：照眠被押车往北。你最好也往北。

晏南枝：月下盟书在北境旧驿。你娘没骗你。

沈维桑：她从不骗我。她只少说一半。

远处传来清明香铃。沈维桑立刻用灰盖住光纹。

### SC02 山道难民沟 / 上午

二人混入逃荒队。妇人推着空粮车，老人背祖牌小盒，少年腿上有白册复核条。

一个诱线奴兵伪装成边墙逃兵，肩上披旧军户围巾，后脑低位却露出金钱马尾。

诱线奴兵：北去路封了。走月下沟，能避清明香。

晏南枝下意识抬头。

沈维桑一把按住她的袖口。

沈维桑：山里没有月下沟。

诱线奴兵笑。

诱线奴兵：你不是本地人。

沈维桑也笑，眼睛看向它腰侧气味牌。

沈维桑：本地人也不把死人路说给活人听。

诱线奴兵手腕一转，白册小签露出。两名缚网奴兵从逃荒队后方张网扑来。

沈维桑把粮车一脚踢翻，杂草包滚满坡道。晏南枝拔出袖中短匕，割断一根黏网绳。

晏南枝：你说你只是猎户。

沈维桑：猎户也知道谁装得不像人。

### SC03 旧井废口 / 午后

两人冲到火后旧井另一侧。井壁有罗青禾曾用草药灰遮住的刻痕。沈维桑用驿令抵住刻痕，白石井壁裂开一道低矮暗门。

诱线奴兵追到井边，故意喊。

诱线奴兵：旧驿后人，按昭明礼，当答三声。

晏南枝脸色一紧，几乎要按礼回应。沈维桑捂住她嘴。

沈维桑：这是清明院改过的礼。

他朝井边丢出一块刻错暗号的木牌。诱线奴兵果然追错方向，踩进沈维桑临时拉起的草绳陷阱，被吊倒在井架下。

沈维桑：真驿人不答三声。答风向。

晏南枝望着他，第一次不再把他当普通山民。

### SC04 旧驿暗道 / 黄昏

暗道窄而低，白石拱顶剥落，日月双印被虫蜡白泥盖过。半枚驿令只能点亮断续路线。

晏南枝用血牒靠近石壁。旧光没有向南亮，反而向北延伸。

晏南枝：不对。南方复翼者说，安全路在南。

沈维桑：复翼者来过残阳坳？

晏南枝沉默。

沈维桑把一枚旧绳结从墙缝里取出。绳结上有沈季衡的测绘记号。

沈维桑：这是我爹的手法。

晏南枝：你父亲是测绘官后代。

沈维桑：我父亲死于山洪。

暗道尽头，一块残图被石灰封住。沈维桑刮开石灰，残图显示路线越过金河、灰烬书院、墙下集市，最后指向北境月下旧驿。

晏南枝轻声读出图上残字。

晏南枝：月下路，不入皇旗。

沈维桑眼神一动。

暗道外，白翳捡起那块刻错暗号的木牌。木牌没有骗过他太久。

白翳：半句真话，半条假路。

他把木牌放入白册，沈家的旧姓旁注又深了一层。

暗道内，沈维桑背起路线包。

沈维桑：去金河。查押车，找船，往北。

晏南枝：去月下。

沈维桑看她。

沈维桑：先找我妹妹的车痕。你的月下，排后面。

金赤残纹在二人脚下熄灭，只剩北方一线微光。

切黑。
