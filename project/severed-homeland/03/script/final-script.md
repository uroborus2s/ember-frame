# 第03集《半枚驿令》Final Script

时长：约2分45秒  
形式：竖屏短剧

## 1. 外景｜北坡山腹黑石缝｜夜未尽｜0:00-0:18

【0:00-0:05｜前5秒钩子】

极近景：沈维桑右手发抖，半枚日月驿令抵进黑石凹槽。断月纹对不上。

同一竖屏画面里，石缝下方先爬进一线清明香，贴着他的鼻血打旋；画面上沿，一截螳刃刮近黑石，刮声尖得像针。

晏南枝一手按住怀里的油纸血牒，一手扣住石壁。

晏南枝：快。

沈维桑用拇指把掌心血灰抹进缺口，狠狠一按。

咔。

石门缝里金赤光线一跳，残缺日月线贴着山腹石缝疾走。黑石像被旧火从里面点亮，门向内错开一掌宽。

【0:05-0:18】

亮光照出沈维桑惨白的脸。左肩布条被血浸透，左臂垂着，完全抬不起来。

门缝冷风涌出，把清明香吹散半寸，又把更苦的一缕香从外面卷回来。

石门外，刃肢第二次刮到凹槽边。

晏南枝低声：追兵闻到门了。

沈维桑收回半枚驿令，先把沈照眠烧焦发绳塞进右腕，才侧身钻入门缝。

沈维桑：那就别把门留给他们。

## 2. 内景｜废驿暗道入口｜夜｜0:18-0:48

两人挤进白石拱道。竖屏上方是剥落日月双印，下方是沈维桑拖着伤腿的脚。

墙上亮起两道残线：一条细线向南，刻着模糊的「隐歌」二字；一条断线向北，断处有车轴形旧印。

沈维桑一眼盯住北线，右手攥紧发绳。

沈维桑：木车绕北坡走。车轴印也在北线。

晏南枝看南线，手指却忽然一僵。

清明香贴着她衣襟钻上来，油纸血牒边缘浮出白蜡状虫纹，像细足从纸里往外顶。虫纹一碰到残缺日月血纹，血纹被逼亮一瞬。

晏南枝立刻用指甲刮掉那层白蜡，指尖被割出血。

沈维桑看见了。

沈维桑：那是什么？

晏南枝：香在记它。

她把油纸压回衣内，声音压得很低。

晏南枝：它被白蜡虫纹逼满三层，隐歌谷的认印石也救不了。到那时，血牒会变成白册能追的味。

沈维桑：我妹现在在车上。

晏南枝抬眼。

晏南枝：我不是叫你丢下她。你追车辙，是救人；我保住血牒，是保她旧名还有被叫回来的证据。

沈维桑举起发绳。

沈维桑：她没有三层机会。

石门外，刃肢刮声逼近。清明香猛地浓了。

## 3. 内景｜白石窄廊｜夜｜0:48-1:08

两人沿窄廊前行。墙面亮纹断断续续，像被人从中间砸烂过。

后方影子先追上来：两道螳刃形黑影交叉，刃尖在石壁上划出白痕。

沈维桑本能反手摸弓。左臂刚抬一寸，肩伤猛地抽紧，弓弦擦过手背，他没能拉开。

箭掉在地上。

晏南枝看见他的左臂。

晏南枝：你拉不了弓。

沈维桑弯腰捡箭，用右手攥成短刺。

沈维桑：那就别让我站远处。

一截黑绿甲壳从拐角掠过，又缩回暗里。刮石声更近。

晏南枝短刀横在身前。

晏南枝：看路。

沈维桑蹲下，右指摸过地上湿痕。

沈维桑：别挡光。

## 4. 内景｜旧驿测绘廊｜夜｜1:08-1:37

测绘廊有三道矮门。墙上旧刻痕被灰盖住，只露出三短一长的测线。

沈维桑没有先看亮纹。他先看地。

左门前湿，水却倒着往门外渗；中门下方灰尘被吸成细线，像下面空了；右门石栓磨痕向外，常年被人推过，门边却有新刮痕。

他吹一口灰。灰进中门，直坠不见；到右门，被一股细风卷回，同时门后传来一声低低刮石。

晏南枝：哪边？

沈维桑：中门下面塌空。右门后有东西。

他伸手碰右门旧石栓。左肩忽然像被火针扎穿，眼角金赤残纹短闪。

半息间，暗道深处响起一声旧铜针般的回音，偏右；随即同一方向传来第二声断裂的闷响，像坏掉的石栓在里面松动。

沈维桑猛地缩手，鼻血重新渗出。

晏南枝：灵翼残纹能带路？

沈维桑摇头，指向右门下方一条不起眼的低缝。

沈维桑：它不找路。旧驿坏掉、断开的地方，会回一声。

他用箭尖拨开低缝灰土，露出一条检修缝。

沈维桑：门后有断栓。走栓下。

## 5. 内景｜检修缝/断栓岔路｜夜｜1:37-2:02

晏南枝先贴地钻进检修缝，油纸血牒被她压在胸口。沈维桑最后过，右手撑地，左臂拖在身侧，血在石缝上擦出一条细线。

后方刃肢探入检修缝口，只伸进半尺就被低石卡住。刃尖刮掉一片石皮，火星溅到沈维桑靴跟。

沈维桑用右脚一蹬，滚进缝后。

两人从检修缝另一端钻出，落到右门背后的断栓岔路。南线刻痕完整，北线被石灰封过，封灰上却有新鲜车轮震落的粉末。

沈维桑抹起粉末，眼神一紧。

沈维桑：他们走过北线。

晏南枝：也可能是引你去死。

沈维桑：新灰盖不住车震。

晏南枝抓住他右腕，力道不轻。

晏南枝：清明籍抹名字前，有人用这旧印保过一批不该被改名的人。

沈维桑看她。

沈维桑：照眠也在里面？

晏南枝没有答满。

晏南枝：所以我必须让血牒活到隐歌谷。

沈维桑盯着北线粉末。

沈维桑：所以我必须让车辙没断。

检修缝口传来刃肢连续敲击。石缝开始裂。

晏南枝松手，短刀反握。

晏南枝：先过塌门。活着的人才有路吵。

## 6. 内景｜塌门石栓｜夜｜2:02-2:30

断栓岔路尽头是一扇半塌石门。门线外是追兵廊，门线内右侧有一道窄口，通向更深的图室，只够一人半侧身挤过。

石门有两枚旧石栓：上栓裂开，顶着门顶；下栓完整，却被旧灰咬死。

沈维桑看一眼门倾斜的方向。

沈维桑：先过门线。门会往外侧落。

晏南枝先挤进右侧窄口，回身把短刀插入下栓缝。沈维桑跨过门线，用右肩抵住石门内侧，左臂仍垂着。

后方，追兵撞开右门。黑影压上墙，只见一对刃肢交错探出。

晏南枝：我撬。你只松栓。

沈维桑掌根顶住下栓尾端，短促吐劲。

闷响很小，只震落栓缝一圈灰。

石栓松了半寸。鼻血滴在石上，左肩布条立刻洇开。

晏南枝抓住这一点缝，短刀反手一撬。旧下栓脱槽，上栓裂纹跟着咬断，半塌石门失重，斜向追兵廊砸下。

轰。

门没有震飞追兵，只把刃肢隔在外侧。门顶卡在断裂上栓上，沈维桑这边留下通向图室的一条窄口。

刃肢在石门外疯狂刮擦，火星从缝里溅进来。

沈维桑跪倒，咳出一口血沫。晏南枝伸手扶他，他先用右手护住怀里的发绳。

沈维桑：别碰左肩。

晏南枝看着他垂死的左臂。

晏南枝：你这样追不到木车。

沈维桑撑着墙，声音哑。

沈维桑：所以走旧驿。

## 7. 内景｜旧驿图室｜夜转灰晨｜2:30-2:45

塌门后的风把灰吹开，露出一间窄小图室。墙上不是画，是密密麻麻的刻线和点位。

半枚驿令在沈维桑右掌发热。他没有按墙，只靠近。

图上南线先亮到「隐歌」二字旁，又迅速暗下。北线一截一截亮起，穿过一个磨盘形小印。小印旁残字浮出：金河。

沈维桑：金河粮磨区。

晏南枝还没回答，更北处一点光越过所有岔线，亮得刺眼。那点旁边，是雪线刻痕和黑墙纹。

沈维桑：北境边墙。

晏南枝下意识把束发红线绕紧，勒得指节发白。

晏南枝：师长说月下在南，不在墙后。

沈维桑看着那点光。

沈维桑：可旧驿说它在墙后。

他艰难站起，左臂完全垂着，肩上的血顺着衣角滴到金河小印下，把磨盘刻痕染暗一角。

晏南枝按住他右肩。

晏南枝：进金河前，先止血。

沈维桑没有说没事。他用牙咬住发绳，把它系在右腕，目光仍钉着北线。

沈维桑：先金河。别让他们抹掉车辙。

塌门外，刃肢刮石声不断。图室里，北境边墙的光点仍在轻响。

黑场。

## 重写说明

本稿保留半枚驿令开启石门的开场，并在前5秒同框加入清明香、刃肢逼近和晏南枝催促，让奇观与追兵压力同时成立。晏南枝的血牒风险改为可见物证：清明香逼出油纸边缘白蜡虫纹和日月血纹，明确血牒被白册记味会危及旧名证据，从而把南下隐歌谷与沈照眠风险接上。路线争执升级为两种救法冲突：沈维桑追北线车辙救妹妹，晏南枝南下保血牒和旧名证据，不把她写成只护纸不顾活人。灵翼残纹台词收窄为只回应旧驿坏掉、断裂处的残响，不找路、不替代观察。暗道空间补清为检修缝、断栓岔路、门线内侧窄口和向追兵侧斜落的塌门；沈维桑明劲只松下栓半寸，塌门主因是旧石栓结构与晏南枝短刀撬动。月下盟书信息改为「有人用这旧印保过一批不该被改名的人」，避免提前说透主题。结尾保留金河磨盘小印与北境边墙亮点，加入晏南枝「师长说月下在南，不在墙后」的具体反应，并用左臂垂死、肩血滴到金河小印和「先进金河前，先止血」衔接第04集伤势。
