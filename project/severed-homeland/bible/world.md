# 世界观

## 核心命题

《断航故土》的核心不是单一的族群反抗，而是一个被篡夺、被改名、被断航、被灭史的旧文明，如何在幸存者手中重新理解“故土”。

第一季固定主题句：**日月照耀皆为故土，但故土不再由帝国旗帜单独定义，而由活人之间互相承认、互相守护的承诺定义。**

## 全剧风格锁定

全剧固定为**低魔东方史诗横越大陆冒险剧**。第一季以逃亡护送为主线骨架，后续季逐步扩展为沿金河、东海、南林、西部高原、白曜旧都、东北沼泽和北境灾源推进的历史追索冒险。它不是时间穿越剧，不是诙谐幽默剧，也不是单纯大陆争霸爽剧，更不是宫廷斗嘴式权谋剧。

推荐并采用的风格配比：

1. 低魔东方史诗：负责世界尺度、旧文明断代、边墙、断航、旧驿和月下盟书。
2. 逃亡护送短剧：负责每集钩子、追捕、路线选择、道具危机和人物互相逼问。
3. 制度悬疑与政治阴谋：负责清明籍、白册、识别者、灭械令、焚帆令和肃明内部清明院与甲军府分歧。
4. 横越大陆冒险：负责让每一季通过一个地理区域找回一层被抹掉的大陆历史。
5. 大陆战争阴影：负责北境雪灾、五百年边墙血战、十三部落联盟和后续多域扩展，但第一季不把大陆争霸当作主线战报。

幽默只允许作为角色在高压中的短促呼吸，例如陆青砾的生存型讥讽、沈维桑的冷回应或薛临墙的干硬反话；不得把压迫、屠村、识别者儿童和历史灭绝写成轻喜剧。

## 全剧故事主题

全剧主题不是“复国成功”，而是“找回历史之后，如何避免把旧帝国的排除逻辑重新复制一遍”。

沈维桑、晏南枝和同伴每抵达一个新区域，都必须找回一段被肃明抹掉、也被昭明旧叙事遮蔽的历史：

1. 月下盟书揭示故土不是皇族独占。
2. 东海断航揭示航权被焚毁，也揭示昭明旧海债。
3. 南林与金河源头揭示帝国工程如何伤害林地、矿脉、河流和边墙。
4. 白曜城与东北沼泽揭示肃明的文明正统叙事同样建立在流放、饥荒、学习、篡夺和伪史上。
5. 北境灾源揭示边墙最初是多族共同抵御灾变的工程，不是永恒敌我分界。

全剧终局必须把这些历史残片转化为当下制度选择：拆除清明籍、公开旧档、重开有限航路、修复边墙阵基、让无姓人和混裔入名，并以新日月盟约取代单一皇权或单一巢权。

## 争霸规则

全剧后期需要战争和权力争夺，但不采用“主角争霸称帝”结构。

固定采用：**盟约战争，而非帝王争霸。**

后期争夺对象是粮仓、渡口、白册总源、旧都档案、东海航路、边墙阵基、金河总闸和多族盟约解释权。沈维桑不称王，晏南枝不以皇血自动号令天下，白曜城终局也不是简单夺回王座。真正的胜负标准是旧史能否公开、旧名能否归还、航路能否重开、边墙能否从尸墙变回共同守护工程。

## 当前时代

当前时代为肃明国统领中央大陆后的肃明历 1226 年。旧王朝灭国于昭明历 5249 年，该年也是肃明历元年。第一季开端固定写作“肃明历1226年”。

肃明国以“肃尽前明，还天下清明”为国号逻辑和统治承诺。它将昭明帝国污称为“前明”或“日月旧朝”，通过清明籍、遗忘术、迁徙、禁书、锁翼咒、边墙军户制度和东海禁岸，系统性清除旧帝国血脉、道路、航权、地名和记忆。

同年，北境雪灾和兽群南压加剧，肃明国借边墙军需加重征粮、征役与清明籍筛查。南部旧文明遗民开始以旧歌、药包暗语和旧驿联络残存据点。

## 昭明帝国

昭明帝国曾以“日月照耀皆为故土”为最高宣言，建立白石城、旧驿道、边墙、龙船船坞、潮星灯塔、结界海航标和星图法阵。它是法术、道路、工程、枪炮、蒸汽和组织力结合的复合文明。

昭明并非纯洁无瑕。帝国曾扩张、砍伐南林、压迫小族、控制海港并伤害海族。第一季不得把昭明写成无罪旧朝，也不得把肃明写成只有怪物本能的敌人。双方都携带历史债务，区别在于肃明把清除活人当成秩序手段，而主角线要寻找不靠清除建立的共同体。

## 法术、战争与记忆规则

世界上没有独立仙族。后世所谓仙族，是昭明旧人族中的云裔法统、天工山门、龙船匠、结界工程师和星图师被肃明伪史切割后的误称。

人族旧法以文字、歌谣、星图、驿道、船舶、结界、呼吸和灵翼经络为基础。它不是个人随手释放的神通，而是把语言、身体、道路、船舶、建筑、仪式、星图和工匠组织成一套文明级技术。高阶法力显化为肩胛、脊柱、手腕和眼角的灵翼纹路，不是人人拥有实体羽翼。

昭明旧战争中的法术主要服务于六类职能：烽燧、星图台和驿道符标的通讯；借风、水、鸟兽和星象完成侦察；短时加固城门、盾墙、船体和边墙阵基；辅助火枪、大炮、蒸汽炮车和投石机点火、测距、校正风偏；延缓失血、压制毒素、稳定伤兵；快速测绘地形、修桥、排水、开矿和筑垒。因此旧帝国强大之处不是“法师统治战场”，而是少数法师、普通士兵、工程师、工匠、枪炮、蒸汽、道路和粮仓共同形成战争系统。

遗忘术能压制语言记忆、历史认知和血脉觉醒，但无法彻底抹除旋律、身体动作、地名、梦境、石刻、织物图案、潮汐直觉和血脉反应。旧驿暗语、禁歌、半枚日月驿令、旧地图和儿童游戏都是记忆残留的可见载体。

锁翼咒压制的也不是单个术式，而是人族把文明记忆编织成法术的能力。灵翼经络没有消失，只退化为残纹、梦境、路线感、短促共鸣和身体动作。沈维桑的旧驿方向感、沈照眠对旧暗语的反应、南部隐歌和边墙武脉，都属于被压碎后仍从身体里残留的旧法回声。

## 武力、技术与边墙武脉规则

昭明旧帝国曾拥有火枪、大炮、军用蒸汽、战争工坊和星图校准体系。肃明建国后推行灭械令，焚毁军工图纸、炮模、枪管量具、火药配方、军械学院和战争工坊，只保留生产、采矿、排水、粮仓、运输和城建所需的蒸汽技术。

灭械令造成的不是单纯技术倒退，而是战争平衡被故意改写。人族失去火器、炮兵、军用蒸汽和精密测距后，数量优势再难弥补与虫族甲胄、北境巨兽、瀛洲兽族和海族海兽之间的体形差距。肃明保留能产粮、采矿和运输的蒸汽，是为了统治继续运转；摧毁能改变战场平衡的军工，是为了让战争回到体能优势族群更占便宜的冷兵器环境。

因此第一季战争以冷兵器、弩、陷阱、盾墙、床弩、虫族体能、北境共生兽、边墙武脉和少量残缺法阵为主。常见战斗不是高魔对轰，而是守门、截路、拖延、诱敌、换牌、救人和从制度缝隙里逃生。不得让主角用法术或火器一招解决危机。

边墙武脉是五百年边墙冷兵器消耗战催生出的身体强化体系。它不等同于旧帝国高阶法术，也不是江湖门派功法，而是普通人族在被剥夺枪炮、完整法术和充分军粮后，为了弥补体形、力量和耐寒差距，把站桩、盘架、行桩、负重、寒暑打熬、筋骨锻打、短爆发步法、墙体借力、骨钟节奏和旧帝国内家残义揉成一套守墙术。

边墙武脉的进阶固定为七层：

1. 筑架：外家入门。把脚、胯、脊、肩、肘和兵器连成一体，能长时间守墙、挨冻、搬石和持械。
2. 明劲：外家第一成。整身发力，劲从脚起，贯通腰背到刀、枪、弓，能短促爆发超出常人的力量。
3. 暗劲：外家深处。劲入筋膜和呼吸，发力更短、更隐、更狠，适合近身破甲和救人突围，但最容易伤肺、伤心脉。
4. 化劲：内外门槛。能听劲、卸力、借力，不再硬碰巨兽、刃肢和冲车。边墙军中能到此境者极少，通常会被称为老墙师。
5. 内息：内家初成。呼吸不只用于爆发，而能调和气血、温养旧伤，开始修复外家阶段留下的骨裂、寒毒和脏腑损伤。
6. 归元：内家深处。心意、呼吸、步法和兵器合一，战斗中能保持清明，不被恐惧、复仇和杀意牵着走。
7. 合天：内家极境。对应昭明古老的天人合一哲学，能在风雪、城墙、地脉和人群情绪中读出同一条势。这不是神仙法术，而是人身、心意与环境秩序短暂相合。

边墙武脉最大的代价来自外家阶段。筑架、明劲和暗劲能让普通人族在极端战场短暂活下来，但会损耗寿命、留下旧伤、造成筋膜僵硬、寒毒入骨、胸肺淤痛和脏腑衰竭。绝大多数边墙武士终身止步外家，因此他们的强悍本身就是战争断代和制度消耗的证据。

真正的修复只能发生在内家阶段。化劲之后，武士不再只用身体硬扛，而是以呼吸、松沉、听劲、气血归根和心意调和修复自身。这个修复路径要求人不再只为复仇、恐惧或求胜而练；能走入内家的人，必须学会让自己与风雪、山墙、地脉和活人承诺重新相合。它服务全剧主题：故土不是靠透支尸墙维持，而是靠修复活人、工程和盟约维持。

沈维桑的能力起点是猎户、引路人、筑架和残缺明劲。他尚未真正进入暗劲，更谈不上内家。他的灵翼残纹只能提供短促共鸣、路线感和记忆反应，不能成为压倒性攻击能力。主线中的武力成长不是从普通人突然变成法师，而是从外家求生、承受代价，逐步理解天人合一与新故土承诺，才获得进入内家修复道路的可能。

## 第一季故事规则

第一季主线为护送旅程：残阳坳被清明籍搜查引爆，沈维桑因母亲遗言护送晏南枝北上，寻找月下盟书旧驿，同时追查父亲死亡和妹妹沈照眠失踪真相。

护送不是单纯“送皇血复国”。每一站都要让晏南枝和沈维桑看见旧帝国正统无法自动容纳的人：奴兵、混裔、灰墙军户、旧书会、北境灾民、被训练成识别者的儿童。

第一季终点不是复国胜利，而是主题反转：月下盟书证明旧帝国末期曾有多族共同守边的残缺承诺。沈维桑带走的不是完整皇旗，而是“故土必须重新写入活人”的证据。

第一季只是全剧完整冒险线路的第一段，不得把月下盟书当作全剧终局。它打开东海、南林、西部源河、白曜旧都、东北沼泽和北境灾源的后续历史追索线。
