# 项目级视觉统一包

## 1. 视觉定位

项目：《断航故土》

季名：残阳驿火

形式：AI竖屏短剧，9:16，单集2分30秒到3分钟。

总体风格：低魔东方史诗奇幻，写实电影质感，带轻度风格化。画面应像真实历史废墟和边境战场，而不是高饱和游戏立绘。人物服饰以中国先秦到秦汉早期的结构感为参考，但不直接复刻历史朝代。法术和灵翼残纹只作为短促视觉证据，不承担华丽战斗特效。

关键词：旧文明被覆盖、白石与虫蜡、清明白墙、残阳火场、蒸汽断代、灰烬书院、雪山边墙、日月残纹、活人承诺。

禁忌方向：

1. 不要仙侠飘逸化。云裔不是独立仙族，第一季不出现仙人式大袖飞升。
2. 不要把沈维桑画成王者铠甲或成熟将军。他是17岁猎户和逃亡者。
3. 不要把晏南枝画成男性，也不要画成衣衫褴褛的乞逃者。她是流亡中的皇族公主女性形象，粗布外衣可以旧、素、蒙尘，但必须清冷整洁，仍保留高冷、克制、受过礼法训练的贵族气质。
4. 不要把虫族做成纯怪物。肃明国上层是人族官制外壳加局部虫相。
5. 不要把旧帝国视觉做成纯正义圣光。昭明遗产应有磨损、裂纹和旧债感。
6. 不要让昭明旗帜与肃明旗帜混淆。昭明必须是朱赤暖旗和“日轮托白月”合徽；肃明必须是冷白官旗和“白翅覆黑日”，黑日要有冷白/暗灰日晕，月纹被故意抹去，白翅要更抽象，不能像写实昆虫翅膀标本。
7. 不要过度华丽。第一季核心是逃亡、清剿、断代和边墙冷兵器生存。

## 2. 画幅与镜头语言

画幅：9:16竖屏。

构图：优先中近景、近景、手部道具特写、低角度压迫镜头、窄空间追逐。少用横向大场面。

运动：逃亡段使用手持感、压低机位、遮挡前景；清明院使用稳定、对称、冷静镜头；边墙使用低角度、风雪遮挡、压迫性垂直线。

人物一致性：每个角色至少保留三类固定锚点：脸部/发型锚点、服装锚点、道具锚点。每集可以脏污、破损、湿冷，但不能换掉身份锚点。

光效：灵翼残纹、旧驿光路、月白玉片和盟书石台只允许短促发光。发光应像旧阵基残留，不像现代霓虹。

## 3. 全剧色彩系统

全剧基础色：灰褐、旧白、黑石、冷青、虫蜡白、暗红、金赤残光。

颜色使用规则：

1. 残阳坳：土褐、草药青、残阳橙、火场黑灰。
2. 肃明/清明院：虫蜡白、冷灰、浅骨色、少量暗黑日轮。
3. 昭明旧物：旧白石、磨损金、月白、金赤裂纹。
4. 金河粮磨：铁灰、蒸汽白、油黑、粮仓木褐。
5. 灰烬书院：焦黑、灰白、残月白、烧纸黄。
6. 墙下集市：灰蓝、陶土、破布褐、暗粮牌黄。
7. 边墙雪线：黑石、雪白、冰青、骨色、干血暗红。
8. 北境角色：冰蓝、皮毛灰、骨白、极光绿点缀。

全剧高亮色只保留三类：

1. 朱赤与金赤：昭明旧帝国旗帜、沈维桑灵翼残纹、旧驿光路、残阳火场反光。
2. 月白：晏南枝月白玉片、盟书石台、碎星图。
3. 清明白：白翳、清明院、白册、虫蜡针和清明香。

## 4. 材质语言

昭明旧文明：白石、磨损铜、旧金纹、裂纹石刻、星图盘、驿道碑、龙骨量尺痕迹。材质必须有年代感，不能崭新。

肃明国：虫蜡、白墙、半透明虫膜、黑日徽记、甲壳纹、无尘纸册、骨白针具。材质干净但不温暖。

边墙：黑石、冻裂木、旧铁、骨钟、粗麻、灰甲、雪泥、裂旗。材质粗粝、冷硬、消耗感强。

人族遗民：麻布、旧皮、草药袋、磨损铜饰、红线、焦痕、手写旧符。要有手工修补和逃亡磨损，但“贫困”不能统一画成破烂乞逃者。衣服可以粗布、旧、打补丁、蒙尘、局部焦痕或修补线明显，但核心角色的服装必须完整可穿，不应大面积撕碎、流苏化或下摆破成布条。

北境：兽皮、骨饰、冰裂纹、皮绳、粗铁、骨笛、低饱和冰蓝火。

## 5. 派系视觉锁

### 昭明旧帝国遗产

符号：日轮托白月、日月双印、旧驿路线刻痕、月下盟书多族印记。

形状：圆形日轮包托月白弧月、日月融成单一徽章、白石拱、星图圆盘、路线网格。

颜色：朱赤旗底、磨损金日、月白、旧白石、金赤裂纹。

旗帜：昭明旗必须是暖色人族帝国旧旗。主形为朱赤或暗红长幅战旗，中央是磨损金色日轮，白月必须放在日轮内部，被日轮托住并融成一枚完整“日月合徽”，不要画成太阳和月亮分离并列。旗边可有旧金织边和磨损裂口。整体感受是旧日光、旧血、旧誓言。

使用方式：城门、驿站、边碑、军械、船坞、星图台和边墙阵基上都可刻有日月双印。它原本的含义是：凡日光可达、月色可临之地，皆受帝国庇护，也皆为帝国应归之土。第一季只出现在旧物、残阵、驿令、血牒、盟书、破损旗布和被虫蜡覆盖的墙面，不作为完整辉煌宫廷出现。

### 肃明国与清明院

符号：白翅覆黑日，月纹被故意抹去。

形状：白墙直线、蜂巢孔、虫蜡封条、薄翼披肩、白色虫翅横压黑色日轮、被刮除的月纹空痕。

颜色：虫蜡白、冷灰、骨白、黑日、极少量暗青。

旗帜：肃明旗必须是冷色官旗。主形为虫蜡白或骨白竖幅旗，中央是带冷白/暗灰日晕的黑色日轮，被抽象白翅从上方或横向覆盖、压住、遮蔽。白翅应像几何秩序、洁白覆盖带、折线翼骨或冷白羽阵的抽象符号，不要像写实昆虫翅膀标本。旧帝国日月徽中的月纹必须被刮掉、涂白或留成空白伤痕。它的意境是虫族以洁净秩序覆盖旧人族日月旗，而不是另起一套中性黑日徽。整体感受是清除、覆盖、官僚秩序和冷洁暴力。

使用方式：白翳、清明院、白册、虫蜡针、审讯空间必须保持干净、冷、无尘，与残阳火场和边墙灰尘形成强反差。

### 边墙与灰墙军

符号：骨钟、灰墙军牌、反面旧日月布条、黑石裂门。

形状：垂直城墙、粗重铁闸、长枪直线、雪中剪影。

颜色：黑石、雪白、灰麻、旧铁、干血暗红。

使用方式：边墙人族不应像正规帝国军。他们是被消耗的军户，服装应混杂、修补、破损。

### 墙下混裔

符号：碎陶护符、黑市木牌、粮牌、无姓刻痕。

形状：窄巷、棚布、窑洞、低矮门洞、切碎的旗布。

颜色：灰蓝、陶土、布褐、暗黄。

使用方式：不要把混裔做成单一“异族奇观”。他们应像边缘生活者，混杂特征非常克制。

### 北境寻灾源派

符号：骨笛、鹿角短饰、雪地药草袋、冰蓝火。

形状：兽骨孔、皮绳结、冰裂纹、迁徙路线痕。

颜色：冰蓝、皮毛灰、骨白、极光绿点缀。

使用方式：第09集只做局部登场，避免大规模部族展示。

## 6. 角色视觉总规则

1. 年龄必须稳定。沈维桑17岁，晏南枝18岁，沈照眠是儿童。
2. 沈维桑必须有少年感：剑眉、清亮但戒备的眼睛、身形瘦硬，不能画成成熟硬汉。
3. 沈维桑衣角焦痕从第02集后持续存在。
4. 晏南枝必须是女性，定位为流亡中的皇族公主。外层是清冷整洁的粗布旅装，可以旧、素、蒙尘和局部磨损，但不能衣衫褴褛；站姿、眼神、发束和内层纹样仍保留周秦式贵族礼法气质。
5. 晏南枝红线束发和月白玉片不能丢。
6. 白翳必须保持无尘、克制、白袍、薄翼披肩和虫蜡针。
7. 陆青砾必须保留耳后细鳞痕、碎陶护符和灰蓝短袄。
8. 薛临墙必须保留冻裂手、骨哨、磨损长枪和反面旧日月布条。
9. 沈照眠有两个状态：残阳坳儿童状态和清明院识别者状态。状态变化要保留烧焦发绳或小草药袋作为亲情锚点。
10. 人族贫民与逃亡者不应共用一种“破布”造型。沈维桑是平民猎户：灰褐短打、皮护腕、弓箭、短刀、陷阱绳和路线小包形成职业轮廓；沈照眠是村中孩童：短襦、围裙/药屋帮工小袋、烧焦发绳形成儿童轮廓；陆青砾是墙下混裔和黑市生存者：灰蓝短袄、隐藏口袋、黑市木牌、碎陶护符和窄刀形成墙下轮廓；薛临墙是边军老军户：旧甲片、厚衬袄、骨哨、长枪、雪线绑腿和反面旧日月布条形成边军轮廓。
11. 补丁和磨损应服务职业信息：猎户服装补强肩肘和护腕，村童补丁小而生活化，墙下服装强调口袋、绳结和交易牌，边军服装强调甲片、旧铁扣和御寒层。不要用同一种撕裂边缘替代职业设计。

## 7. 场景视觉总规则

第一季路线必须保持由暖到冷、由土村到黑石边墙的视觉迁移：

1. 第01-03集：残阳、土墙、火场、暗道。
2. 第04-06集：蒸汽、灰烬、白墙。
3. 第07集：墙下灰蓝混杂区。
4. 第08-12集：雪、黑石、骨钟、地下星图厅。

不能在第一季中大量使用南林巨树、东海龙船、瀛洲兽纹灯笼、西部矮人熔炉。它们只可作为后续季接口或极短符号，不进入主视觉。

## 8. 资产生成顺序

第一批参考资产：

1. 风格板：全剧风格、肃明/昭明/边墙三套材质板。
2. 角色板：沈维桑、晏南枝、白翳、沈照眠、陆青砾、薛临墙。
3. 核心道具板：日月驿令、月白玉片、清明白册、虫蜡针、旧驿图、盟书残页。
4. 核心场景板：残阳坳、旧驿暗道、清明院外署、锁喉关、月下盟书旧驿。

第二批参考资产：

1. 孟归藏、鹿弥、厉螳、罗青禾。
2. 金河粮磨区、灰烬书院、墙下集市、鸣骨岭、寒鸦堡。
3. 服装破损/火场/雪地状态图。

第三批资产应等第01集 `final-script` 和 director-room 镜头表完成后再生成。

## 9. 提示词使用原则

项目级提示词必须工具中立，只描述主体、视觉锚点、材质、色彩、画幅和禁用项，不写 ComfyUI 节点、采样器、步数、模型名或 LoRA。

后续 Prompt Room 可以把这些项目级提示词转换为 ComfyUI、Midjourney、SDXL 或其他工具的具体提示词。
