# 连续性 Bible

## 1. 核心规则

1. 世界上没有独立仙族。所谓仙族是旧帝国云裔人族高阶法统被虫族伪史切割后的误称。
2. 沈维桑第一季不是强法师。他主要依靠猎弓、短刀、陷阱、地形判断、残缺边墙武脉和短促灵翼残纹。
3. 灵翼残纹只能提供方向共鸣、旧驿感知和短暂视觉提示，不能一招击败敌人。
4. 肃明国没有完全摧毁蒸汽文明，只摧毁能改变战争平衡的军工、火器、炮兵和测距体系。
5. 白翳不是单纯残暴反派。他相信清除旧记忆能阻止战争，这种信念必须保持稳定。
6. 晏南枝早期相信血统和旗帜的政治价值，但不能写成冷血利用他人；她是真心认为没有象征就无法凝聚遗民。
7. 昭明帝国既是被篡夺的旧文明，也有扩张和压迫旧债。第一季必须逐步揭露双面性。
8. 每集结尾悬念应来自可见事实：道具、路线、身份、追兵、旧档案或角色选择，不依赖旁白解释。

## 2. 道具连续性

半枚日月驿令：第02集由罗青禾交给沈维桑；第03集打开废驿石门；第11集唤醒月下盟书旧驿；第12集只唤醒残阵，不变成万能钥匙。

月白玉片：晏南枝随身物，第02集露出，第05集与星图碎片共鸣，第11集贴近盟书石台时发亮。

日月血牒：晏南枝分段携带，不可一次性解释所有身世。第一季只证明她是旁支血脉和盟书线索持有人。

清明白册：第01集用于验村，第06集出现沈照眠名字，第12集出现在清明院远景。

清明香：白翳追踪工具。第04集沾上沈维桑衣角，第06集导致险些暴露，第11集被地下旧驿潮气短暂压住。

焦痕衣角：沈维桑第02集后保留，作为屠村记忆视觉锚点，第一季不更换成干净英雄服。

## 3. 情感连续性

沈维桑：

1. 第01-03集：失去小家后的震惊和履约本能。
2. 第04-06集：发现父亲、妹妹和旧史被连接，愤怒从私人伤口扩大。
3. 第07-09集：被混裔和北境受害者冲击，开始怀疑单一复国叙事。
4. 第10-12集：接受自己不能只救妹妹，也必须阻止清明籍继续吞掉所有人的来处。

晏南枝：

1. 第01-03集：把沈维桑视为临时护卫，隐藏信息。
2. 第04-06集：为使用血脉线索和隐瞒风险付出信任代价。
3. 第07集：被陆青砾质问后第一次沉默。
4. 第11-12集：承认月下盟书比日月血牒更重要。

白翳：

1. 第01-03集：以清除旧污的监察者姿态出现。
2. 第04-06集：确认沈维桑不是普通村民，追捕升级。
3. 第10集：揭示他处死沈季衡，并把沈照眠作为心理压迫筹码。
4. 第11-12集：发现盟书会破坏清明叙事，选择烧毁主卷。

## 4. 地理连续性

第一季路线：残阳坳 -> 废驿暗道 -> 金河粮磨区 -> 灰烬书院 -> 清明院外署 -> 墙下集市 -> 锁喉关 -> 鸣骨岭 -> 寒鸦堡 -> 月下盟书旧驿 -> 坍塌烽燧。

路线逻辑：二人不是直奔南方，而是被父亲旧驿图和月下盟书线索牵引到北境边墙。季终再转向南方隐歌谷寻找解读完整盟书的人。

## 5. 后续部门注意事项

美术资产部门应优先建立沈维桑、晏南枝、白翳、沈照眠、陆青砾、薛临墙六个稳定角色资产。

导演分镜部门应把每集控制为8到12个镜头单元，每集只设置一个高难度镜头。

编剧部门写对白时应避免大量背景说明，把世界观藏进搜查、道具、旧碑、白册、童谣、交易和追捕动作中。
