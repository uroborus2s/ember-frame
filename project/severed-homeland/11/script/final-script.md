# 第11集：月下不是皇座

时长目标：4分45秒
格式：AI竖屏短剧详细剧本
本集功能：完成“月下盟书不是复国王座”的主题反转，迫使晏南枝承认旧帝国旧债。

## 场景表

1. 月下旧驿入口：石门开启，中央不是皇座。
2. 地下星图厅：血牒启动石台，多族印记显现。
3. 多族印记墙：白翳追至，指出盟书记录昭明旧债。
4. 坍塌出口：旧驿震裂，众人被迫逃向烽燧。

## 成片剧本

### SC01 月下旧驿入口 / 夜 / 5秒钩子

月下旧驿石门轰然打开。晏南枝抬头，里面没有皇座，只有一面刻满陌生印记的断墙。

陆青砾低声。

陆青砾：你家祖宗，把王座弄丢了？

旧白石门上，日月双印裂成两半。沈维桑把半枚日月驿令按入石缝，金赤残纹只亮一段，又迅速暗下。

沈维桑：门还认路，不认人。

晏南枝取出血牒残段，指尖发白。

晏南枝：那就让它认昭明。

血牒贴上月白石台。冷光铺开。断墙上密密刻着日月、鹿角、灰墙、短锤、枝叶、海陆双线和许多残缺手印。

鹿弥走到墙前，摸过鹿角印。

鹿弥：我们守誓派的印，在这里。

陆青砾靠墙看出口。

陆青砾：那无姓人的印呢？

陆青砾用袖口擦墙，露出一块被剜掉的空白掌印，边缘还有虫蜡刮痕。

陆青砾：有人把我们连印都刮走了。

晏南枝没有立刻答，血牒在她指间压出红痕。

远处清明香飘入入口。

沈维桑：白翳到了。

### SC02 地下星图厅 / 连续

星图厅地面有断续金赤路线，像被人一段段掐灭。晏南枝把血牒按在石台中央，墙上的印记依次发亮。

石台铭文不是整句浮出，而是分成几块冷光：边军守墙、鹿角守雪线、短锤修矿道、海陆押粮线、枝叶见证林誓。每一块亮起，背后都有一道黑痕。

晏南枝读到第三块，声音低下去。

晏南枝：这不是召回旧臣的诏令。

鹿弥：也不是你一家的功劳。

陆青砾：也不是给皇血盖章的契书。

晏南枝看向沈维桑。

晏南枝：顾师说，只要找到月下，旧人族就会重新聚到日月旗下。

沈维桑：我娘说，送你来月下，不是送你来复国。

晏南枝：那她要我看什么？

沈维桑看着断墙。

沈维桑：看旧日月欠了谁。也看肃明偷走了谁的名字。

墙上一个残缺海陆印短暂亮起，又熄灭。晏南枝伸手追过去，却只摸到裂痕。

晏南枝：有些印，连名字都没留下。

陆青砾：这句我爱听，也讨厌听。

### SC03 多族印记墙 / 片刻后

白翳走入星图厅，白袍在尘灰中仍干净得刺眼。身后清明院人员封住入口。

白翳：月下盟书，最大旧污证据。很好，省得我继续找。

晏南枝挡在石台前。

晏南枝：这是多族守边承诺，不是旧污。

白翳用虫蜡针点碎一处亮印。亮印背面渗出黑字：征林材、借海核、役工匠、压边军。

晏南枝脸色更白。

白翳：这才是皇血不爱读的半面。

晏南枝：我读。

白翳：读完，你还敢举日月？

鹿弥握紧骨笛。

鹿弥：肃明也没有还债。你们让雪灾南压，让掠粮派和墙军互杀。

白翳平静。

白翳：所以清明籍要抹掉所有旧账。没有记忆，就没有追债。

沈维桑上前，把陆青砾拉到自己身后。

沈维桑：没有记忆，也没有人。

白翳看着他。

白翳：你父亲拒绝皇旗，你也拒绝。可你更糟，你要把这些互相仇恨的人重新叫到一面墙前。

沈维桑：不是叫他们跪旗。是叫他们看见彼此还活着。

陆青砾忽然问晏南枝。

陆青砾：现在回答我。日月旗里，有没有我们的姓？

晏南枝看向她耳后的细鳞痕，看向鹿弥，看向沈维桑被烫伤的手。

晏南枝：若没有，就让它改。

白翳：皇血无权改旧盟。

晏南枝：那我就不以皇血改。

石台震动。多族印记同时亮起又碎裂，旧驿开始坍塌。

### SC04 坍塌出口 / 连续

顶石落下，金赤路线像火星一样断开。入口被清明院堵死，只剩通向旧烽燧的坍塌出口。

薛临墙在外侧喊声传入，模糊而急。

薛临墙：走烽燧！排烟道还通！

沈维桑扶住晏南枝。

晏南枝回头看断墙。

晏南枝：盟书还没读完。

沈维桑：活着才能读下一页。

鹿弥吹响短促骨笛，震落的共鸣让一块坠石偏开。陆青砾割断石台旁红线残缚，露出一页半脱落的盟书残页。

陆青砾：这东西能带走？

晏南枝伸手去取，白翳的虫蜡针从尘中刺来。沈维桑用短刀挡住，左肩被震得发僵。

白翳：带走它，只会带走下一场战争。

沈维桑：那你为什么追得这么急？

白翳第一次沉默半息。

更大的裂声响起。众人被迫冲向烽燧暗道。晏南枝最后看一眼印记墙。

晏南枝：月下不是皇座。

沈维桑：是墙。也是债。

断墙在尘雾中裂开，金赤残光坠向下方。

切黑。
