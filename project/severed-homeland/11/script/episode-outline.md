# 第11集 Outline：月下盟书

1. 钩子：盟书石台显出北境骨笛印。目的：打破复国预期。情绪：震惊。视觉：月白石台、多族印记。
2. 旧驿开启：半枚驿令、月白玉片和背墙图共同点亮星图厅。目的：三条线索汇合。视觉：金赤、月白、石刻。
3. 盟书真相：晏南枝发现盟书不是皇室檄文，而是多族共守誓约。目的：主题反转。情绪：信念动摇。
4. 无姓印记：陆青砾看到墙下混裔祖先作为“墙下民”留印。目的：承认无姓人。情绪：第一次被看见。
5. 薛临墙选择：老墙师决定帮他们守住入口，因为边墙军不该只做虫族尸墙。目的：灰墙军转向。情绪：迟来承诺。
6. 白翳到场：白翳说旧盟只会重启战争，用清明香压制残阵。目的：最终对峙。视觉：白袍入月白厅。
7. 主卷焚毁：白翳烧毁盟书主卷，沈维桑抢下残页。目的：让胜利不完整。情绪：痛与决断。
8. 结尾转折：地下旧驿开始坍塌，沈维桑看见一条通向烽燧的残光路。目的：进入终集逃亡。视觉：光路、落石。
