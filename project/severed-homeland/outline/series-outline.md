# 全剧故事大纲

## 标题

断航故土 / Severed Homeland

## 全剧类型与主题

全剧固定为**低魔东方史诗横越大陆冒险剧**，叠加历史追索、制度悬疑、盟约战争和多方政治博弈。

这里的“穿越冒险”不是时间穿越，而是主角队伍横穿大陆、海域、边墙、旧都、沼泽和北境，在一站一站的冒险中找回被肃明国抹掉的大陆历史。每一季都必须把一个地理区域、一个历史真相和一个当下选择绑在一起：不是坐在屋里讲史，而是在逃亡、交易、潜入、守城、航行、谈判和战争中把历史挖出来。

全剧主题句：

**找回大陆历史，不是为了复辟旧帝国，而是为了让被旧帝国和肃明国共同排除的人，重新获得名字、道路、航权、边界和故土。**

第一季主题句仍固定为：

**日月照耀皆为故土，但故土不再由帝国旗帜单独定义，而由活人之间互相承认、互相守护的承诺定义。**

## 是否需要争霸

全剧需要战争与权力争夺，但不采用传统“主角争霸称帝”的主类型。

采用方案：**盟约战争，而非帝王争霸。**

原因：

1. 如果完全没有争霸，故事后期无法承载白曜城、金河粮仓、东海航权、北境边墙和肃明中枢的世界级冲突。
2. 如果变成主角称王争霸，会破坏“故土不由单一旗帜定义”的核心主题。
3. 最合适的后期压力，是围绕道路、粮仓、白册、港口、边墙阵基、旧都档案和多族盟约展开的战略争夺。

因此，后期确实会出现军队、城池、粮道、港口、边墙和旧都的争夺，但主角目标不是登基，而是拆除清明籍、公开旧史、重开航路、修复边墙灾源，并建立不靠抹名和血统排除维持的“新日月盟约”。

## 全剧叙事引擎

每一季都使用同一套叙事引擎：

1. 一个冒险目的地：主角队伍必须抵达某个被封锁、被误解或被改名的地理区域。
2. 一个历史残片：该区域保存一段大陆历史真相。
3. 一个当下代价：找回真相会让某个活人群体付出风险。
4. 一个价值修正：沈维桑、晏南枝和同伴必须修改原有故土观。
5. 一个更大的追捕或战争后果：真相不能免费带走，下一季压力由此升级。

全剧历史恢复顺序：

1. 月下盟书：故土不是皇族独占，旧帝国也有旧债。
2. 东海断航：航路不是失落神话，而是被焚毁、被封禁、也被旧债污染的大陆边界。
3. 南林与金河源头：昭明旧债不只在史书里，直接影响林地、矿脉、河流和边墙阵基。
4. 虫族起源与白曜城：肃明不是天生文明正统，而是从被流放、被启蒙、再篡夺旧都的历史中成长出来。
5. 北境灾源与新日月盟约：真正要争夺的不是帝位，而是大陆能否停止靠遗忘、饥饿和边墙消耗维持秩序。

## 全剧完整冒险线路

全剧路线是一条从小村出发，穿越大陆后回到旧都的闭环路线：

### 第一段：金河南岸到北境边墙

残阳坳 -> 旧驿暗道 -> 金河粮磨区 -> 灰烬书院 -> 清明院外署 -> 墙下集市 -> 锁喉关 -> 鸣骨岭 -> 寒鸦堡 -> 月下盟书旧驿 -> 坍塌烽燧。

功能：从小家出发，找回月下盟书第一残页，完成“故土不等于皇旗”的第一层反转。

### 第二段：沿金河东下到东海断航线

坍塌烽燧 -> 墙下集市残线 -> 金河北岸军道 -> 白曜城外水门与旧都影子 -> 龙骨三角洲 -> 白浪残港 -> 潮骨群岛 -> 碎帆屿 -> 雾灯礁 -> 潮星灯塔废址。

功能：找回焚帆令真相、旧港籍总册残页和潮星灯塔启动密钥。主角队伍第一次意识到“断航”不是世界自然断裂，而是肃明制度和昭明旧债共同造成的海陆断裂。

### 第三段：由东海折向南林，再逆金河至西部高原

雾灯礁 -> 结界海边缘 -> 南部隐歌谷 -> 月苔林 -> 万雨瀑布 -> 南缘红线驿 -> 金河中游隐渡 -> 赤砧山脉 -> 千锤矿道 -> 禁枪炉 -> 金河源头雪线。

功能：找回南林开拓令、珊瑚灵核案、矮人工程旧约和金河源头阵基。晏南枝必须正面承认昭明对南林、海族和矮人的旧债；沈维桑则发现边墙雪灾与金河源头、旧阵维护和肃明资源抽取有关。

### 第四段：从西部源头回切中央旧都与东北沼泽

金河源头 -> 风门关 -> 金河北岸旧驿干线 -> 白曜城地下旧驿总署 -> 清明院本部外围 -> 书壳院暗库 -> 东北流放旧堡 -> 母巢黑台 -> 芦骨战场。

功能：找回虫族启蒙、东北流放制、肃明建国伪史和清明籍白册总源。白翳的信念被动摇：他相信的“清明新朝”并不是从无污泥中诞生，而是同样继承了昭明的流放、利用和篡改。

### 第五段：北境灾源、边墙修复与白曜城终局

东北沼泽 -> 寒鸦堡旧档线 -> 边墙阵基群 -> 北境冰湖 -> 巨兽迁徙源地 -> 断脊边墙 -> 金河粮仓总闸 -> 白曜城白石台地 -> 日月宫城旧址。

功能：完成北境雪灾真相、边墙阵基修复、清明籍拆除和新盟约公开。终局不是沈维桑称帝，而是白曜城被迫公开旧档，晏南枝放弃单一皇权号令，沈维桑拒绝王座，多族代表共同重签新日月盟约。

## 全剧分季大纲

### 第一季：月下不是皇座

规格：12 集，每集 3分30秒到5分钟。

主类型：逃亡护送 + 制度悬疑 + 边墙冒险。

冒险路线：残阳坳 -> 金河粮磨区 -> 灰烬书院 -> 清明院外署 -> 墙下集市 -> 锁喉关 -> 鸣骨岭 -> 寒鸦堡 -> 月下盟书旧驿。

历史真相：月下盟书不是皇族复国凭证，而是多族共同守边的残缺旧誓，并记录昭明旧债。

主冲突：沈维桑只想救妹妹和完成母亲遗言；晏南枝想证明皇血能召回旧人族；白翳要把旧名、旧路和旧血脉清除为“旧污”。

季终：沈维桑放弃完整血牒，先救活人，再带走盟书残页。沈照眠成为清明院识别者，白翳准备利用妹妹追捕哥哥。

主题完成：故土不能只属于旧姓、皇旗和血牒。

### 第二季：东海有灯

主类型：断航冒险 + 海陆悬疑 + 追捕救亲。

冒险路线：坍塌烽燧 -> 金河北岸军道 -> 白曜城外水门 -> 龙骨三角洲 -> 白浪残港 -> 东海岛链 -> 潮星灯塔废址。

历史真相：焚帆令不仅焚毁船坞，也焚毁港籍、潮星灯塔和结界海星图；东海岛链人族不是海匪后代，而是被肃明断籍的旧港遗民。但昭明旧帝国也曾强征海族珊瑚灵核，东海不会因为日月旗自动回归。

主冲突：沈照眠被清明院带上追捕线，能准确读出沈维桑的旧驿习惯；东海遗民不信中陆人，海族不信日月旗，白翳试图把护送队塑造成破坏海禁的旧污海匪。

季中转折：主角队伍夺回一段港籍总册残页，却发现上面同时记录肃明焚港罪证和昭明海债。

季终：潮星灯塔被短暂点亮，结界海出现一条窄航道。沈照眠短暂恢复对哥哥的身体记忆，却仍被白册残锁牵制。东海愿意提供航路，但要求未来偿还旧海债。

主题完成：故土不能没有航路；但重开航路必须承认海上的债。

### 第三季：金河源头

主类型：南林旧债 + 西部工程冒险 + 盟友裂缝。

冒险路线：东海雾航线 -> 南部隐歌谷 -> 月苔林 -> 万雨瀑布 -> 南缘红线驿 -> 金河中游隐渡 -> 赤砧山脉 -> 千锤矿道 -> 禁枪炉 -> 金河源头雪线。

历史真相：南林开拓令、珊瑚灵核案、矮人工程旧约和边墙阵基失修彼此相连。昭明曾用林材、海族灵核、矮人工程和金河源头阵基支撑帝国扩张；肃明夺国后继承这些工程，却只抽取资源，不维护平衡。

主冲突：南部精灵不愿为昭明血脉再开林门；矮人不愿公开火器和阵基图；复翼者要求晏南枝借机称旗；沈维桑反对把新故土变成旧帝国复辟。

季中转折：晏南枝主动承认南林和海族旧债，失去一批复翼者支持，却赢得精灵见证者和矮人工程师的有限信任。

季终：主角队伍取得金河源头阵基图和禁枪炉残械，但发现肃明正把源头水脉与边墙阵基反向抽取，用来维持白曜城白册总源。北境雪灾因此继续恶化。

主题完成：故土不是只修复人族记忆，也要修复被帝国工程伤害的土地、河流和盟友。

### 第四季：白曜旧都

主类型：旧都潜入 + 政治暗战 + 虫族起源真相。

冒险路线：金河源头 -> 金河北岸旧驿干线 -> 白曜城地下旧驿总署 -> 清明院本部外围 -> 书壳院暗库 -> 东北流放旧堡 -> 母巢黑台 -> 芦骨战场。

历史真相：肃明国并非天然文明正统。虫族早期从东北沼泽、流放贵族、人族制度和饥荒中学习文字、税粮、军阵和国家形式。肃明对昭明的控诉有一部分真实伤口，但它把伤口变成了抹名、奴役和清除活人的制度。

主冲突：人族阵营内部开始分裂：复翼者要趁机攻白曜城、恢复皇旗；新血派要求先拆白册、救奴兵和混裔；灰墙军只愿意保护粮道和边墙；东海与南林要求偿债。肃明内部也分裂：清明院要彻底抹名，甲军府要军管大陆，部分虫族旧派不愿继续为白曜城中枢牺牲。

季中转折：白翳在书壳院暗库中看见肃明早期虫族也曾被昭明贵族当作工具，并发现清明籍正在吞噬虫族自身的旧记忆。他的信念开始裂开，但他仍不能接受沈维桑的混杂盟约。

季终：护送队公开部分肃明建国伪史，白曜城爆发白册动摇和奴兵哗变。清明院启动白册总源，准备把中央大陆所有旧名反应一次性清除。盟约战争正式爆发。

主题完成：找回历史不是证明某一族无罪，而是阻止所有统治者继续靠伪史和抹名制造秩序。

### 第五季：新日月盟约

主类型：盟约战争 + 北境灾源 + 旧都终局。

冒险路线：东北沼泽 -> 寒鸦堡旧档线 -> 边墙阵基群 -> 北境冰湖 -> 巨兽迁徙源地 -> 断脊边墙 -> 金河粮仓总闸 -> 白曜城白石台地 -> 日月宫城旧址。

历史真相：北境雪灾不是单一神罚，也不是北境诸族贪婪南侵，而是昭明末期边墙阵基失修、肃明长期资源抽取、金河源头反向压榨和巨兽迁徙链崩坏共同造成。边墙最初不是分割敌我的墙，而是多族共同调节灾变的工程。

主冲突：清明院要用白册总源完成最后清除；甲军府要以边墙战争接管大陆；复翼者中激进派要抢先称帝；北境掠粮派要趁乱破墙；东海、南林、西部和新血派都不愿再接受单一中心命令。

季中转折：沈照眠反向利用识别者能力读出白册总源的旧名结构，帮沈维桑找到拆册路径。白翳承认清明籍不能带来无伤口的和平，但他必须为自己制造的伤口付出代价。

终局高潮：白曜城争夺战不是夺位战，而是档案、粮仓、水门、白册总源和日月宫城旧址的多线争夺。沈维桑选择先开粮仓、救奴兵和稳定金河总闸，而不是冲上王座；晏南枝公开放弃“皇血自动号令天下”，以昭明残脉身份承认旧债；陆青砾代表无姓混裔写入新盟；薛临墙代表灰墙军要求边墙不再是尸墙；鹿弥代表北境守誓派要求修复迁徙线；东海、南林、矮人与部分虫族见证新约。

最终状态：肃明旧制度崩塌，但大陆没有立刻进入乌托邦。白曜城不再是单一皇都，而成为公开旧档和重签盟约的危险中心；金河粮仓、东海航路、南林边界、西部矿道和北境边墙都进入新约监督。沈维桑拒绝称王，继续作为旧驿引路人和新盟约执行者；晏南枝不再只是皇血，而是旧债承认者；沈照眠从识别旧污的孩子变成识别旧名、归还旧名的人。

主题完成：日月照耀皆为故土，最终不再是一句帝国宣言，而是一套必须持续偿债、公开历史、保护差异、共同守边的活人盟约。

## 主要角色全剧弧线

沈维桑：小家守护者 -> 护送引路人 -> 旧史拼接者 -> 新盟约拒王者。他始终不适合称帝，适合把路打开、把人带到彼此面前。

晏南枝：皇血逃亡者 -> 旧债承认者 -> 盟约见证者 -> 放弃单一皇权的昭明残脉。她的胜利不是登基，而是让日月旗不再只照旧姓。

沈照眠：被救妹妹 -> 清明院识别者 -> 白册裂缝 -> 旧名归还者。她的能力从追捕哥哥的工具，转为拆除清明籍的关键。

陆青砾：墙下交易者 -> 新血派证人 -> 无姓人组织者 -> 新盟约签名者。她保证故事不会滑回皇族和旧臣叙事。

白翳：清除旧污的监察使 -> 理性追捕者 -> 信念裂开者 -> 清明籍代价承担者。他不需要被洗白，但必须让观众理解制度为何能自称善。

薛临墙：灰墙军生存主义者 -> 边墙旧债证人 -> 守墙军代表 -> 边墙重订者。他让战争线始终有粮、尸体、寒冷和现实代价。

鹿弥：北境伤者 -> 寻灾源证人 -> 守誓派联络者 -> 北境迁徙线代表。她确保北境不是最终敌人，而是灾变共同体的一部分。

## 全剧终局原则

1. 不以主角称帝结尾。
2. 不以昭明简单复国结尾。
3. 不以肃明虫族全灭结尾。
4. 不让所有族群立刻和解。
5. 必须拆除清明籍、公开旧档、重开有限航路、修复边墙阵基并重签新日月盟约。
6. 终局应保留治理难题：旧债如何偿还、粮仓如何分配、混裔如何入名、虫族平民如何脱离肃明旧制度、北境迁徙线如何恢复。

## 第一季制作约束

第一季仍按现有 12 集短剧规格执行。它只打开完整故事的第一扇门，不提前展开东海、南林、西部高原、东北沼泽和白曜城主线。

第一季每集节奏继续遵守：5秒钩子，30秒危机，中段升级，结尾反转或悬念。每集主要角色控制在 3-5 人，主要场景控制在 2-4 个。

第一季分集详见 `outline/episode-outline-index.md`。
