# Showrunner 备选方案记录

## 来源

`showrunner-agent` 在主方案落盘后返回了一版备选第一季方案，题名为《断乡：残阳驿火》。

## 备选方案核心差异

备选方案同样采用12集AI竖屏短剧、单集2分30秒到3分钟，但中段路线不同：

1. 第1-6集仍从残阳坳屠村推进到灰烬书院、清明院识别者和妹妹线。
2. 第7-8集提前进入南部精灵、月苔林、隐歌谷和南方隐歌仪式。
3. 第9-12集再转入边墙、锁喉关、寒鸦堡和月下盟书。

备选方案的优点：

1. 更早展示南林、精灵和隐歌谷，世界版图更丰富。
2. 晏南枝的南方来源能在第一季中段得到可见场景支撑。
3. 月下盟书坐标由南方隐歌仪式解出，推理链更外显。

备选方案的风险：

1. 第一季地理跨度过大，会从残阳坳、金河、灰烬书院、清明院、南林、隐歌谷一路跳到北境边墙，AI短剧制作和观众记忆负担更高。
2. 精灵和南林旧债会抢占第一季中段篇幅，削弱“残阳坳到边墙旧驿”这条护送主线的压迫感。
3. 当前主方案已将南林、隐歌谷设为第二季接口；若提前使用，会减少后续季的展开空间。

## 当前决策

主方案暂不替换。第一季继续采用“残阳坳 -> 旧驿道 -> 金河粮磨 -> 灰烬书院 -> 清明院外署 -> 墙下集市 -> 锁喉关 -> 鸣骨岭 -> 寒鸦堡 -> 月下盟书旧驿”的集中路线。

备选方案可作为后续改版方向：

1. 如果平台要求第一季更快展示大世界观，可把第7-8集替换为南林与隐歌谷。
2. 如果想强化晏南枝的南方旧帝国残脉身份，可提前加入澜枝、顾怀章和隐歌仪式。
3. 如果要保持AI制作可控，建议仍把南林放到第二季开场。
