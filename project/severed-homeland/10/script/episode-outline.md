# 第10集 Outline：寒鸦堡暗格

1. 钩子：墙内档案露出沈季衡处死记录。目的：父亲真相实锤。情绪：震怒。视觉：虫蜡裂开、旧档。
2. 潜入寒鸦堡：陆青砾带路，晏南枝用礼制判断军府动线。目的：进入反派空间。情绪：紧张。视觉：虫膜走廊。
3. 甲清分歧：厉螳与白翳争执，是杀逃犯还是活捉样本。目的：反派内部裂缝。情绪：冷压。视觉：黑甲与白袍对照。
4. 暗格发现：沈维桑找到父亲档案和旧驿背墙图。目的：私人伤口与主线合一。情绪：愤怒失控。
5. 审讯厅遭遇：白翳截住沈维桑，承认处死沈季衡但称其“拒绝清明”。目的：主反派正面对撞。情绪：价值对峙。
6. 逃出军府：晏南枝和陆青砾切断虫膜门，沈维桑带图脱身。目的：动作脱险。视觉：虫膜撕裂、短刀。
7. 结尾转折：白翳提到沈照眠已能识别旧驿暗语。目的：最大心理压迫。视觉：清明白册中妹妹画像。
