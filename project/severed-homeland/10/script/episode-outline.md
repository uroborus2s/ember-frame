# 第10集 Episode Outline：寒鸦堡旧档

## 结构节拍

### 1. 5秒钩子

虫蜡档案墙传出沈季衡临死前的审讯记录：月下路，不能交给任何皇旗。

### 2. 30秒危机

厉螳封锁寒鸦堡，白翳接管旧档，护送队被困膜廊。

### 3. 中段升级

沈维桑得知父亲因藏匿月下地图被处死，薛临墙承认曾沉默换粮。

### 4. 结尾悬念

沈照眠被送到寒鸦堡远程识别旧驿暗号，白翳发现她能读出沈维桑路线习惯。

## 分场方向

- 主要角色：沈维桑、晏南枝、白翳、厉螳、薛临墙、沈照眠
- 主要场景：虫蜡档案墙、寒鸦堡膜廊、审讯厅外侧、远识白册台
- 动作要求：所有重大信息必须落在可见动作、道具、空间选择或对白冲突上。
- 节奏要求：开场立刻进入危险或问题，中段用追捕、交易、审讯、救援或空间坍塌升级，结尾必须保留下一集推动力。

## 角色推进

本集推进由“眼前求生”转向“故土定义被不断扩展”。沈维桑每次选择都优先保护活人和路线；晏南枝每次遭遇都削弱单一血统复国逻辑；白翳和清明院的压力必须持续把旧名、旧路和亲情变成可追捕证据。

## 连续性锚点

- 半枚日月驿令、血牒残段、清明香、白册、虫蜡针、旧驿暗语和月下多族印记均按出现顺序递进。
- 沈照眠第一季不得真正团圆，只能以识别者伤口和追捕线推动后续。
- 月下盟书不得写成皇族复国诏书。
