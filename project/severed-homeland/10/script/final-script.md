# 第10集：寒鸦堡旧档

时长目标：4分40秒
格式：AI竖屏短剧详细剧本
本集功能：揭示沈季衡死因，压实薛临墙沉默换粮旧债，并让沈照眠读出哥哥路线习惯。

## 场景表

1. 虫蜡档案墙：沈季衡审讯记录响起。
2. 寒鸦堡膜廊：厉螳封堡，白翳接管旧档。
3. 审讯厅外侧：父亲真相与薛临墙旧债。
4. 远识白册台：沈照眠成为追捕线核心。

## 成片剧本

### SC01 寒鸦堡虫蜡档案墙 / 暗白冷光 / 5秒钩子

虫蜡档案墙里，一道沙哑男声响起。

沈季衡录音：月下路，不能交给任何皇旗。

沈维桑僵住。

沈维桑：爹？

墙里浮着断裂审讯字条、旧测绘符号和被封存的声音。沈维桑伸手碰墙，虫蜡立刻烫伤指尖。他没有退。

沈维桑：他们说你死在山洪里。

档案墙浮出清明院判词：藏匿月下地图，拒交皇旗归属，处决。

晏南枝站在他身后，眼神动摇。

晏南枝：他不是替旧旗送命。

沈维桑：他是因为不让路变成旗死的。

堡外铁闸轰然落下。

### SC02 寒鸦堡膜廊 / 连续

黑石堡墙内覆着半透明虫膜，像旧人族堡垒被白蜡吞住。厉螳的铁闸令牌敲地，甲军虫兵封住出口。

厉螳：寒鸦堡闭门。旧档、旧人、旧路，一个都不许出。

白翳从膜廊另一端走来，白袍无尘，高筒官帽后细长虫须微动。

白翳：军府只会把伤口砍大。旧档交清明院。

厉螳刃肢弹出半寸。

厉螳：你审了八年，墙外还是在饿。

白翳擦拭虫蜡针。

白翳：所以更不能让他们记起，自己曾经有路可走。

沈维桑等人藏在膜廊暗格后。陆青砾压低声音。

陆青砾：两边互咬，我们是不是该道声谢？

薛临墙粗声：别笑。一个要你死，一个要你忘了自己死过。

晏南枝看向薛临墙。

晏南枝：你知道这里藏着沈季衡的旧档？

薛临墙沉默。

### SC03 审讯厅外侧 / 稍后

护送队沿旧墙裂缝潜到审讯厅外。虫蜡档案墙再次亮起，沈季衡的身影模糊出现：偏瘦、衣摆带路灰，手指比划方向。

清明院审讯声：交出月下地图，交给昭明残脉，沈家疑名可销。

沈季衡录音：地图不归皇旗。谁守这条路，才归谁。

沈季衡录音：月饼藏信，藏的是出路，不是玉玺。

清明院审讯声：昭明已亡。

沈季衡录音：那更不能让下一面旗，拿它去称王。

沈维桑呼吸变重，左肩旧伤发僵。

沈维桑：他们拿沈家威胁他？

薛临墙终于开口。

薛临墙掰开旧甲片，里面夹着一枚发黑粮牌，边角刻着“三日”。

薛临墙：寒鸦堡断粮。清明院把三日粮牌放我手里，说我只要装作没听见沈季衡这个名字，粮门就开三日。

晏南枝：你拿了？

薛临墙把粮牌按回甲片，日月压痕硌得掌心出血。

薛临墙：我没拿。也没摔。三百人吃了三日。第四日，照样啃皮带。

沈维桑看着他，眼里有怒，却没有拔刀。

沈维桑：所以我爹死了，墙还是饿。

薛临墙：是。

白翳的声音从厅门外传来。

白翳：沈季衡至少明白一件事，皇旗会让旧路再起战火。

白翳走入，平静看向沈维桑。

白翳：你比你父亲更危险。他不把路交给皇旗，你却想把路交给所有册外人。

沈维桑：你怕的不是战争。你怕他们互相认得。

白翳：一认亲，就会翻旧账；旧账一翻，就会开战。

晏南枝上前半步。

晏南枝：忘了，就不是伤口？

白翳看她。

白翳：忘了，至少不会把孩子送上下一场旗战。

沈维桑：你已经把我妹妹送上来了。

白翳没有提高音量。

白翳：正因如此，她会比谁都明白，除根有多要紧。

厉螳在外撞开铁闸，刃肢扫过旧墙。沈维桑拉起晏南枝，薛临墙用长枪顶住塌落石门，陆青砾割断虫膜标记绳。

薛临墙：废话够了，去月下！

### SC04 远识白册台 / 同时

寒鸦堡另一侧，一方白册台前，沈照眠穿白色儿童袍，腕上虫蜡识别环微亮。她面前摆着几枚旧驿暗号碎片。

白册官吏：读。

白册官吏把三枚旧驿碎片推到她面前：亮雪、背阴石缝、烽燧排烟道。

沈照眠先把“亮雪”推远，指尖停在烧焦发绳上。

沈照眠：哥哥不会走亮处。雪会把人影还给追兵。

白册官吏一怔。

沈照眠的眼神短暂失焦，手指摸向烧焦发绳。

沈照眠：哥哥说，草乌不能碰雪水......

白翳走入，停在她身旁。

白翳：继续。

沈照眠被压回平静。

沈照眠：他会去月下旧驿。若入口坍，他会走烽燧排烟道。

白翳看向寒鸦堡外的雪。

白翳：很好。从今天起，沈维桑走哪条路，由沈照眠来读。

远处，沈维桑一行冲出寒鸦堡侧门，寒鸦群惊飞，月下方向的雪线亮起微弱金赤残纹。

切黑。
