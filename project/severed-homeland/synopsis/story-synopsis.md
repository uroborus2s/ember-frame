# 全剧梗概

肃明历1226年，虫族肃明国统治中央大陆已进入第一千二百二十六个年头。旧人族昭明帝国被污称为前明，日月旗被抹去，白石旧都被占，龙船船坞被焚，东海断航，边墙人族被困在五百年血战中，中央村落则被清明籍、征粮和遗忘术一点点剥掉旧名。

残阳坳猎户沈维桑只想守住母亲罗青禾和妹妹沈照眠。他不信复国，不信血脉，也不信任何宏大旗帜。清明院监察使白翳追捕来自南方的旧帝国血脉晏南枝，搜查残阳坳时发现旧驿暗格，将村落定为旧污窝藏点。火场中，罗青禾临死把半枚日月驿令交给沈维桑，要求他护送晏南枝去月下，而不是送她去复国。沈照眠则被清明院押走，后来被训练成能识别旧暗语的白衣识别者。

第一季，沈维桑和晏南枝沿旧驿残段北上，从金河南岸的小村走到北境边墙背后的月下盟书旧驿。他们在金河粮磨区看见肃明如何保留生产蒸汽却毁掉战争能力，在灰烬书院看见历史如何被焚毁又藏进儿童歌声，在清明院外署看见沈照眠被制度改造成追捕工具。墙下混裔陆青砾、灰墙军薛临墙和北境鹿族萨满鹿弥不断逼问晏南枝：日月旗里是否有无姓人、守墙人和北境灾民的位置。季终，月下盟书证明旧帝国末期曾有多族共同守边的残缺承诺，也记录昭明旧债。沈维桑放弃完整血牒，先救活人，再带走盟书残页。

第二季，护送队沿金河东下，靠近白曜城外水门与龙骨三角洲，再进入东海岛链。他们追查焚帆令、港籍总册、潮星灯塔和结界海断航真相。东海遗民不是海匪后代，而是被肃明断籍的旧港人；但海族也记得昭明曾强征珊瑚灵核。白翳利用沈照眠追读沈维桑旧驿习惯，把护送队逼成海禁通缉犯。季终，潮星灯塔短暂重亮，结界海出现窄航道，沈照眠短暂恢复对哥哥的身体记忆，却仍被白册残锁牵制。

第三季，队伍由东海折向南部丛林，再逆金河进入西部高原。南部精灵要求晏南枝承认南林开拓旧债，矮人要求主角方承认昭明曾利用工程师与矿道支撑扩张。沈维桑发现北境雪灾、金河源头阵基、矮人工程和肃明资源抽取互相牵连。晏南枝失去部分复翼者支持，却赢得精灵见证者和矮人工程师的有限信任。季终，队伍取得金河源头阵基图和禁枪炉残械，确认肃明正在抽取源头水脉维持白曜城白册总源。

第四季，队伍从西部源头回切中央旧都与东北沼泽，潜入白曜城地下旧驿总署、清明院本部外围和书壳院暗库，再追到东北流放旧堡、母巢黑台和芦骨战场。他们找回虫族启蒙、东北流放制和肃明建国伪史：肃明不是天然文明正统，而是从昭明流放制度、虫族饥荒、人族文字和国家机器中成长出来。人族阵营内部也爆发分裂，复翼者要复辟称旗，新血派要先拆白册救无姓人，灰墙军只关心粮道和边墙。季终，护送队公开部分肃明伪史，白曜城白册动摇，清明院启动白册总源，盟约战争正式爆发。

第五季，故事进入北境灾源、边墙修复与白曜城终局。队伍从东北沼泽回到边墙阵基群，深入北境冰湖与巨兽迁徙源地，确认雪灾不是单一神罚，也不是北境诸族贪婪南侵，而是昭明末期失修、肃明长期抽取、金河源头失衡和巨兽迁徙链崩坏共同造成。终局的白曜城争夺战不是夺位战，而是档案、粮仓、水门、白册总源和日月宫城旧址的多线争夺。沈照眠反向利用识别者能力拆解白册旧名结构；晏南枝公开放弃皇血自动号令天下；沈维桑拒绝称王，选择开粮仓、救奴兵、稳住金河总闸，把各族代表带到旧都白石台地上重签新日月盟约。

全剧最终不以昭明简单复国、主角称帝或虫族全灭结尾。肃明旧制度崩塌，清明籍被拆，旧档被公开，东海窄航路重开，边墙阵基开始修复，金河粮仓进入新约监督。白曜城不再是单一皇都，而成为公开旧史、偿还旧债和持续争论的危险中心。日月照耀皆为故土，最终不再是一句帝国宣言，而是一套必须持续偿债、公开历史、保护差异、共同守边的活人盟约。
