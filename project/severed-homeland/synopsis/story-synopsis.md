# 《断航故土》故事梗概

肃明国统治中央大陆一千二百年后，人族旧帝国昭明只剩残名、禁歌、旧驿暗语和被虫蜡封住的碑文。虫族政权宣称自己继承文明正统，以“肃尽前明，还天下清明”为国策，推行清明籍，筛查旧血脉、禁私学、禁航权和旧日月符号。

残阳坳猎户沈维桑一直以为自己只需要守住母亲、妹妹和村口山路。直到白袍监察使白翳带着虫族监察队进村，搜查一名来自南方的旧帝国血脉逃犯。清明香封路，虫蜡针验血，旧井暗格暴露，残阳坳被定为“旧污窝藏点”。火场中，沈维桑救出逃犯晏南枝，母亲临死前把半枚日月驿令交给他，要求他护送晏南枝去找“月下盟书”。

晏南枝相信自己身上的日月血牒能重新凝聚旧帝国遗民。沈维桑却不信皇室、不信复国，也不信任何把活人推向死亡的旗帜。他护送晏南枝，只因为答应过母亲，也因为妹妹沈照眠在屠村中失踪。

两人沿旧帝国驿道逃亡。金河粮磨区让他们看见肃明国保留生产蒸汽，却磨平军械刻度；灰烬书院让他们听见被焚毁的旧史还藏在儿童禁歌里；清明院外署让沈维桑发现妹妹被送去训练成识别旧帝国暗语的工具；墙下混裔集市则让晏南枝第一次面对一个问题：若旧帝国回来，那些混血、失籍、军户、海匪和奴民算不算故土的一部分？

逃亡进入绝境山脉后，沈维桑看见边墙的双重真相。它确实抵御北境寒灾和巨兽南下，却也被肃明国改造成消耗人族军户的牢笼。老墙师薛临墙教他，边墙武脉不是英雄功法，而是人在被夺走火枪、大炮和法术后，为活下去付出的身体代价。北境萨满鹿弥又让他明白，边墙外的“敌人”同样是寒灾受害者。

寒鸦堡暗格中，沈维桑查到父亲沈季衡并非死于山洪，而是因保存旧驿密图被白翳处死。白翳告诉他，沈照眠已经在清明院学会识别旧驿暗语，未来会亲手把哥哥从人群中认出来。

最终，沈维桑、晏南枝和临时盟友陆青砾进入边墙地下旧驿，找到残缺的月下盟书。它不是复国檄文，而是旧帝国末期人族、边墙军、北境部族、矮人和其他势力共同守护土地的誓约。晏南枝意识到，日月旗若只代表皇室血统，就无法召回真正失散的故土。沈维桑也意识到，只守住眼前几个人已经不够，因为清明籍会把所有人的来处逐一抹掉。

白翳烧毁盟书主卷，试图证明记忆只要被清除，就不会再制造战争。沈维桑以半枚驿令唤醒旧驿残阵，在坍塌烽燧中带着盟书残页突围。第一季结尾，他和晏南枝转向南方隐歌谷，寻找能读完整盟书的人；清明院白墙内，沈照眠睁开眼，念出哥哥小时候教她的旧驿暗语。

第一季完成的不是复国，而是立誓：故土不能再由单一帝国旗帜、虫族白册或纯血谱系定义。它必须由被历史切碎的人重新承认彼此。
