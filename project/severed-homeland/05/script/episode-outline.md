# 第05集 Outline：灰烬书院

1. 钩子：儿童禁歌响起，碎星图盘转动。目的：让失史可听可见。情绪：诡异。视觉：灰烬、月白光。
2. 入院：孟归藏说明这里曾是人族书库，现在只剩虫蜡封条。目的：场景建立。情绪：沉重。视觉：焦黑拱廊。
3. 月白共鸣：晏南枝玉片贴近星图，显出“月下”残字。目的：推进盟书线。情绪：希望。视觉：玉片、星点。
4. 识别者埋伏：儿童背出旧驿暗语，引沈维桑残纹显现。目的：清明院能力展示。情绪：惊险。视觉：白衣儿童、虫蜡识别环。
5. 晏沈冲突：晏南枝想带走残页，沈维桑先问妹妹下落。目的：私人目标与大叙事冲突。情绪：压抑争执。
6. 残库逃脱：识别者引来巡查，孟归藏烧掉假残页掩护。目的：动作升级。视觉：火星、书灰。
7. 结尾转折：白册残页写出“沈照眠”，沈维桑第一次失控。目的：进入第06集清明院外署。视觉：黑字浮现。
