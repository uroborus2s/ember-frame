# 第05集：灰烬书院的孩子

时长目标：4分35秒
格式：AI竖屏短剧详细剧本
本集功能：把灭史、禁歌、焚帆令伏笔和沈照眠被训练为识别者的真相落地。

## 场景表

1. 焦黑书架后：白衣儿童哼旧驿暗号。
2. 灰烬书院拱廊：清明院封锁筛歌。
3. 星图台残座：孟归藏救人并展示残页。
4. 儿童识别角落：沈照眠下落。

## 成片剧本

### SC01 灰烬书院焦黑书架 / 夜 / 5秒钩子

焦黑书架后，一个白衣儿童抱着膝，轻轻哼出沈家旧驿暗号。

沈维桑本能停步。

白衣儿童：日月重明，雪夜开门。

晏南枝脸色发白。

晏南枝：又是这句。

沈维桑按住短刀，没有靠近孩子。

沈维桑：照眠不是从书里学的。谁教你？

孩子抬起脸，腕上有虫蜡识别环。

白衣儿童：白册教的。它说，谁听见这句会抬头，谁家就有旧根。

他忽然捂住自己的嘴，像才想起这句话会害人。

白衣儿童：我不叫，他们会让我再学一遍。

沈维桑：谁在背后听？

孩子看向书架缝。缝里一粒清明香灰亮了一下。

远处清明香铃响起。

### SC02 灰烬书院拱廊 / 连续

焦黑拱廊被虫蜡封条切成一段一段。白册官吏让几个白衣儿童站成一排，逐句哼禁歌。奴兵藏在拱廊阴影里，等谁有反应就扣腕。

白册官吏：旧歌不用唱完。半句就够认祖，半句就够拿人。

沈维桑拉着晏南枝贴墙后退。晏南枝看着那些孩子，手指攥住血牒。

晏南枝：他们用孩子筛旧人？

沈维桑：你们南方不知道？

晏南枝没有回答。她想走向孩子，被沈维桑按住。

沈维桑：你现在拿血牒出去，只会让他们多筛一个皇族。

一枚骨片从黑暗中飞来，打灭拱廊尽头的小灯。

孟归藏从塌书架后探身。

孟归藏：跟我走。脚别踩白灰，白灰里有香粉。

### SC03 星图台残座 / 片刻后

三人躲入半塌星图台。台面碎裂，能看见东海星图一角、焚帆令残页和被烧黑的旧军械铭牌。

孟归藏背着焦边书箱，半边眉毛被火燎过。他不向晏南枝行礼，只把残页压在石上。

孟归藏：别亮血牒。书院被烧的时候，最先喊复国的人，烧得最快。

晏南枝：你知道我是谁？

孟归藏：知道你金贵。金贵人最容易把真相当成官印，往哪儿一盖就算数。

孟归藏把三样东西摊开：磨平的铭牌、烧焦的航图、写着童籍的白签。

孟归藏：先折兵器，再烧路，最后教孩子认人。肃明怕的不是一个皇姓，是百家还知道往哪儿聚。

孟归藏：他们怕的，是月饼里还夹着字，白灰底下还藏着火。

沈维桑指向一块小白签。

沈维桑：这是什么？

孟归藏：识别童的转运签。残阳坳后，有个女童被押去清明院外署。

沈维桑的手僵住。

晏南枝低声：沈照眠。

孟归藏看了她一眼。

孟归藏：你既记得她名字，就别把她当檄文里的一笔血债。

### SC04 儿童识别角落 / 黎明前

逃离前，沈维桑又看见那个白衣儿童。孩子被奴兵带回识别角落，嘴里还机械地哼着旧歌。

沈维桑把一小包草药塞到书架缝里，悄悄推给他。

白衣儿童：白册说，给东西的是旧污。

沈维桑：白册还说孩子不饿吗？

孩子愣住，迅速把草药包藏进袖里。

晏南枝望着这一幕，终于把血牒收回衣内。

晏南枝：如果月下盟书是真的，它该先救这些孩子。

沈维桑：不是盟书救。是拿着盟书的人做选择。

孟归藏把焚帆令残页撕下一角交给晏南枝，又把转运签交给沈维桑。

孟归藏：一个去东海以后用。一个去外署现在用。都不完整，别问我要整页。

清明院士兵发现暗门，虫族小兵钩镰切开焦书架。

孟归藏：走北廊。记住，灰烬书院没有死，它只是被迫小声。

三人冲入残廊。身后，白衣儿童忽然把禁歌唱错一个音，诱使奴兵看向另一侧。

白册官吏：重背。

白衣儿童：旧歌不必记完......唱错半句，也能让他们抓错人。

他把最后一个音压低，奴兵误向另一侧。

沈维桑回头一瞬。

孩子没有看他，只低头背白册条目。

切黑。
