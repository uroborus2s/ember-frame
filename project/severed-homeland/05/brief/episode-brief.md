# 第05集 Brief：灰烬书院

时长：3分钟。

前5秒钩子：焦黑书架之间传来儿童禁歌，碎裂星图盘自动转动半圈。

出场人物：沈维桑、晏南枝、孟归藏、识别者儿童。

新增设定：焚书旧案、锁翼咒残迹、禁歌保存历史、清明院识别者。

主要场景：灰烬书院、碎裂星图台、书院地下残库。

故事梗概：孟归藏带二人进入灰烬书院寻找月下盟书线索。晏南枝用月白玉片触发碎星图，听见南方隐歌的一段旋律。一个清明院识别者儿童埋伏在残库里，能识别旧帝国暗语并诱发沈维桑灵翼残纹。沈维桑得知清明院会把记忆力强的孩子训练成追捕工具。

结尾钩子：识别者儿童在白册残页上写下“沈照眠”三个字。

制作重点：灰烬、残书、星图光点和儿童声音；不要让解谜变成大段解释。
