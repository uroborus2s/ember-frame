# 第12集：日月照着谁

时长目标：4分50秒
格式：AI竖屏短剧详细剧本
本集功能：完成第一季价值选择：沈维桑先救活人，晏南枝改写日月宣言，沈照眠成为第二季追捕钩子。

## 场景表

1. 坍塌烽燧内：白翳把日月旗布按进雪泥。
2. 断塔石阶：盟书残页、陆青砾、晏南枝和出口不能同时保住。
3. 雪坡逃亡线：沈维桑先救活人，再撕下残页。
4. 清明院远景：沈照眠读出哥哥下一条旧驿暗语。

## 成片剧本

### SC01 坍塌烽燧内 / 暴雪 / 5秒钩子

暴雪灌进断塔。白翳弯身，把磨损朱赤日月旗布按进雪泥。

白翳：记忆断了，仗也就断了。

旧驿出口连着断裂烽燧。塔内石阶开裂，金赤残纹像将熄的炭火。外面是雪坡，里面是塌落的月下旧驿。

晏南枝扶着石壁站稳，脸上有尘和血。

晏南枝：你怕的不是这块布，是有人还记得，它曾照过哪些人。

白翳：我怕的是有人又说，只能照自己。

塔身猛震。陆青砾被断阶卡住，腰间绳扣被石缝压死。另一侧，盟书残页挂在断墙边缘。出口处雪坡正在塌滑。

鹿弥在后方吹骨笛稳住惊乱兽影，但声音越来越弱。

沈维桑扫过三处：陆青砾，残页，出口。

断阶再塌一掌宽，陆青砾的绳扣被压得发响；残页外缘已被雪水浸黑；出口风声忽然变细。

沈维桑：选错一个，剩下两个也没了。

沈维桑：来不及全拿。

晏南枝看向血牒残段。血牒能压住石台裂缝，换一点时间，但会被落石吞掉。

晏南枝：血牒留下。

沈维桑：那是你证明身份的东西。

晏南枝：若这身份只能换一块石头开门，不如换一个活人出去。

### SC02 断塔石阶 / 连续

陆青砾咬牙想割断自己外袄。

陆青砾：别管我，拿那张纸。纸比没户籍的人值钱，我懂行情。

沈维桑冲过去，把短刀塞给她。

沈维桑：你欠我的路钱还没结。

陆青砾愣住，随即笑骂。

陆青砾：这时候讨债，真像活人。

沈维桑用残缺明劲顶开压石，左肩旧伤发出闷响。他脸色一白，仍把陆青砾拽出。

白翳的虫蜡针刺向盟书残页。

白翳：残缺的证据，只会招来残缺的人，继续互相寻仇。

晏南枝挡住他，血牒残段贴在裂缝上。石台残光反冲，血牒开始碎裂。

晏南枝：日月照耀皆为故土。

白翳：昭明旧话。

晏南枝将碎裂血牒按进雪泥，朱赤血线不再指向自己，而是流向被剜空的盟书边角。

晏南枝：今日先照那些被刮掉的名字。

陆青砾：还有没姓的。

鹿弥：还有雪线外的。

沈维桑：还有守门饿死的。

晏南枝抬头。

晏南枝：照得到，就要认账。认了账，才配说故土。

白翳声音仍平。

白翳：你认了旧账，就会有人上门来讨。

晏南枝：那就让他们来讨。总好过被你写成从未存在。

### SC03 雪坡逃亡线 / 连续

塔顶坠落。出口只剩半人高。沈维桑推陆青砾先出去，又把鹿弥的药草袋抛给她。

陆青砾：沈维桑！纸！

沈维桑回身，晏南枝还压着血牒裂缝。盟书残页在风里摇晃。

沈维桑没有去抓完整血牒。他先抓住晏南枝的手，把她从裂缝边拽回。

晏南枝：残页！

沈维桑用短刀割下墙边半页盟书，残页只保住一角：多族印记、旧债残句、海陆双线的一小段。

白翳伸手来夺。沈维桑把残页塞进陆青砾抛回的碎陶护符袋里。

沈维桑：完整的你们拿去烧。剩下这一角，活人带走。

沈维桑：月饼一角也能传信，盟书一角也能传路。

白翳看着他。

白翳：残缺会被各自解释。

沈维桑：那就一路解释，一路还账。

众人滚下雪坡。断塔轰然塌住入口。暴雪里，只剩朱赤旗布半截露在雪泥外。

晏南枝跪在雪地上，回头看坍塌烽燧。

晏南枝：顾师要我把日月举回旧都。

沈维桑喘着气，按住左肩。

沈维桑：你现在还要吗？

晏南枝：我要把它先从旧姓里拿出来。

陆青砾靠在雪坡边，声音发颤却仍带刺。

陆青砾：那先写上我的姓。没有姓就现编一个。

鹿弥摸着血盐袋。

鹿弥：也写上雪线和迁徙路。

沈维桑看向远处金河方向的旧水标残纹。

沈维桑：先活过今晚。

### SC04 清明院远景 / 同夜

清明院白墙内，沈照眠坐在白册前。虫蜡识别环微亮。桌上摆着断塔灰、旧水标碎石和一截烧焦发绳。

白册官吏：若沈维桑逃出烽燧，他下一条暗语是什么？

沈照眠盯着烧焦发绳，眼神短暂湿润，又被白册冷光压回。

沈照眠：灰烽不回头，顺水找灯。哥哥会避开亮雪，走金河旧水标。

白册官吏记录。

白翳从阴影中走出，白袍边缘仍沾着一点雪泥。他没有擦掉。

白翳：让她参与下一次追捕。

沈照眠抬头。

沈照眠：追捕沈维桑，还是哥哥？

白翳看着白册上的沈维桑旧姓旁注。

白翳：白册上没有哥哥。

沈照眠手指摸向发绳，嘴里极轻地背出一个草药名，又立刻停住。

远处雪坡上，沈维桑一行在暴雪中扶着彼此前行。碎陶护符袋里，盟书残页发出微弱金赤余光。

第一季结束。
