# 第12集 Outline：日月不只照旧朝

1. 钩子：沈维桑把日月旗布按进雪泥，说不能只剩旗。目的：明确主题转向。情绪：决绝。视觉：雪泥、碎旗、焦痕衣角。
2. 烽燧逃亡：旧驿坍塌，沈维桑用残纹看见出口。目的：动作开场。情绪：急迫。视觉：落石、金赤光路。
3. 白翳追上：白翳用清明香封住出口，质问没有主卷谁会信他们。目的：最终价值对峙。情绪：压迫。视觉：白烟、白袍。
4. 晏南枝选择：她拿出血牒，却没有以皇室名义号令，而是把残页交给陆青砾保管一角。目的：角色转弧。情绪：放下骄傲。
5. 突围：沈维桑引爆旧烽燧残火，薛临墙远处敲响骨钟扰乱追兵。目的：动作高潮。视觉：火光、雪、骨钟声。
6. 新路线：众人确认下一站南方隐歌谷，寻找能用禁歌读盟书残页的人。目的：第二季接口。情绪：疲惫但方向明确。
7. 清明院尾声：沈照眠在白墙内接受识别训练，念出旧驿暗语。目的：妹妹追捕线启动。视觉：白册、虫蜡环、小草药袋。
8. 最终钩子：白册浮出沈维桑路线，白翳合上册子说“让她去认”。目的：季终悬念。视觉：妹妹眼神短暂动摇。
