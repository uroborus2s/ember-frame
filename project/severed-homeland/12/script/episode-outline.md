# 第12集 Episode Outline：日月照着谁

## 结构节拍

### 1. 5秒钩子

暴雪中，白翳把日月旗布按进雪泥，说没有记忆就没有战争。

### 2. 30秒危机

旧驿坍塌，盟书残页、陆青砾、晏南枝和出口不能同时保住。

### 3. 中段升级

沈维桑放弃完整血牒，先救活人，再撕下可带走的盟书残页，晏南枝主动改写日月宣言。

### 4. 结尾悬念

清明院远景中，沈照眠准确念出哥哥会选择的下一条旧驿暗语，白翳下令让她参与追捕。

## 分场方向

- 主要角色：沈维桑、晏南枝、白翳、陆青砾、沈照眠、鹿弥
- 主要场景：坍塌烽燧内、断塔石阶、雪坡逃亡线、清明院远景
- 动作要求：所有重大信息必须落在可见动作、道具、空间选择或对白冲突上。
- 节奏要求：开场立刻进入危险或问题，中段用追捕、交易、审讯、救援或空间坍塌升级，结尾必须保留下一集推动力。

## 角色推进

本集推进由“眼前求生”转向“故土定义被不断扩展”。沈维桑每次选择都优先保护活人和路线；晏南枝每次遭遇都削弱单一血统复国逻辑；白翳和清明院的压力必须持续把旧名、旧路和亲情变成可追捕证据。

## 连续性锚点

- 半枚日月驿令、血牒残段、清明香、白册、虫蜡针、旧驿暗语和月下多族印记均按出现顺序递进。
- 沈照眠第一季不得真正团圆，只能以识别者伤口和追捕线推动后续。
- 月下盟书不得写成皇族复国诏书。
