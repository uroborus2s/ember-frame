# 第12集 Targeted Reference Frames Art Direction

## Scope

本次 Art Room 范围覆盖第12集 targeted reference frames，不补全第12集全部资产批次。当前目标资产包括 `SC002-SH003` 与 `SC003-SH003` 的 `reference_frame`。

## Shot Intent

`SC002-SH003` 是道具插入关键帧。画面重心必须落在断塔石阶上的盟书残页，让残页承担“半页证据、路线承诺、共同故土旧债”的信息；陆青砾的愣住与笑骂只作为中景反应和人物压力，不抢走前景道具焦点。

`SC003-SH003` 是血盐袋与药草袋的道具插入关键帧。画面重心必须落在雪坡前景的血盐袋，让鹿弥、北境迁徙、救援线索和“先救活人”的价值选择通过道具关系被读出；沈维桑推陆青砾向出口移动只作为中景动作，不抢走前景道具焦点。

## Visual Lock

- 镜头：16:9 横屏，近景，俯视读路线和人群，静止锁定。
- 场景：断塔内裂开石阶、雪泥、落石、半人高出口、风雪冷光。
- 色彩：黑石、雪白、冰青、旧白石、干血暗红、少量磨损朱赤和金赤残痕。
- 动作：前景残页被雪泥和裂阶夹住；虫蜡针、血牒残段和人物手部形成三方压力；陆青砾被压石卡住，沈维桑递刀和拉人只在中景可读。
- 低魔边界：金赤残纹只可短促、局部、旧债感，不得变成全屏法术光。

### SC003-SH003 Visual Lock

- 镜头：16:9 横屏，近景，俯视读路线和人群，静止锁定。
- 场景：雪坡逃亡线、断塔裂阶、雪泥、落石、半人高出口、风雪冷光。
- 色彩：深河青、湿木褐、蒸汽白、雪白、暗红盐晶、少量金赤余烬。
- 动作：前景血盐袋和雪地药草袋落在雪泥里；碎陶护符和盟书残页半埋；沈维桑推扶陆青砾向出口，白翳在背景形成白袍追捕压力。
- 信息预算：前景道具高细节，中景人物身份色和动作可读，背景只保留出口、断阶和追兵剪影。

- `P004` / `PROP_INSECT_WAX_NEEDLE`: 骨白虫蜡针，洁净制度感，非毒针奇观。
- `C003` / `CH_LU_QINGLI`: 灰蓝短袄、陶土内层、碎陶护符、黑市木牌、耳后细鳞痕。

- `P012` / `PROP_BROKEN_POTTERY_CHARM`: 陆青砾碎陶护符和墙下身份锚点，只做前景/中景识别，不抢主体。

## Rejection Rules

判退：盟书残页不在前景、生成可读文字或新徽记、画成皇族圣物、虫蜡针像现代医疗器械、场景变成普通雪地或圣殿、陆青砾/沈维桑抢成角色海报、全画面灰黑糊成一片、缺少前景/中景/背景层次、出现现代 UI/水印/字幕。

`SC003-SH003` 额外判退：血盐袋不在前景、药草袋缺失、道具变成现代背包或游戏装备、雪坡变成空白雪地、坍塌变成全景灾难奇观、白袍追兵抢主体、碎陶护符或盟书残页生成可读文字、前中背景没有层次。

## SC002-SH001 Added Direction

`SC002-SH001` 是断塔石阶的横屏远景建立镜头，使用 FLF2V，需要首帧和尾帧参考资产。画面目标不是道具特写，而是让观众立刻读懂三重压力：陆青砾被断阶困住、盟书残页悬在破墙边、半人高出口正在被风雪和落石压缩。

- 首帧 `E12_R001` / `r001e12.png`: 眼平横屏远景，慢推起点。前景读黑石断阶、湿雪、布料和红褐旧痕；中景读陆青砾、沈维桑、晏南枝的行动三角；背景读半人高出口、盟书残页、鹿弥削弱的骨笛支撑和白翳的白色制度压力。
- 尾帧 `E12_R002` / `r002e12.png`: 同一空间的慢推终点，略近但仍保留地理。前景更清楚读陆青砾被卡住的衣料/绳扣和小工具动作；沈维桑更接近她，左肩受限的身体代价可见。
- 安全改写：生图提示词把模型可见的直接血/刀表述改为 `dark red-brown weathering`、`small utility tool`、`old record bundle`，生产 metadata 中的连续性 ID 保持不变。
- 信息预算：只给陆青砾被困、沈维桑移动、出口、盟书残页和前景石阶/布料高细节；晏南枝、鹿弥、白翳为中低细节压力层；远处碎石、雪线和墙裂只做氛围，不做全画面微细节。
