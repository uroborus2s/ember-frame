# 第09集：鸣骨岭有笛声

时长目标：4分25秒
格式：AI竖屏短剧详细剧本
本集功能：让北境“兽潮”叙事裂开，揭示雪灾、阵基失衡与多族旧誓。

## 场景表

1. 鸣骨岭雪脊：风过骨孔，沈维桑辨出求救骨笛。
2. 鸣骨岭雪坑：护送队救出鹿弥，晏南枝想把灾情当筹码。
3. 骨笛谷口：甲军斥候与北境掠粮派夹击。
4. 边墙外废烽燧：鹿弥认出月下印记是多族旧誓。

## 成片剧本

### SC01 鸣骨岭雪脊 / 暴雪 / 5秒钩子

半埋雪中的巨骨被风吹响。声音三短一长，像有人在骨头里哭。

陆青砾脸色发白，窄刀贴着掌心。

陆青砾：鬼在吹笛？

沈维桑按住左肩，听完第二声。

沈维桑：不是鬼。是有人在求救。

晏南枝握住月白玉片。玉片在风声里微微发烫。

晏南枝：翻过鸣骨岭就是寒鸦堡。过了寒鸦堡，月下旧驿就不远。

沈维桑蹲下拨开雪，露出旧兵器上的白翅黑日刻痕。

沈维桑：甲军府来过。这里不只是北境人的路。

骨笛声又响一遍。沈维桑已经朝雪坑跑去。

### SC02 鸣骨岭雪坑 / 连续

雪坑底部有冰蓝火将灭。鹿弥倒在碎石边，鹿耳覆霜，骨笛落在手边，药草袋被血浸湿。

鹿弥警觉抬手。

鹿弥：中陆人，退后。

沈维桑停在三步外，把短刀插回鞘里。

沈维桑：你吹的是求救。

鹿弥：我吹给守誓派，不吹给墙里人。

晏南枝看见鹿弥肩上的伤，又看见远处灰黑旗影。

晏南枝：后面是甲军府，前面是掠粮派。她若知道雪灾从哪儿起，我们到了寒鸦堡才有话可谈。

鹿弥伤口再次渗血，骨笛从指缝滑落。晏南枝伸出的手停在半空，改为把玉片塞回怀里，撕下护腕压住伤口。

晏南枝：我刚才想拿你去谈。现在先拿我自己的手救你。

鹿弥：记住这句，等你到了月下再说一遍。

沈维桑撕开自己的灰麻披风，压住鹿弥伤口。

沈维桑：她现在不是筹码，是会死的人。

晏南枝一顿，没有反驳。

陆青砾从雪里摸出一块碎骨牌。

陆青砾：这不是掠粮派的号牌。上面有旧墙阵纹。

陆青砾把碎骨牌翻面，旧墙阵纹下还粘着一点虫蜡封痕。

陆青砾：北境人的牌，不会用清明院的官蜡封口。

沈维桑：有人先让墙坏，再给两边各指一个仇人。

鹿弥盯着骨牌，声音发冷。

鹿弥：墙阵三年前就该修。肃明只抽粮，不修墙，还逼我们南迁抢食。雪线一乱，巨兽也醒错了时节。

晏南枝：边墙阵基失衡？

鹿弥：不是白册上写的“兽潮贪食”。是有人断了迁徙路。

沈维桑抬头，看见雪脊上甲军斥候的铁闸令牌反光。

沈维桑：离坑。

### SC03 骨笛谷口 / 稍后

谷口一侧出现甲军斥候，另一侧传来北境掠粮派兽骨号。护送队被夹在雪岭中间。

甲军斥候：交出昭明余脉，墙外之人免斩。

北境掠粮哨兵：交出鹿弥，谷粮归我们。

陆青砾贴墙笑了一声。

陆青砾：两个价，一条命。真公平。

晏南枝把血牒残段按在袖中，向前一步。

晏南枝：我是昭明日月血脉。若你们仍认旧誓--

鹿弥猛地打断。

鹿弥：旧誓认盟，不跪皇血。

晏南枝咬住后半句话，把血牒放低，露出空掌。

晏南枝：那旧誓认什么？我听。

沈维桑把骨笛递回鹿弥。

沈维桑：你能让掠粮派停一息吗？

鹿弥：一息不够逃。

沈维桑：够我改路。

鹿弥摸了摸血盐袋，吹响骨笛。谷口远处的共生兽躁动一滞。沈维桑把半枚日月驿令贴在冻裂石壁上，金赤残纹亮起一小截，指向废烽燧下方排烟暗口。

陆青砾：你这旧法每次都像快死了。

沈维桑硬拉开石闩，左肩痛得一抖。

沈维桑：所以快走。

甲军斥候扑近。沈维桑用猎弓短击斥候膝弯，晏南枝以藏匕割断铁闸令牌绳。令牌落雪，斥候失去调兵信号。

一行人钻入暗口。

### SC04 边墙外废烽燧 / 暮色

废烽燧内残存白石旧墙，金赤路线断断续续。鹿弥靠墙坐下，目光落到晏南枝的月白玉片上。

鹿弥：给我看。

晏南枝握紧玉片。

晏南枝：这是昭明旧物。

鹿弥：我认得上面的角纹。那不是你一家人的印。

晏南枝没有把月白玉片举在胸前，而是放到白石旧墙上。鹿弥用血指擦开灰，陆青砾也伸手抹去另一处积尘，残纹里露出不同族徽。

鹿弥：鹿角、灰墙、月白石、短锤、海陆双线。残缺，但还在。

鹿弥：你看，它从来不是一个姓刻完的。

陆青砾轻声笑了一下。

陆青砾：日月旗里，原来也不是只有旧姓。

晏南枝没有反驳。她第一次把玉片放低，让其他人都能看见。

沈维桑看向废烽燧深处。

沈维桑：月下路还在前面。

鹿弥：你们去月下，我也去。我要亲耳听，你们把旧誓读成什么。

远处雪岭上，甲军府黑旗重新聚拢。寒鸦飞起，朝寒鸦堡方向盘旋。

晏南枝收起玉片，没有再说“复国”。

晏南枝：那就去把它读完整，不去把它抢成自己的。

切黑。
