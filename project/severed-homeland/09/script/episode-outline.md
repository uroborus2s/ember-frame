# 第09集 Outline：鸣骨岭的敌人

1. 钩子：北境少女求沈维桑救幼兽。目的：打破敌我定式。情绪：迟疑。视觉：骨笛、幼兽血迹。
2. 鸣骨岭穿行：风吹骨孔发声，陆青砾害怕，晏南枝戒备。目的：场景压迫。视觉：雪中骨骸。
3. 救援选择：沈维桑放下刀，帮鹿弥止血，晏南枝担心暴露。目的：价值冲突。情绪：紧绷。
4. 寒灾真相：鹿弥讲述草场冻死、兽群失路、南迁战派和寻灾源派分裂。目的：世界观推进。视觉：冰蓝火、骨饰。
5. 追兵逼近：厉螳派出的边墙斥候要杀鹿弥灭口。目的：证明虫族也利用边境仇恨。情绪：危险。
6. 合作脱身：鹿弥用骨笛引开巨兽，沈维桑设雪坑困斥候。目的：临时跨族合作。视觉：骨笛、雪坑塌陷。
7. 结尾转折：鹿弥说“月下盟书在寒鸦堡背后，你们人族自己也忘了”。目的：导向第10集。视觉：寒鸦堡远影。
