# 第09集 Episode Outline：鸣骨岭有笛声

## 结构节拍

### 1. 5秒钩子

鸣骨岭风吹骨孔，沈维桑听见的不是鬼声，而是北境萨满的求救骨笛。

### 2. 30秒危机

甲军府追兵和北境掠粮派同时逼近，护送队被夹在骨雪岭上。

### 3. 中段升级

鹿弥揭示雪灾异常、迁徙线断裂和边墙旧阵失修可能有关，沈维桑选择先救伤者。

### 4. 结尾悬念

鹿弥认出月下盟书印记不是皇姓单独的印，而是多族旧誓。

## 分场方向

- 主要角色：沈维桑、晏南枝、鹿弥、陆青砾、甲军斥候
- 主要场景：鸣骨岭雪脊、雪坑、骨笛谷口、边墙外废烽燧
- 动作要求：所有重大信息必须落在可见动作、道具、空间选择或对白冲突上。
- 节奏要求：开场立刻进入危险或问题，中段用追捕、交易、审讯、救援或空间坍塌升级，结尾必须保留下一集推动力。

## 角色推进

本集推进由“眼前求生”转向“故土定义被不断扩展”。沈维桑每次选择都优先保护活人和路线；晏南枝每次遭遇都削弱单一血统复国逻辑；白翳和清明院的压力必须持续把旧名、旧路和亲情变成可追捕证据。

## 连续性锚点

- 半枚日月驿令、血牒残段、清明香、白册、虫蜡针、旧驿暗语和月下多族印记均按出现顺序递进。
- 沈照眠第一季不得真正团圆，只能以识别者伤口和追捕线推动后续。
- 月下盟书不得写成皇族复国诏书。
