# 第一季台词旁白 99 分标准

更新时间：2026-06-05
适用范围：`01`-`12` 集 `script/final-script.md`，同步影响后续 `script-v01.md`、评分报告与连续性说明中涉及台词口径的部分。

## 总原则

台词和旁白必须让中国观众一听就知道压迫关系、身份风险和历史隐喻，但不能把设定解释成说明书。保留清明籍、白册、旧驿、月下、血牒、日月、清明香等本世界名词；每个关键名词必须落到族谱、户籍、粮牌、押车、改供、逆案、讨债、墙、粮、路、灯、月饼、红花、白莲、河不清等可直觉理解的语义。

## 官府与制度台词

- 官府说话要像判词、册文、押车口令：短、硬、带后果。
- 优先使用 `逆案`、`窝户`、`封户口`、`改供`、`押车`、`翻祖账`、`销旧账`、`粮门照封`、`白册记账`。
- 避免 `旧污`、`真名`、`封名`、`旧案窝点` 这类只懂世界观才懂的词。
- 不能只说“识别”“记忆”“血脉反应”，必须让观众听出后果：入册、押车、换名、查祖、清户、连坐。

## 民间暗语与歌谣

- 歌谣必须像民间流传的半句，不像主角宣言。
- 可借用中国观众熟悉的联想：`红花`、`月饼藏信`、`白莲香`、`河不清`、`留灯`、`祖账`、`族谱匣`。
- 暗语要贴场景，不孤立堆典故。雪夜用灯，粮门用粮账，白墙用白莲，旧路用月饼与碎陶。
- 禁歌示范：`红花不落雪留灯。` 这类句子比 `日月重明` 更隐晦，也更像悼念前朝和接应逃亡者的旧歌。

## 角色口吻

- 沈维桑：短句、实用、先救活人；说路、风、粮、墙，不说大义长论。
- 晏南枝：前期有皇族姿态，后期必须把口号落到人、粮、姓、账、活路。
- 白翳：冷静官府逻辑，把压迫说成清账、除根、止战。
- 罗青禾、孟归藏：民间经验和旧知识，不讲设定，讲藏信、藏祸、旧火、旧路。
- 陆青砾：钱、粮、名额、册外人；一句话要带刺。
- 鹿弥、薛临墙：雪线、墙、粮、守誓、断粮；不要抽象民族叙述。
- 沈照眠：孩子语言被白册训练成条目，但偶尔漏出药名、发绳、哥哥等私人记忆。

## 旁白与动作描述

- 旁白只补压迫感和可拍动作，不替角色解释世界观。
- 可用物件承载含义：白蜡旗、虫蜡针、粮牌、押车铜环、碎陶、血牒缺角、旧水标、烧焦发绳。
- 避免把主题写成结论；让角色用动作和代价完成主题。

## 99 分判定

- 每集台词旁白 99 / 100，允许保留极少设定名词，但不得有需要额外解释才能懂的关键句。
- 每集 `Dialogue` 分项维持满分，评分报告需记录 99 分台词旁白 pass。
- 精修不得改变主线事件、角色归属、道具规则、路线、季终钩子或 downstream handoff。
