# 《断航故土》肃明基层虫吏多状态角色板美术方向 v01

版本：v01  
日期：2026-06-03  
范围：项目级可复用角色资产，补齐肃明国 / 清明院体系下“虫族底层官员、白册官吏、粮税小吏”的基层官僚层。

## 1. 定位

本轮不重新设计任何阵营徽章。已经完成的肃明 / 清明院“白翅覆黑日”徽章继续作为锁定根参考，只能以极小的物理引用出现在旧铁账牌、蜡印、白册角章、帽扣或腰牌上。

肃明基层虫吏不是白翳，不是肃明虫族小兵，也不是人族衙役。他们是把肃明统治落到村口、粮仓、渡口和清明院外署登记桌上的中下层官僚：开册、点名、征粮、验气味牌、盖蜡印、封门和调用奴兵。镜头里他们承担“制度下令”：官吏开册，奴兵扣人，虫族小兵封锁外围。

画板目标是让 Prompt Room 和后续镜头稳定复现“低阶虫族官僚”形象：近人形官僚轮廓，但虫相、服色、发型和道具必须清楚地区别人族官员。

## 3. 角色锁

肃明基层虫吏必须读作低阶虫族官僚。

- 身高：1.8 到 2.1 米，接近直立，但肩背略前倾。
- 体态：比白翳低矮、僵硬、行政化；比小兵更像官僚，不能低伏或反关节奔袭。
- 脸部：暗褐或黑曜石复眼，无眼白、虹膜或普通瞳孔；骨白颊壳或下颌小甲片；嘴部可被窄硬领和短颌片遮住。
- 虫化点位：暗绿颈侧软膜、节状手指、脊背细甲线、袖口下露出的细甲腕、偶发短触须束。
- 发型/头部：没有完整人族头发、发髻、鬓角或胡须。低筒虫蜡小冠、半圆账吏帽或骨白短冠遮住头顶；后脑从冠沿下露出一根细硬钱尾式虫须小辫，或两到三根被修剪的短触须束。它们是虫族官籍标记，不是人族束发。
- 服装主形：蜡白灰或珍珠灰短官袍，袍摆到膝下或小腿；暗绿黑内层从领口、袖口和侧摆露出。
- 制度道具：白册夹板、虫蜡印盒、旧铁账牌、气味牌、骨算盘、粮牌、封门令签、蜡封小槌。
- 色彩锁：蜡白灰、骨白、珍珠灰、暗绿黑、旧铁、黑日小牌、极少冷金小钉；不得使用人族麻布黄褐、书生青衫、朱红官袍或温暖木色主调。

## 4. 画板结构

生成一张竖版多状态角色板，图片内部不出现文字、标签、字幕、UI 或水印。

### 三个全身状态

1. 虫族底层官员
   低筒虫蜡小冠，蜡白灰短官袍，暗绿黑硬领，旧铁腰带和黑日小牌，手持虫蜡令签或封门小槌。用于村口台阶、清剿队前列、粮仓门口下令。

2. 白册官吏
   半圆账吏帽，较窄短袍，袖口骨白虫蜡绑带，随身白册夹板、蜡笔针和小印盒。动作细碎、快速、重复，用于验名、划册、盖蜡印、闻气味牌。

3. 粮税小吏
   袍色更偏蜡灰青，腰侧挂粮牌、骨算盘、旧铁秤钩、蜡封袋和气味牌。用于金河粮仓、磨坊、边墙粮门和征粮队。

### 细节面板

- 低筒虫蜡小冠 / 半圆账吏帽 / 骨白短冠。
- 冠后钱尾式虫须小辫与短触须束。
- 暗色复眼、骨白颊壳、暗绿颈膜。
- 节状手指按白册、盖蜡印或捏气味牌。
- 蜡白灰短官袍、暗绿黑内层、骨白袖绑。
- 旧铁账牌、黑日小牌、腰侧气味牌。
- 白册夹板、虫蜡印盒、蜡笔针、骨算盘、粮牌、旧铁秤钩。
- 材质色条：蜡白灰、骨白、珍珠灰、暗绿黑、旧铁、黑日黑、冷金小钉、虫蜡淡黄。

## 5. 阶层差异规则

- 与白翳区分：基层虫吏不能有高筒清明官帽、无尘落地白袍、薄翼披肩、2米以上高瘦压迫身形、上层清俊近人脸或虫蜡针主导气质。
- 与肃明虫族小兵区分：基层虫吏必须直立、持册、持印、穿短官袍；不能低伏奔袭、反关节腿、爪足、护颚口器、折叠臂刀或短矛主导。
- 与肃明奴兵区分：基层虫吏是下令者和登记者，不是扣腕押车的混血执行层；他们有复眼、虫冠、虫须小辫和官吏道具，不有人族羞惧表情。
- 与人族官吏区分：不得有完整黑发、发髻、布巾、胡须、普通人眼、温暖肤色、青衫长袍、朱红官袍、毛笔文士气或普通衙役造型。

## 6. 禁忌

- 不重新设计肃明 / 清明院徽章。
- 不把黑日小牌放大成新胸章、新旗帜或新 logo。
- 不画成白翳缩小版。
- 不画成人族书生、账房先生、县衙师爷、普通人类衙役或戴面具的人。
- 不画成纯虫族战斗小兵、厉螳式螳刃军官、甲壳怪物或虫潮。
- 不使用完整人族头发、发髻、鬓角、胡须、普通人眼、书生袍、麻布黄褐长衫或暖色人族官服。
- 不出现可读文字、中文、英文字母、标签、UI 或水印。
- 不出现现代制服、现代警具、枪械、科幻机甲、赛博虫、高魔法阵、高饱和游戏海报或血腥酷刑展示。
