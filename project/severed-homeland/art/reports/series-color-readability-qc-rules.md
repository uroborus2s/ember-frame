## 口径修正

“暖色多彩”不是要求所有画面都温暖、明亮、花哨，也不是禁用阴暗、寒冷、灰褐或压迫感。它只是此前用于反对“全剧默认灰褐滤镜”的临时术语，容易误导。

新的准确口径是：故土本身要有美，山河、村落、土地、旧驿、集市、雪原、河渡、月色和建筑尺度都要能让观众相信这片土地值得守、值得归。时代的痛、制度的冷、战争的脏、流亡的苦是人造成的创伤，应压在这份美上，形成反差，而不是把自然、土地、衣服、人物和天空全部涂成同一种灰。

## 新全局规则

1. 色调服从场景情绪：该明亮时明亮，该阴暗时阴暗，该宏伟时宏伟，该萧瑟时萧瑟；禁止所有资产默认压抑灰暗。
2. 场景卡必须先画出故土之美：地貌、天空、水系、田地、村落、建筑、旧文明遗迹和地域材质要有可读层次。
3. 痛感来自人祸和制度：边墙、清明院、军府、审讯、征发、焚毁、逃亡、贫困和压迫要作为侵入美景的伤口出现，不能把整片土地都画成无差别脏灰。
4. 灰、冷灰、灰褐、黑灰、烟灰、石灰、雪雾可以根据场景和材质逻辑使用在泥土、木石、旧皮革、旧麻布、阴影、尘土、灰烬、石墙、旧化、磨损、雪雾或灾后痕迹上，但不得成为全剧统一滤镜。
5. 服饰、盔甲、平民衣着、群像模板和场景卡要有地域、势力、阶层、职业或自然环境带来的可读差异；不是简单追求色相数量，而是避免全员灰麻、全场灰褐。
6. 人物、服装、甲片、行动区和背景必须通过色相、明度、材质、边缘光或构图留白分离，不能融成同一块灰褐。

## 硬拒绝项

- 拒绝全剧统一压抑灰暗、冷灰滤镜、黑灰铁块化、全员灰麻；不拒绝符合场景、天气、材质和剧情逻辑的阴暗与灰褐。
- 拒绝灰墙、灰地、灰衣同色导致人物放入场景后消失。
- 拒绝把暖光或彩色只做成小点缀，而主体仍是灰底。
- 拒绝把北境、清明院、灰烬书院等本来可有冷色/白色/灰烬的场景处理成全剧通用灰滤镜。
- 拒绝用低饱和表示贫困或历史感；贫困、战争和制度伤害必须通过人造痕迹、破坏位置、衣物修补、器物消耗和空间压迫表达。
- 拒绝场景只有痛感没有土地美感；尤其是地点卡，必须让观众看见“故土为什么值得失去与守护”。

## 类型要求

| 类型 | 新要求 |
| --- | --- |
| 服装 K | K001-K006 全部重做；灰褐可按角色、地域、材质和场景条件使用，但不能把贫穷、军制、旧化都画成同一种灰。 |
| 角色 C | 角色服饰必须继承新版 K 母版；身份色、阶层色、势力色和生活痕迹必须比灰尘和旧布更可读。 |

## K001-K006 新合同摘要

| 资产 | 暖多色主读要求 |
| --- | --- |
| K001 | 灰褐猎衣可以作为沈维桑的外层实用色保留，但草药青、旧皮棕、麦金/旧驿金赤必须共同形成可读结构，不能让灰褐成为整板主读。 |
| K002 | 雾紫、深墨蓝、朱赤、宝蓝、月白、磨损金必须比冷灰遮蔽外层更醒目。 |
| K003 | 儿童村中服与清明院服不能灰白一片；草药绿、烧焦发绳、虫蜡白、冷金识别环必须分层。 |
| K004 | 清明院制度白/黑不能变成整板灰；绯色、冷金、蜡灰青、奴兵杂旧甲需要层级色差。 |
| K005 | 北境不是灰白雪地滤镜；枯草黄、风蚀褐、骨白、冰青、暗血誓绳要共同主导。 |
| K006 | 平民地域差异必须多色化：麦金、草药青、深河青、旧染蓝、干血暗红、红线、旧白石粉等不能被灰麻吞掉。 |

## 执行要求

- 后续重生成时，旧 canonical 图必须先归档到对应 `history/`，只有通过“故土美感与人祸反差”QC 的候选才能写回 canonical 路径。
- 角色卡和镜头参考帧继承服装时，必须优先使用新版 K001-K006。
- 场景卡必须同时通过“真实可拍空间”、“人物放置区可读性”和“故土景色美感”检查。
- 任何生成结果只要人放进场景会融为一体，即使文件格式正确，也判定失败。
