范围：项目级可复用角色资产，补齐肃明国 / 甲军府基层武力单位“肃明虫族小兵”。

## 1. 定位

本轮不重新设计任何阵营徽章。已经完成的肃明 / 清明院“白翅覆黑日”徽章继续作为锁定根参考，只能以小型物理引用出现在甲军府门牌、虫蜡封印、旧铁小牌或甲壳烙印上。

肃明虫族小兵不是白翳，也不是厉螳。它们是甲军府和地方清剿队的基层武力单位，承担封村、押运、守门、追捕和粮仓巡逻。画板目标是让 Prompt Room 和后续镜头生成能稳定复现“二到三只小兵即可形成封锁感”的镜头识别点。

## 3. 角色锁

肃明虫族小兵必须是明显非人形虫族步兵。

- 身高：2.0 到 2.3 米，低伏强壮。
- 体态：胸背硬化、前倾低伏、反关节腿、爪足。
- 头部：窄复眼位于前额两侧，口器被骨白护颚遮住。
- 武装：主臂持短矛或钩镰，前臂外侧有可折叠短刃。
- 背部：退化小硬翅变成叠甲板，不是装饰翅膀。
- 配色：黑绿或深褐甲壳、骨白护颚、虫蜡白绑带、旧铁束具、暗绿绳带、极少冷金小钉和暗红刃痕。
- 阵营识别：小型白翅覆黑日旧铁牌、虫蜡封印或甲壳烙印，必须看起来来自现有徽章系统，不能成为新徽章。

## 4. 画板结构

生成一张竖版多状态角色板，图片内部不出现文字、标签、字幕、UI 或水印。

### 三个全身状态

1. 封村巡逻状态  
   短矛竖持，低伏站姿，单道虫蜡白关节绑带，旧铁胸带，粗制甲军府门牌。用于村口、井边、门槛、白册验村场景。

2. 押运 / 粮仓守门状态  
   钩镰或短杆兵器，胸背旧铁束具更清楚，肘膝两道虫蜡白绑带，暗绿绳带和封条碎片。用于押运粮车、守粮仓、封锁路口。

3. 追捕突进状态  
   低位前扑，前臂短刃展开，短矛反持为副武器，反关节腿蓄力，复眼反光和护颚压迫感增强。用于巷道追捕、门框刃影和近景威胁。

### 细节面板

- 窄复眼与骨白护颚
- 折叠 / 展开前臂短刃机构
- 旧铁束具和暗绿绳带
- 微型锁定徽章引用：旧铁牌、蜡印或烙印
- 虫蜡白绑带和裂纹蜡质
- 背部退化硬翅叠甲
- 反关节腿和爪足
- 短矛、钩镰武器头
- 材质色条：黑绿甲壳、深褐甲壳、骨白护颚、虫蜡白、旧铁、暗绿绳、冷金小钉、暗红刃痕、黑石雪尘

## 5. 阶层差异规则

- 与白翳区分：小兵不能有高筒清明官帽、虫蜡白官袍、近人形脸、薄翼披肩、白册或虫蜡针。
- 与厉螳区分：小兵体量更低、更批量化，不出现指挥令牌、背甲高阶刻纹、长螳刃、黑短披挂或将领姿态。
- 与人类士兵区分：小兵不穿完整布衣、盔甲制服、靴子或披风；“服装系统”来自甲壳、旧铁束具、虫蜡绑带和小牌。

## 6. 禁忌

- 不画成人类士兵戴虫盔。
- 不画成厉螳复制体或精英 Boss。
- 不画成白翳式清明官员。
- 不出现新肃明徽章、新虫族 logo、大旗帜、大胸章或重绘徽章。
- 不出现文字、标签、说明字、中文、英文字母、UI、水印。
- 不做科幻机甲、赛博虫、怪物潮、血腥恐怖、高魔法阵或高饱和游戏海报。
