# 《断航故土》区域色彩系统

版本：v01  
日期：2026-06-02  
范围：第一季《残阳驿火》项目级区域色彩锁  
状态：规划完成，等待总风格板与场景资产合流  

## 1. 设计原则

区域色彩不是单纯换滤镜，而是把地理、权力、记忆和角色心理绑定成可延续的视觉语法。第一季路线从残阳坳到边墙，再转向南方线索，画面必须让观众不用旁白也能感到：他们正在穿过被改名、被征粮、被清明籍筛查、被战争消耗的同一片故土。

全局规则：

1. 每个区域保留 60%-75% 的本地底色，15%-25% 的权力覆盖色，5%-10% 的旧帝国或低魔提示色。
2. 肃明国覆盖层进入任何区域时都使用虫蜡白、骨白、珍珠灰、黑日、暗绿甲壳，但不能把所有区域统一洗成清明院白色。
3. 昭明旧帝国记忆使用朱赤、宝蓝、月白、磨损金、金赤断线，必须残缺、被遮挡或被保存，不做无瑕复国圣光。
4. 人族生活空间可以旧、冷、缺粮、烟熏，但必须有职业秩序和可居住感，不能被调成破布流民色。
5. 低魔色只做短促局部：灵翼残纹金赤，清明香冷白，月白玉片一线反光，虫蜡封条淡黄白，血牒干血红。

## 3. 残阳坳

色彩定位：残阳坳不是“贫民窟”，而是被压榨但仍有生活秩序的农猎村。它的温度来自土墙、药屋、麦色、旧井和草药青；压迫来自白册、虫蜡针和冷白清明香。

配色：

| 类型 | 色名 | Hex | 用途 |
| --- | --- | --- | --- |
| 主底色 | 夯土褐 | `#6E563E` | 土墙、村道、门槛、旧井边泥土 |
| 主底色 | 晚麦金 | `#B08A3C` | 黄昏、麦束、晒谷场、残阳边缘 |
| 辅助色 | 草药青 | `#4F6B4A` | 罗家药屋、药篮、沈家生活锚点 |
| 辅助色 | 旧木褐 | `#3A2B21` | 药柜、弓架、门框、水桶 |
| 覆盖色 | 虫蜡白 | `#D8D0BC` | 封门牌、虫蜡针、白册边缘 |
| 覆盖色 | 黑绿甲 | `#1E2A22` | 螳刃小兵局部甲壳、门口剪影 |
| 点色 | 月白一线 | `#C8D7D8` | 井口、水面、玉片未命名反光 |
| 危机色 | 火场暗红 | `#7F2D22` | 第02集火场、焦痕衣角、灰烬边缘 |

镜头规则：

- 黄昏红只照边缘，不把村子拍成温暖田园。
- 清明香出现时，让冷白细线贴地爬过草鞋、门槛和井沿，压低原有土褐。
- 火后保留旧井、药柜、暗格和弓架的轮廓，让观众认出“被毁的是原来的生活空间”。
- 沈维桑离开后，焦痕衣角继续携带残阳坳暗红，不能被新区域完全洗掉。

禁忌：

- 不用过饱和夕阳、金色田园滤镜。
- 不用满屏黑灰火场吞掉空间关系。
- 不把村民衣物调成无职业信息的破布灰。

## 4. 金河

色彩定位：金河是宽阔水路和生产控制的区域。它比残阳坳更开阔、更湿、更机械，但不是现代港口，也不是欢乐蒸汽朋克。它的核心矛盾是：水、蒸汽和齿轮仍在服务粮食运输，战争能力却被磨平。

配色：

| 类型 | 色名 | Hex | 用途 |
| --- | --- | --- | --- |
| 主底色 | 深河青 | `#214C4E` | 河面、河雾阴影、水标倒影 |
| 主底色 | 雾灰 | `#9FAAA6` | 清晨河雾、远岸、湿空气 |
| 辅助色 | 湿木褐 | `#4E3A2A` | 码头桩、粮船、修船棚 |
| 辅助色 | 铁灰 | `#4D5353` | 水泵、轨车、磨盘、齿轮 |
| 覆盖色 | 粮牌暗黄 | `#A88435` | 征粮木牌、粮袋标签、检查口 |
| 覆盖色 | 油黑 | `#171817` | 轴承、磨平接口、地窖机械阴影 |
| 点色 | 磨损铜金 | `#8A6A3C` | 被磨平炮架刻度、旧水标残痕 |
| 低魔色 | 金赤断纹 | `#B35A2B` | 半枚驿令与旧水标短促共鸣 |

镜头规则：

- 构图比残阳坳横向更开阔，但竖屏中用码头桩、烟囱、水轮和粮仓门保持纵向压迫。
- 蒸汽白必须半透明，露出齿轮和人手，不把画面变成纯白雾。
- 金河的“金”来自粮、湿木和旧铜，不是华丽金光。
- 白翳或清明香进入金河时，冷白只作为追踪线和封牌，不覆盖整个河色。

禁忌：

- 不出现现代港口、现代轮船、科幻机械 UI。
- 不使用橙黄蒸汽朋克游乐场质感。
- 不把大河拍成小溪或装饰水景。

## 5. 清明院

色彩定位：清明院是全剧最接近单色的空间，但它的单色必须来自制度洁净，不是医院科幻白、宗教圣殿白或怪物洞穴白。这里的白会吞掉人的生活色，让角色衣物显得不合时宜。

配色：

| 类型 | 色名 | Hex | 用途 |
| --- | --- | --- | --- |
| 主底色 | 虫蜡白 | `#DDD8C8` | 墙面、封条、针具托盘、走廊反光 |
| 主底色 | 骨白 | `#CFC6B4` | 桌面、门洞、识别环、儿童白衣 |
| 辅助色 | 珍珠灰 | `#B8B9B3` | 地砖、灯槽、白册阴影 |
| 权力色 | 黑日 | `#101111` | 白翅覆黑日徽记、册封印、门心 |
| 权力色 | 冷金 | `#9C8C5F` | 官制小钉、边角金属、白翳高阶标识 |
| 武力色 | 暗绿甲壳 | `#1B2B24` | 门卫复眼、护颚、刃肢局部 |
| 危险点色 | 干血红 | `#6E1F1E` | 验血、血纹、白册边缘极小痕迹 |
| 低魔色 | 清明冷白 | `#E6ECE7` | 清明香、虫蜡针反光、记忆室冷光 |

镜头规则：

- 顶光偏冷，阴影边缘硬，人物被白墙和重复门洞压扁。
- 人族角色进入后，不改变服装本色，但让草药青、焦痕红、雾紫等身份色被压低饱和。
- 清明院内部的血色必须少，越少越刺眼。
- 白翳的高阶虫族身份通过白、黑日、冷金、复眼暗面和薄翼质感表现，不画成人戴面具。

禁忌：

- 不做纯怪物巢穴、肮脏地牢或血腥实验室。
- 不做蓝白现代医疗空间。
- 不使用暖色宗教圣光。

色彩定位：边墙是旧帝国工程、防灾防线和肃明牢笼的叠合。它必须比金河更硬、更高、更冷，且黑石与雪白形成强烈垂直压迫。

- 巨兽只用黑影、角影、撞击裂纹和雪尘表现，避免满屏怪物潮。
- 金赤阵基只在剥落处、雪下或墙脚短促出现，不能变成大范围法阵。

禁忌：

- 不把边墙军画成闪亮英雄军团。
- 不用蓝紫高魔战场光。

色彩定位：北境不是纯白雪原。它有枯草、冻土、风蚀山、骨标、兽皮、冰湖和低矮火光。北境颜色要让观众理解：这里的人和兽不是天生怪物，而是被寒灾推向南方。

- 不把北境拍成浪漫牧歌。
- 不把所有北方镜头调成纯冷蓝。

- 不做无瑕宫殿、亮丽宫斗或纯正义圣殿。
- 不做高饱和南国奇观。
- 不把南方遗民拍成现代秘密组织会议室。

## 9. 辅助区域

### 灰烬书院

主色为焦黑 `#1C1A18`、书灰 `#6F6A62`、裂铜 `#7B5A36`、灰白尘 `#B7B0A5`。月白星图残光 `#C7D2D1` 只能从碎裂星图盘、塌顶冷光和纸灰边缘露出。清明覆盖层为虫蜡封条与识别者白衣，不做恐怖地牢。

### 墙下集市

主色为灰蓝棚布 `#596B78`、陶土 `#8A5F45`、旧布黄 `#A38B58`、黑木牌 `#27221D`、烟灰 `#6C675E`。它是混裔、失籍者和军户家属的生存空间，色彩应杂而有序。日月旗在这里应褪色、被撕、被交易，不做神圣旗面。

### 旧驿与月下盟书

主色为旧白石 `#BDB7A5`、湿石灰 `#777B75`、黑影 `#161817`、月白 `#D6DAD5`、金赤断线 `#B85B2C`。月下盟书旧驿可出现低饱和多族印记，但每一种印记都要像被磨损的石刻或旧墨，不做彩色联盟徽章墙。

## 11. 角色携带色规则

1. 沈维桑：灰褐猎户色、草药青内层、旧皮褐和焦痕暗红必须随区域迁移保留。进入清明院时被压低，进入边墙时与灰墙布融合，但焦痕暗红仍可在衣角出现。
2. 晏南枝：早期外层雾紫/冷灰，核心身份色为月白、宝蓝和磨损金，只在玉片、内衬、卷轴绳和旧印反光中露出。南方旧帝国区域才允许提高宝蓝与朱赤占比。
3. 白翳：无论进入哪个区域，都携带虫蜡白、珍珠灰、黑日和冷金。他的色彩像一块制度切片插入本地环境。
4. 沈照眠：残阳坳时期为草药青与儿童旧布色；清明院后转为儿童白衣、虫蜡识别环和被压低的旧发绳色。
5. 陆青砾：灰蓝、陶土、黑木牌和碎陶护符色，不被日月朱赤或清明白吞没。
6. 薛临墙：灰墙布、旧铁、骨哨、冻裂肤色和隐藏日月压痕，不能亮成旧帝国军官。
7. 鹿弥：枯草黄、兽皮暗棕、骨白、冰蓝火，与边墙灰和清明白区分。

## 12. 调色与 Prompt Room 交接

Prompt Room 使用本系统时，应把区域色彩写成“本地底色 + 权力覆盖 + 旧记忆点色 + 禁忌”，不要只列色名。

示例结构：

```text
regional palette: Canyang Ao, earthen brown walls, late wheat gold dusk edge, medicine green baskets, old wood medicine cabinet; Qingming overlay only as insect wax white seals, cold white incense thread, black-green chitin guard silhouettes; old memory only as one moon-white glint at the well and a broken gold-red relay mark under the cabinet; no idyllic golden village, no ragged beggar slum, no high-magic glow
```

场景资产和参考图应优先生成以下区域色彩板：

1. `style_regional_color_route_map_v01`：第一季路线色彩总览。
2. `style_canyang_to_jinhe_palette_v01`：残阳坳到金河的暖土/河雾/生产控制过渡。
3. `style_qingming_institution_palette_v01`：清明院制度白与黑日系统。
4. `style_border_to_north_palette_v01`：边墙黑石雪线到北境枯草冻土过渡。
5. `style_southern_old_empire_palette_v01`：南方旧帝国保存色与第二季接口。
