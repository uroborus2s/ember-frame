# 《断航故土》阵营符号系统美术设定

版本：v01  
日期：2026-06-02  
范围：第一季《残阳驿火》项目级阵营符号延展  
交付状态：美术设定完成；PNG 生成提示词已写入 `prompts/project-faction-symbol-system-image-prompts.json`

## 1. 定位

本文件不重做已经完成的阵营徽章。现行三张徽章继续作为根参考：

| 阵营 | 当前徽章 |
| --- | --- |
| 昭明帝国 | `assets/style/faction-emblems/emblem_zhaoming_empire.png` |
| 肃明国 / 清明院 | `assets/style/faction-emblems/emblem_suming_qingming_state.png` |
| 北境万兽联盟 | `assets/style/faction-emblems/emblem_northern_beastfolk_alliance_wanshou.png` |

本轮要设计的是“阵营符号系统”：徽章如何延展到道具、服装、建筑、封条、旗布、军令、甲片、护符和场景细节里。生成 PNG 时，前三个阵营必须直接引入对应徽章图片作为输入参考；灰墙军和墙下混裔没有正式国徽，应做成非官方、物件化的符号系统，不能生成新的国家徽章。

## 2. 总规则

1. 图片里不要出现文字、标签、水印、UI 排版或说明性字幕。
2. 符号必须落在具体材质上：石、蜡、纸、布、金属、骨、木、陶、皮绳。
3. 保持低魔写实，不做高饱和发光法阵或现代 logo。
4. 阵营差异来自材质和使用场景，不只是换颜色。
5. 所有符号都要能服务竖屏近景：袖口、封印、墙角刻痕、甲片压痕、白册章、黑木牌、碎陶护符。

## 3. 昭明帝国日月体系

根参考：现行昭明航海日月徽章。

设计方向：高级旧文明、航权、星图、驿道、日月双印、朱赤漆、宝蓝织物、月白陶玉、磨损金。它不是无瑕正义圣徽，所有延展都要带裂纹、磨损、遮挡或旧债感。

可生成的符号板内容：

| 延展项 | 视觉 |
| --- | --- |
| 旧帝国主印 | 航海星盘式日月双印，磨损金与月白环 |
| 半枚驿令纹 | 断裂半环、残缺日月、边缘磨平 |
| 旧驿路线纹 | 金赤短线沿白石墙脚断续亮起 |
| 血牒暗纹 | 纸面干血红细线包住月白残印 |
| 雪泥残旗 | 朱赤旗布被雪泥压住，只露不完整日月 |

禁忌：完美圣光、现代国旗、普通太阳图标、英雄胜利旗、早期流亡装正面大徽。

## 4. 肃明国 / 清明院白翅覆黑日体系

根参考：现行用户确认的肃明/清明院白翅覆黑日徽章。

设计方向：白翅不是天使羽翼，而是制度化的翼刃、虫膜和覆盖层；黑日不是普通太阳，而是被洁净秩序压住的权力核心。整体要冷、净、硬、无尘。

可生成的符号板内容：

| 延展项 | 视觉 |
| --- | --- |
| 清明院中枢印 | 完整白翅覆黑日，珍珠灰裂环，少量冷金钉 |
| 白册章 | 黑日圆章嵌进白册框线 |
| 虫蜡封条 | 虫蜡白封条覆盖被刮掉的日月旧纹 |
| 清明香线 | 冷白细烟线缠过封条和衣角 |
| 甲军府烙牌 | 小型白翅黑日压在暗绿甲壳和旧铁束具上 |

禁忌：大虫身体当徽章、柔软天使翅膀、怪物巢穴、血腥实验室、现代医院蓝白空间。

## 5. 北境万兽联盟体系

根参考：现行北境万兽联盟骨木徽章。

设计方向：多族共同体，不是单一动物吉祥物。符号必须由骨、角、旧木、兽皮绳、灰烬、赭土和冰蓝火构成，像能绑在盾、鼓、皮帐、骨饰和誓约绳上的低技术标记。

可生成的符号板内容：

| 延展项 | 视觉 |
| --- | --- |
| 万兽誓结 | 骨木中央结，周围多族刻痕 |
| 角枝纹 | 鹿角、羊角、兽角弧线交错 |
| 爪齿刻 | 狼耳缺口、熊爪刻痕、野猪獠牙弧 |
| 羽影木片 | 乌鸦或鹰羽影刻在旧木片上 |
| 冰蓝火点 | 小范围生存火，不是攻击法术 |

禁忌：金属、高科技、宝石、单一狼头或狮头、可爱动物、怪物潮。

## 6. 灰墙军体系

根参考：无正式徽章；参考边墙、灰墙军、薛临墙视觉锁。

设计方向：灰墙军是守墙军户，不是复国英雄军团，也不是肃明亲兵。符号来自黑石墙、旧铁、骨哨、长枪、雪线绑腿、粮令、祖传旧甲片上的磨损日月压痕。

可生成的符号板内容：

| 延展项 | 视觉 |
| --- | --- |
| 墙脊纹 | 黑石墙块垂直压迫，上缘积雪 |
| 骨哨孔 | 骨白哨面两个风孔，磨损绳结 |
| 长枪墙线 | 长枪与拒马组成低矮防线 |
| 旧甲压痕 | 甲片上几乎看不清的旧日月凹痕 |
| 粮令缺口 | 粮牌被墙纹切开，暗示守墙换粮 |

禁忌：完整昭明徽章、闪亮英雄军徽、肃明白翅黑日从属标、乞丐破布民兵。

## 7. 墙下混裔体系

根参考：无正式徽章；参考墙下集市、陆青砾、墙下无姓人视觉锁。

设计方向：墙下混裔不应有国家级徽章。他们的符号是能藏、能换粮、能交易、能割断的生活标记：灰蓝棚布、陶土、黑木牌、碎陶护符、断旗绳结、隐藏口袋和耳后细鳞暗痕。

可生成的符号板内容：

| 延展项 | 视觉 |
| --- | --- |
| 碎陶护符 | 不规则陶片，两孔、折线、磨边 |
| 黑木通行牌 | 窄木片，一角被削，穿灰蓝绳 |
| 断旗绳结 | 撕下的朱赤日月旗碎片被灰蓝绳压住 |
| 细鳞暗痕 | 很小的暗绿弧片，只用于近景身份提示 |
| 新血折线 | 不闭合三段折线，像岔路和裂墙 |

禁忌：正式国徽、神圣联盟徽、现代帮派 logo、贫民窟乞讨标记、怪物化混裔标志。
