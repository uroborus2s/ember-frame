版本：v01  
日期：2026-06-03  
范围：项目级可复用角色资产，补齐北境万兽联盟攻城兵种的通用建模与服装系统。

## 1. 定位

本轮不重新设计任何阵营徽章。已经完成的北境万兽联盟骨木徽章继续作为锁定根参考，只能以小型物理引用出现在骨木誓结、盾内侧、战鼓边、工具吊牌或旧木刻片上。

“北境攻城兵种通用”不是一个无名怪物潮，也不是单一兽人战士。它是一张可复用的攻城兵种建模板，用于后续锁喉关、边墙、鸣骨岭、寒鸦堡外线和北境军势镜头。画板必须让 Prompt Room 能稳定拆出可识别的攻城角色模型：黑牙破门斧兵、铁角盾墙兵、雪獾掘地兵、雪羽攀墙斥候，以及小型辅助道具细节。每个模型都应有清晰物种、体态、武器和服装锚点。

## 3. 通用攻城服装系统

北境攻城兵种的“服装”不是统一军服，而是由部族物种、共生兽后勤和攻城职能组合出的可读装备系统。

- 共通层：冻裂兽皮内层、厚毛边披挂、骨片护肩、旧木护胸或背架、兽皮绳、暗血誓绳、雪泥绑腿、旧铁钉和黑石粉尘。
- 阵营识别：小型骨木誓结或旧木刻片可引用已锁定万兽徽章语言，但不能变成新胸章、新旗徽或单一动物 logo。
- 北境色彩：枯草黄、冻土灰、风蚀褐、兽皮暗棕、骨白、少量冰蓝火和暗血红。不能把北境统一成纯白雪原或高饱和蓝紫魔法战场。
- 攻城磨损：盾面箭痕、门闩碎木、绳索磨毛、骨片刮痕、黑石雪尘、旧铁锈迹。不能做血腥恐怖或怪物屠戮画面。

## 4. 可识别兵种模型

### 4.1 黑牙破门斧兵 / 破门状态

- 物种：高大半兽人重步兵，獠牙外露，肩背覆厚毛。
- 体态：前肩宽、颈短、臂长，站姿像跟随猛犸脚步推进的破门者。
- 服装：巨角盔、厚皮肩披、骨片肩甲、旧木胸板、暗血誓绳、黑石雪泥绑腿。
- 武器：骨柄大斧或破门槌，刃口有旧铁包边和门闩木屑。
- 识别锚点：外露獠牙、巨角盔、双手破门器、厚毛肩背、跟随巨兽的沉重重心。

### 4.2 铁角盾墙兵 / 顶盾状态

- 物种：牛族与牦牛共养部落出身，额角短粗，体格宽厚。
- 体态：低重心、胸背厚，脚步像顶住城墙箭雨的移动墙。
- 服装：半人高骨铁盾、牦牛毛护颈、旧铁盾箍、骨白额角护套、皮绳盾背背带。
- 武器：长柄锤或短撞锤，盾面可挂小型骨木誓结。
- 识别锚点：短粗额角、半人高骨铁盾、牦牛毛护颈、方厚盾墙轮廓。

### 4.3 雪獾掘地兵 / 破基状态

- 物种：矮厚雪獾族，强爪，宽掌，低伏。
- 体态：低矮结实，肩腰同宽，适合排水沟、雪坑和粮仓地基突入。
- 服装：短皮甲、黑石楔袋、冰下呼吸囊、骨铲背挂、膝肘厚皮补片。
- 武器与工具：骨铲、黑石楔、短凿、粗绳索。
- 识别锚点：矮厚比例、强爪、工具带、背部呼吸囊、黑石楔。

### 4.4 雪羽攀墙斥候 / 攀墙状态

- 物种：鹰族或鸦族轻兵，骨轻，臂披羽纹斗篷。
- 体态：瘦长、前臂展开，重心贴墙，适合爪钩、绳索和短翼滑翔。
- 服装：羽纹短斗篷、骨爪钩、细皮绳索、轻骨护臂、窄腰工具包。
- 武器与工具：爪钩、短刀、小火罐、割旗刀。
- 识别锚点：羽纹臂披、爪钩、轻骨护臂、贴墙动作，不做大规模飞行兵。

## 5. 画板结构

生成一张竖版多状态角色板，图片内部不出现文字、标签、字幕、UI 或水印。

### 四个全身状态

1. 黑牙破门斧兵：正面或三分之二角度，全身重步兵，双手持骨柄破门大斧或破门槌，巨角盔和獠牙可读。
2. 铁角盾墙兵：顶盾推进状态，半人高骨铁盾占据轮廓，短粗额角与牦牛毛护颈可读。
3. 雪獾掘地兵：低伏破基状态，强爪、骨铲、黑石楔袋和冰下呼吸囊可读。
4. 雪羽攀墙斥候：攀墙或投绳状态，羽纹臂披、爪钩、轻骨护臂和细绳索可读。

### 细节面板

- 巨角盔、獠牙与厚毛肩背
- 骨铁盾正反面和盾背皮绳
- 骨柄大斧 / 破门槌头
- 黑石楔、骨铲和冰下呼吸囊
- 雪羽爪钩、羽纹臂披和短翼绳索
- 小型骨木誓结 / 旧木刻片：只引用既有万兽徽章语言，不重画新徽章
- 兽皮绳、暗血誓绳、雪泥绑腿、旧铁钉
- 材质色条：枯草黄、冻土灰、风蚀褐、兽皮暗棕、骨白、旧木、旧铁、暗血红、冰蓝火、黑石雪尘

## 6. 阶层与物种差异规则

- 与鹿弥区分：攻城兵不能贵族化、萨满化、清秀化或华丽皮草化；不能带骨笛作为主身份。
- 与灰墙军区分：北境攻城兵不是人族守墙军，不能穿灰墙军厚衬袄、旧日月甲片或人族军户制服。
- 与肃明虫族区分：不能出现虫蜡白制度绑带、黑绿甲壳、虫族复眼或白翅覆黑日引用。
- 与普通怪物区分：必须是有装备逻辑、后勤逻辑和部族身份的攻城兵，不是无脑兽潮。

## 7. 禁忌

- 不重新设计北境万兽联盟徽章。
- 不出现新万兽徽章、新动物 crest、大胸章、大旗帜徽、现代 logo 或 sticker overlay。
- 不把北境画成单一狼头、狮头、鹿头或兽人游戏公会。
- 不把四个兵种画成同一种兽人换武器；每个模型必须有可识别的物种比例和职能装备。
- 不出现文字、标签、说明字、中文、英文字母、UI、水印。
- 不做高饱和游戏海报、科幻金属甲、现代军装、血腥怪物潮、浪漫雪原或大规模空军。

## 8. 下游使用

Prompt Room 应把此角色板作为北境攻城镜头的通用建模锁。镜头级提示词可以按需要只抽取其中一个兵种，但必须保留该兵种的物种比例、装备锚点和北境骨木皮草材质。若完整万兽徽章进入镜头，只能作为已有骨木誓结或旧木刻片的小型物理引用，不得重新设计、放大或变形。
