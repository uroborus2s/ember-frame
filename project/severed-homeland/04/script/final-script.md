# 第04集：金河磨齿

时长目标：4分25秒
格式：AI竖屏短剧详细剧本
本集功能：把灭械令和肃明生产控制具象化；沈维桑与晏南枝通过粮磨区脱身，追捕线登上渡船。

## 场景表

1. 金河粮磨区：磨平军械接口钩子。
2. 粮牌检查口：伪籍危机。
3. 蒸汽磨盘区：换牌脱身。
4. 渡河蒸汽船：清明香铃响起。

## 成片剧本

### SC01 金河粮磨区 / 白雾清晨 / 5秒钩子

巨大蒸汽齿轮转动。齿轮旁，一处被磨平的炮架接口露出旧帝国军械编号，像被剃掉的牙根。

沈维桑停住脚，手指摸过编号。

沈维桑：这不是磨粮用的。

晏南枝看见编号下残留的日月纹，声音低下去。

晏南枝：昭明军工的旧号。肃明下了灭械令，把炮架、枪管都磨成水轮磨盘，只许人推粮，不许人守粮。

沈维桑：他们要人能产粮，不能守粮。

远处，粮税小吏举起骨算盘。

粮税小吏：渡口查牌。没粮牌的，人粮都别过河。

渡口钟敲三下，白蜡闸门开始下落。

粮税小吏：半刻后落闸。临时牌一律换清明蜡章，白盘验不过，人粮一起扣下。

沈维桑看向渡口钟。

沈维桑：我们只有这一船。

### SC02 粮牌检查口 / 上午

逃荒者排成曲线。白册官吏查旧姓，粮税小吏量袋重，奴兵扣住手腕闻气味牌。蒸汽白雾从阀门间喷出，遮一下又散开。

沈维桑递上两块临时粮牌。

粮税小吏：一个山猎户，一个南缘女眷。女眷抬脸。

晏南枝垂着眼，雾紫外袍遮住大半面容。她一抬脸，周围几个苦役和船工都有一瞬失神。

粮税小吏的复眼停住。

粮税小吏：这张脸，不像南缘小户。

晏南枝：不像，是因为南缘小户也曾有姓。

沈维桑在她袖下掐了她一下。

沈维桑：她病后嘴硬。官爷看牌。

粮税小吏把伪牌放上虫蜡白盘。白盘浮出裂纹。

粮税小吏：伪籍。

奴兵同时上前。沈维桑看向蒸汽阀，看向粮袋轨车，看向渡口钟。

### SC03 蒸汽磨盘区 / 连续

沈维桑突然把一袋潮粮踢进磨盘边。湿粮卡住齿轮，蒸汽阀爆出大片白雾。

粮税小吏：关阀！

苦役四散。沈维桑拖着晏南枝钻进粮袋轨车下，把自己的伪牌塞进一个空袋，又从船工腰绳上换下一块湿木船牌。

晏南枝把血牒按回衣内，转身扶起被撞倒的苦役，又把半袋潮粮推到追来的奴兵脚前。

晏南枝：我不是让他们跪的。

沈维桑看她一眼，没有说话。

晏南枝：偷牌？

沈维桑：借命。

一名渡船船工发现他们，手已经摸向喊铃。沈维桑没有拔刀，只把半枚驿令露出一点。

沈维桑：你船底有旧驿水标。你祖上不只是运粮的。

船工脸色变了。

船工压低声音。

船工：我爹的驿名，三十年没人敢喊。

沈维桑：今日不喊名，只借一条路。

船工把一捆湿绳丢到他们脚边，挡住奴兵视线。

船工把湿木牌塞回他掌心。

船工：路在船底，人不在册上。船开后，别上甲板。

晏南枝回头看磨平的炮架接口。

晏南枝：旧帝国靠的不只是血脉。

沈维桑：你终于看见人了。

晏南枝没有反驳。

### SC04 渡河蒸汽船 / 黄昏

渡船离岸。金河水面深青，河雾贴着船舷。船舱里挤满逃荒者、粮袋和低声哭的孩子。

晏南枝坐在暗处，手指收紧发上红线。

晏南枝：工匠、船工、驿人都还记得路，复翼者为什么只让我带血牒？

沈维桑盯着船舱门。

沈维桑：因为血牒好让人下跪。路不让人跪，路只让人走。

船舱另一头，一个诱线奴兵换下湿军户围巾，露出蜡灰青短军袍。它腰侧的清明香铃轻轻响了一声。

沈维桑抬手，示意晏南枝别动。

船工在甲板上大喊。

船工：雾大，今晚不停岸。

白翳的声音却从船舱外响起，平静得像白纸落水。

白翳：不停岸，就在河上验。

舱门被虫族小兵的钩镰从外面扣住。清明香铃一声接一声，像追上来的白册翻页。

沈维桑握住短刀，低声对晏南枝说。

沈维桑：这次不能跳河。金河太冷。

晏南枝拔出袖中短匕。

晏南枝：那就让他们下船。

船舱灯火一晃，白翳的影子落到门缝上。

切黑。
