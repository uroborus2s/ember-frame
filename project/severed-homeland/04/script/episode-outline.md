# 第04集 Outline：金河粮磨

1. 钩子：磨平炮架刻度的蒸汽磨盘转动。目的：一句视觉说明灭械令。情绪：压迫。视觉：齿轮、蒸汽、磨痕。
2. 潜入换粮：沈维桑和晏南枝伪装苦役入内。目的：补给与新环境。情绪：低调紧张。视觉：粮牌、人族苦役。
3. 封械地窖：沈维桑发现断枪管和烧黑炮模。目的：揭示战争文明断代。情绪：震动。视觉：旧铭牌、虫蜡。
4. 孟归藏登场：亡书人用旧书箱挡住监察巡查。目的：引入旧书会。情绪：可疑盟友。视觉：烧黑书箱。
5. 白翳追近：清明香在磨坊蒸汽中变成细白线，逼出沈维桑位置。目的：追捕升级。情绪：急迫。视觉：白烟与蒸汽混合。
6. 磨坊脱身：沈维桑射断皮带，齿轮停摆造成混乱。目的：动作脱困。情绪：险胜。视觉：巨轮停转、粮粉飞扬。
7. 结尾转折：孟归藏说清明院最喜欢记性好的孩子，沈维桑想起照眠背草药名。目的：引出第05-06集。视觉：小草药袋特写。
