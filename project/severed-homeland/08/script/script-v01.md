时长目标：4分50秒
格式：AI竖屏短剧详细剧本
本集功能：把边墙战争、肃明控粮和灰墙军现实推到主角面前；鹿弥作为北境伤者出场并让“兽潮”叙事裂开。

## 场景表

1. 锁喉关主门：破门猛犸撞响骨钟。
2. 粮仓门：薛临墙考虑交出晏南枝换粮。
3. 墙顶雪道：沈维桑守侧门。
4. 旧日月阵基：鹿族伤者吹骨笛。

## 成片剧本

### SC01 锁喉关主门 / 暴雪 / 5秒钩子

骨钟在暴雪中自鸣。门外，破门猛犸的铁包巨角顶上拒马，黑牙斧兵在巨兽脚边砍门闩。

墙上灰墙军户冻得手指开裂，仍把长矛压下去。雪羽攀墙斥候的钩爪挂上墙垛。

薛临墙吹骨哨，声音短硬。

薛临墙：补左垛，水袋倒门缝，别让铁角贴门！

沈维桑、晏南枝和陆青砾被军户押上墙道。陆青砾小声说。

陆青砾：欢迎来到粮门下面的旧驿入口。

### SC02 锁喉关粮仓门 / 稍后

粮仓门内堆着肃明封蜡粮袋。门外伤兵等粮。粮税小吏站在门槛内，身后重甲虫士兵像门闩一样堵住路。

粮税小吏：未交昭明血牒疑犯，粮仓不开。

薛临墙看向晏南枝，又看向墙上不断倒下的军户。

薛临墙：一个女娃，换一仓粮。很划算。

晏南枝抬起下颌。

晏南枝：灰墙军也曾守日月旗。

薛临墙冷笑。

薛临墙：日月旗没给我孙子发过粮。

沈维桑挡到晏南枝前面。

沈维桑：交她，清明院也不会开仓。他们只会知道你还能再饿一天。

粮仓外，墙顶一声惨叫。侧门排水沟被雪獾掘地兵破开，虫族小兵也趁乱向旧驿入口方向压来。

薛临墙盯住沈维桑。

薛临墙：你会看墙？

沈维桑：我会看山水。墙也是会塌的山。

### SC03 墙顶雪道 / 连续

沈维桑被推上侧门雪道。风雪打得人睁不开眼。他看见排水沟下方的雪色发灰，立刻判断掘地兵已经接近门基。

沈维桑：火油别倒门，倒沟。沟里有空声。

老军户迟疑。薛临墙直接下令。

薛临墙：听他的。

火油灌入排水沟，沈维桑点燃旧布。火线沿沟底窜开，逼出雪獾掘地兵。北境黑牙斧兵趁乱冲门，沈维桑用猎弓射断吊石绳，落石砸偏攻城木。

他的左肩被震得僵住，险些坠墙。晏南枝扑上来拉住他，手掌被黑石磨破。

晏南枝：你救的是要交出我的人。

沈维桑喘着气。

沈维桑：他们也被交出去很多次了。

陆青砾在一旁割断雪羽斥候的钩绳。

陆青砾：少聊天，门还活着。

### SC04 旧日月阵基裸露处 / 暴雪稍停

战场短暂停顿。墙下，一个鹿族年轻女子倒在旧阵基旁，柔软鹿耳沾血，骨笛挂在胸前。她不是攻城兵，更像被冲散的萨满。

军户要补枪。晏南枝本能抬手阻止。

晏南枝：她不是斧兵。

薛临墙：墙外的人，都是风险。

沈维桑已经翻下半截雪坡，把鹿族女子拖到阵基背风处。她睁眼，浅色鹿眼警惕。

鹿弥：别碰骨笛。

沈维桑：我碰你伤口，不碰笛。

门外破门猛犸再度被驱赶，巨角顶向拒马。鹿弥挣扎着摸出骨笛，吹出一段低而颤的音。

巨兽忽然停步。驭兽者也愣住，黑牙斧兵的节奏乱了。

墙上所有人都看见了。

陆青砾：兽潮会听笛？

鹿弥咬牙。

鹿弥：那不是兽潮。是饿疯的人，把也会疼的兽推到墙上。

薛临墙沉默。

粮税小吏见攻势停顿，仍不肯开仓。

粮税小吏：疑犯未交，粮门照封。

薛临墙把长枪重重插到粮仓门前。

薛临墙：今天先守墙。明天再算谁是疑犯。

他看向沈维桑。

薛临墙：旧驿入口在粮门下。你若骗我，我拿你补墙。

沈维桑扶起鹿弥。

沈维桑：先让她活过今晚。

鹿弥看见晏南枝袖口露出的日月残纹，眼神忽然变了。

鹿弥：你们找月下？

晏南枝：你知道？

鹿弥按住骨笛，低声说。

鹿弥：那不是你们皇族的地方。

暴雪再次盖住墙门。沈维桑、晏南枝和薛临墙同时看向她。

切黑。
