# 第08集 Outline：锁喉关雪夜

1. 钩子：巨兽撞门，旧日月阵基露出。目的：边墙危机和旧帝国工程并置。情绪：震撼。视觉：雪夜、城门裂纹。
2. 混入关内：陆青砾用黑市牌带二人进关。目的：进入边墙。情绪：紧张。视觉：骨钟、黑石墙。
3. 薛临墙识破：老墙师看出沈维桑站架和晏南枝贵族礼。目的：新导师和阻力。情绪：冷硬。视觉：冻裂手、长枪。
4. 锁粮冲突：厉螳宣布守墙不力减粮，灰墙军沉默。目的：展示虫族控制方式。情绪：愤怒压住。视觉：粮牌被掰断。
5. 雪夜守墙：北境巨兽撞门，沈维桑用猎弓射信绳，薛临墙以化劲卸冲。目的：动作高潮。情绪：生死协作。
6. 武脉代价：沈维桑强发明劲后咳血，薛临墙说“你这样练，活不到三十”。目的：成长代价。视觉：血落雪面。
7. 结尾转折：日月阵基与驿令共鸣，暗道指向边墙背后。目的：导向鸣骨岭和旧驿。视觉：金赤光从雪下透出。
