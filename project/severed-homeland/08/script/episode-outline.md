# 第08集 Episode Outline：锁喉关的粮

## 结构节拍

### 1. 5秒钩子

骨钟在暴雪中自鸣，门外破门猛犸的铁包巨角顶上拒马。

### 2. 30秒危机

北境攻城开始，肃明粮门拒绝给人族军户开仓，薛临墙考虑交出晏南枝换粮。

### 3. 中段升级

沈维桑用猎户判断和残缺明劲守住侧门，晏南枝看见灰墙军的生存困境。

### 4. 结尾悬念

被救下的鹿族伤者吹响骨笛，巨兽短暂停步，说明攻城不是无脑兽潮。

## 分场方向

- 主要角色：沈维桑、晏南枝、薛临墙、陆青砾、鹿弥
- 主要场景：锁喉关主门、粮仓门、墙顶雪道、旧日月阵基裸露处
- 动作要求：所有重大信息必须落在可见动作、道具、空间选择或对白冲突上。
- 节奏要求：开场立刻进入危险或问题，中段用追捕、交易、审讯、救援或空间坍塌升级，结尾必须保留下一集推动力。

## 角色推进

本集推进由“眼前求生”转向“故土定义被不断扩展”。沈维桑每次选择都优先保护活人和路线；晏南枝每次遭遇都削弱单一血统复国逻辑；白翳和清明院的压力必须持续把旧名、旧路和亲情变成可追捕证据。

## 连续性锚点

- 半枚日月驿令、血牒残段、清明香、白册、虫蜡针、旧驿暗语和月下多族印记均按出现顺序递进。
- 沈照眠第一季不得真正团圆，只能以识别者伤口和追捕线推动后续。
- 月下盟书不得写成皇族复国诏书。
