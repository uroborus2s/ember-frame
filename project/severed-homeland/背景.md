# 背景设定 Bible

## 0. 构思分析

这个故事的核心不是“人族反抗虫族”这么简单，而是一个文明被篡夺、被改名、被断航、被灭史之后，幸存者如何重新理解“故土”的故事。

最强戏剧点有四个：

1. 身份反转：虫族占据中央大陆后以“正统文明继承者”自居，真正的人族反而被降为边民、奴民、军户、海匪和失史之人。
2. 文明断代：虫族不只焚书，还焚毁航权、造船、星图、港籍、云裔法统和东海记忆，让中央人族失去靠近海岸与出海的能力。
3. 护送旅程：故事从虫族监察搜查逃犯开始，搜查导致村落被屠，主角被迫护送一名来自南方的旧帝国血脉穿越大陆。
4. 故土重释：人族旧帝国以“日月照耀皆为故土”为核心视觉符号；主角最终要让这句话从帝国扩张口号，变成不同族群共同守护土地的誓言。

本版修正一条核心规则：**世界上没有独立仙族。** 后世所谓仙族，是旧帝国人族中的云裔法统、天工山门、龙船匠和结界工程师被虫族伪史切割后的误称。结界海、龙船、边墙、星图和远航术都属于旧人族文明遗产。

## 1. 世界总览

整个世界主体被称为六域大陆，由六个核心大区和一个东海岛链带组成：

1. 中央大陆：最富饶、最文明的旧人族核心地带，现由虫族中央政权统治。
2. 东部瀛洲：中央大陆东侧的封闭岛国群，隔结界海相望，由瀛洲兽族诸藩统治。
3. 北部极寒：荒原、雪地、林地、草原与狂暴瀚海组成的寒灾之地，北部诸族与巨兽群共生。
4. 东北沼泽：虫族发源地，兼具沼泽、黑土地、山地、平原、林地和沙地。
5. 南部丛林地：多雨密林，精灵聚居，并庇护部分保留旧文明的人族遗民。
6. 西部高原地：连绵大山与地下矿城，矮人王国和高原种族的主要生活地。
7. 东海岛链：中央大陆东海岸与瀛洲之间的海岛带，人族旧港遗民在千年封锁中变成海盗、走私者和雾航向导。

### 1.1 当前时代

当前时代为虫族统领中央大陆后的第 1200 年。人族旧帝国的书籍、法术、史官、贵族系统、云裔法统、龙船船坞、潮星灯塔和港籍总册都已被摧毁或改写。

第一季开端对应帝国历 1226 年。北方十三部落联盟借千年雪灾与边墙疲敝大举入寇，肃明国以边墙军需为名加重征粮、征役和清明籍筛查；中央大陆人族村落普遍处于饥荒、欠税、失籍和随时被监察的状态。南方保留旧文明的人族遗民也在这一年加紧联络旧驿、禁歌和药包暗语，开始酝酿反抗。

虫族中央政权国号定为：**肃明国**。

其建国口号为：**肃尽前明，还天下清明**。在肃明国官方语境中，“明”既指昭明帝国的日月旧朝，也指前朝遗留下来的血脉、礼法、船坞、驿道、私学、禁歌和地方记忆。虫族政权声称，只有清除日月旧痕与前朝残影，天下才能进入“清明世界”。“肃明”是手段，“清明”是承诺。

肃明国的官方视觉为“白翅覆黑日”：白色虫翅压住黑色日轮，月纹被故意抹去，象征虫族以洁净秩序覆盖旧人族日月旗。

### 1.2 人族旧帝国核心符号

人族旧帝国国号定为：**昭明帝国**。

“昭”取日月所照，“明”取日月并悬。昭明帝国的国号本身就把光照、秩序和故土绑定在一起，因此虫族后来把它污称为“前明”或“日月旧朝”，并把清除日月符号视为建国合法性的第一步。

旧帝国核心宣言：**日月照耀皆为故土**。

帝国军旗为“日轮托白月”。城门、驿站、边碑、军械、船坞、星图台和边墙阵基上都刻有日月双印。它原本的含义是：凡日光可达、月色可临之地，皆受帝国庇护，也皆为帝国应归之土。

这句话具有双面性。它曾保护道路、港口和边境，也曾成为帝国扩张、吞并小族和控制海陆的正当性。主线旅程会逐步重释这句话：故土不该只由帝国旗帜定义，而应由活人之间的承诺定义。

### 1.3 世界级冲突

1. 正统之争：谁才是中央大陆文明的真正创造者与继承者？
2. 记忆之争：虫族用遗忘法术、清明籍和伪史维持统治，人族用歌谣、驿令、旧图、星纹和血脉梦保存真相。
3. 航权之争：旧帝国云裔人族的龙船技术被焚毁，结界海失控，海族洋流与瀛洲锁国共同封锁东方。
4. 生存之争：北部极寒的千年雪灾持续压迫北境诸族南下，边墙战争没有真正结束。
5. 清除之争：肃明国试图用“清明籍”登记、筛查、抹除所有旧帝国血脉与地方记忆。
6. 故土之争：旧帝国的“日月照耀皆为故土”和肃明国的“肃尽前明，还天下清明”，代表两套互相冲突的世界秩序。
7. 武力断代之争：旧帝国曾拥有火枪、大炮、蒸汽机和战争工坊，肃明国在昭明灭国后摧毁大部分战争文明，只保留能服务生产和运输的蒸汽技术，使现今战争退化为冷兵器为主。

### 1.4 法术与记忆规则

人族旧法以文字、歌谣、星图、驿道、船舶、结界、呼吸和灵翼经络为基础。所谓“翅膀”不是每个人都具备实体羽翼，而是高阶法师显化法力时展开的灵翼纹路。觉醒者施法时，淡金、银白或赤曜色纹路会沿肩胛、脊柱、手腕和眼角展开。

虫族篡夺后，通过屠杀法师、焚毁典籍、改写文字、毁灭船坞和施加锁翼咒，使多数人族后代无法完整施法。锁翼咒压制的不是单个术式，而是人族把语言、道路、建筑、船舶、仪式和集体记忆编织成文明级法术的能力。

遗忘法术不是万能的。它能压制语言记忆、历史认知和血脉觉醒，却无法彻底抹除旋律、身体动作、地名、梦境、石刻、织物图案、潮汐直觉和血脉反应。因此南部人族把历史藏入歌谣和仪式，岛链人族把航路藏入禁歌和船骨，边墙人族则在不理解含义的情况下继续传唱旧歌。

### 1.5 武力背景与战争形式

昭明旧帝国不是单纯法术文明，而是法术与科技高度结合的复合文明。少部分人族拥有完整灵翼经络，能够施展法术；大部分普通人族没有稳定法术天赋，只能依靠科技、阵地、纪律、火器、蒸汽机械、工程城防和组织能力，对抗体形、力量、速度和天赋能力都不对等的外族。

法术在旧帝国战争中重要，但不能单独决定战争。它主要承担六类辅助功能：

1. 通讯：用烽燧法阵、星图台和驿道符标传递军情。
2. 侦察：借风、水、鸟兽和星象判断敌军方向。
3. 防护：短时间加固城门、盾墙、船体和边墙阵基。
4. 点火与校准：辅助火枪、大炮、蒸汽炮车和投石机点火、测距、校正风偏。
5. 医疗与恢复：延缓失血、压制毒素、稳定伤兵。
6. 工程：快速测绘地形、修复桥梁、排水、开矿和筑垒。

旧时代战争已经使用火枪、大炮和蒸汽机。昭明军队曾拥有火门枪、燧发火枪、轮式炮车、城防巨炮、蒸汽牵引车、蒸汽抽水机、矿道轨车和蒸汽锻炉。旧帝国强大之处不是人人会法术，而是能把少数法师、普通士兵、工程师、工匠、枪炮、蒸汽和道路系统组织成大规模战争机器。

灭国之后，肃明国发动“灭械令”。虫族销毁火枪图纸、炮铸模具、军用蒸汽机、火药作坊、测距器、军械学院和战争工坊，处决或迁徙掌握军工技术的人族工匠。虫族并未完全毁灭蒸汽文明，而是保留了服务生产、采矿、排水、粮仓、运输和城建的部分蒸汽技术。现今中央大陆还能看到蒸汽磨坊、矿井抽水机、粮仓升降机和轨车残余，但大规模军用火器几乎绝迹。

肃明国这样做有两个目的：一是防止数量庞大的人族奴民和边墙军户重新获得能弥补体形差距的火器；二是让战争回到虫族、北部巨兽、瀛洲兽族等体能优势族群更占便宜的冷兵器环境。

现今战争以冷兵器为主，常见形式包括：

1. 边墙守城：长矛、弩、滚石、热油、骨钟、陷坑和少量旧式床弩。
2. 野地遭遇：盾阵、长枪阵、骑兽冲锋、弓弩齐射和伏击。
3. 城寨攻防：云梯、冲车、破门兽、矿道爆破和巷战。
4. 巨兽战争：北部诸族驱使猛犸、岩背犀、冰湖甲龟等巨兽冲击城门和阵列。
5. 虫族清剿：螳刃突袭、甲胄虫正面推进、蜂巢术士压制记忆与士气。
6. 岛链海战：小船穿雾、钩索登船、火油罐、弩炮、礁石伏击和残缺龙骨稳定船体。
7. 矮人阵地战：重甲盾墙、符文弩、矿道陷阱和少量基础火枪。

矮人帝国是少数仍保留基础火枪制作能力的势力。他们能制造结构简单、射速慢、维护困难的火门枪和短管火铳，也能铸造小型黑火炮。但火药配方、精密枪管和旧帝国军工标准已经断代，产量极低，且矮人受契约和贸易压力限制，不敢公开大规模武装人族。

五百年的冷兵器边境战争催生了边境人族遗民的武士进阶功法。这套功法不等同于旧帝国高阶法术，而是普通人族为了弥补体形与力量差距，在极端战争环境中发展出的身体强化体系，统称为“边墙武脉”。

边墙武脉充分参考旧帝国古老的内家拳理，修炼路径是从外而内。外家阶段依靠站桩、盘架、行桩、负重、寒暑打熬、筋骨锻打和短爆发步法，能在短时间内提高力量、耐力、反应和抗击打能力，但代价是损耗寿命、留下旧伤、筋膜僵硬、寒毒入骨，严重者内脏衰竭。绝大多数边墙武士都止步于外家，他们是五百年消耗战逼出来的残酷答案。

真正进入内家后，武士不再只靠透支肉身作战，而是以呼吸、松沉、听劲、气血归根和心意调和修复身体。外家打熬造成的骨裂、筋膜硬化、寒伤和脏腑淤伤会被内息缓慢修复。但入内家极难，需要强大的机缘、师承、心态和旧帝国“天人合一”的哲学配合。心中只有复仇、恐惧或求胜的人，往往越练越伤；能入内家的人，必须先学会让自己与风雪、山墙、呼吸、地脉和活人承诺重新相合。

边墙武士常见进阶：

1. 筑架：外家入门。站桩、盘架、负重行走，先把脚、胯、脊、肩、肘连成一体，能长时间守墙、挨冻和持械。
2. 明劲：外家第一成。整身发力，劲从脚起，贯通腰背到刀、枪、弓，能短时间爆发超出常人的力量。
3. 暗劲：外家深处。劲入筋膜和呼吸，发力更短、更隐、更狠，适合近身破甲和救人突围，但最容易伤肺、伤心脉。
4. 化劲：内外门槛。能听劲、卸力、借力，不再硬碰巨兽和刃肢。边墙军中能到此境者极少，通常会被称为“老墙师”。
5. 内息：内家初成。呼吸不只用于爆发，而能调和气血、温养旧伤，开始修复外家阶段留下的骨裂、寒毒和脏腑损伤。
6. 归元：内家深处。心意、呼吸、步法和兵器合一，战斗中能保持清明，不被恐惧和杀意牵动。
7. 合天：内家极境。对应昭明帝国古老的天人合一哲学，能在风雪、城墙、地脉和人群情绪中找到同一条“势”。这不是神仙法术，而是人身、心意与天地秩序短暂相合。

沈维桑的战斗起点不是法师，而是猎户、引路人和边境武脉的半成品。他只学过筑架和残缺明劲，尚未真正进入暗劲，更谈不上内家。他的灵翼残纹会逐步觉醒，但在很长一段旅程中，他仍主要依靠弓、短刀、陷阱、地形、旧驿道判断和粗浅武脉活下来。主线中他的武力成长，不是从普通人突然变成法师，而是从外家求生，逐步理解“天人合一”与故土承诺，才获得进入内家的可能。

## 2. 世界场景

### 2.1 中央大陆

中央大陆是旧人族帝国的核心，也是肃明国最重要的统治中心。这里有肥沃平原、密集河网、低丘、旧人族大道、古都群和大型粮仓。城市外观仍保留人族白石宫殿、拱廊、广场和法术塔，但虫族在门楣、宫墙和地下空间加上了甲壳纹、虫蜡、巢室和孵化厅。

统治势力：肃明国虫皇、巢母议会、甲军府、书壳院、清明院、各地监察司。

关键场景：

1. 白曜城：人族旧皇都，白石宫墙仍在，宫门被覆盖虫族甲壳纹饰，广场中央立着肃明国开国碑。
2. 灰烬书院：被焚毁的人族书库，半塌拱廊、焦黑书架、碎裂星图和残留封印阵纹是旧文明被毁的证据。
3. 蜕壳宫：虫族王庭议政地，外观像人族皇宫，内殿却布满半透明虫膜、巢室隔间和湿亮甲壳旗。
4. 清明院：筛查旧血脉、训练识别者、重写儿童记忆的机构。白墙无尘，地底却有蜂巢状记忆室。
5. 金河粮仓带：大河两岸的农田、磨坊、水车和征粮道，是虫族向边墙输送粮草、也向村落征收苦役的地方。
6. 残阳坳：主角出生村落，位于大陆中部偏南旧驿道上，表面归顺，暗中保留旧驿暗语。
7. 蒸汽粮磨区：肃明国允许保留的生产型蒸汽设施，齿轮、水泵、磨坊和升降机仍在运转，但所有军械齿轮、炮架接口和测距刻度都被磨平。
8. 封械地窖：旧帝国军械库遗址，被虫族封入地下，只剩碎炮轮、断枪管、烧黑炮模和被虫蜡覆盖的军械铭牌。

冲突用途：监察搜查、伪史揭露、焚书旧案、清明籍计划、街头追捕、虫族内部派系斗争。

视觉锚点：白石古城、虫甲纹宫门、焦黑书库、被凿掉脸的旧人族雕像、黑甲虫族卫兵、清明籍白册、人族奴印、湿黑虫蜡、只准生产不准战争的蒸汽齿轮。

### 2.2 绝境山脉与边墙

绝境山脉横亘于中央大陆与北部极寒之间，是天然屏障，也是旧帝国边墙工程的核心区域。旧帝国云裔人族与山门工程师曾依托山脉修筑边墙和城堡，用来抵御北境灾变。虫族接管中央大陆后，将这条防线变成人族军户的牢笼和消耗场。

地貌：高耸雪峰、刀削峭壁、狭窄山口、深谷、碎石坡、冰瀑、废弃矿洞、山腰梯田。边墙沿山脊和山口修筑，城堡像钉子一样嵌进山体。

统治势力：肃明国北境军府。实际防守主体是被迁徙到此的人族军户、征发兵、矿奴和堡寨民。虫族掌握粮仓、军令、识字权和法术禁制。

关键场景：

1. 断脊边墙：巨型城墙沿山脊蜿蜒，墙体由黑石、旧人族白石和虫族甲壳状金属混筑。墙面布满五百年战争留下的爪痕、火烧痕和撞击坑。
2. 锁喉关：绝境山脉最窄山口，边墙主门所在。铁闸上刻着肃明国白翅黑日徽记，剥落处露出旧帝国日月双印。
3. 寒鸦堡：边墙最高军事堡垒，虫族北境将军驻地。外观是人族旧堡，内部被改造成虫巢式军府，有地下孵化室、审讯厅和虫膜走廊。
4. 鸣骨岭：边墙外三十里，历代兽潮和守军尸骨堆积成岭，风吹过骨孔会发出笛声般的声音。
5. 旧帝国烽燧群：人族帝国时期修建的信号塔，如今多被虫族占用，部分地下仍保留人族法术阵。
6. 月下盟书旧驿群：隐藏在边墙背后的旧驿站网络，传说存放旧帝国末期各族共同防御盟书。
7. 武脉校场：边墙人族私下训练身体强化功法的灰石场，表面是搬石与修墙劳作地，地下刻有站桩步位、明劲走架、暗劲吐纳图，以及被虫族刮掉的“天人合一”古训。

冲突用途：边境大战、兽潮压迫、武脉训练、主角寻找父亲旧图、晏南枝寻找月下盟书、虫族利用人族消耗北境。

视觉锚点：雪山边墙、黑石城堡、骨钟、冻裂旗帜、旧日月碑文、虫族军吏、人族军户、负重练功的边墙武士、暴雪中的巨兽黑影。

### 2.3 北部极寒

北部极寒由荒原、冻土、雪林、冰湖、草原裂谷与狂暴瀚海组成。这里物资极度匮乏，常年寒冷，千年雪灾使食物链崩坏，北部诸族与巨兽群被迫不断南下。

北部诸族不是简单“兽潮”。他们是被寒灾驱赶的驭兽部落联盟，最突出的共同能力，是通过血誓、骨笛、萨满梦境和兽群盟约控制体形庞大的野兽。

主要场景：

1. 黑牙兽庭：半兽人部落集会地，兽骨柱、皮帐、火坑、战利品和极光祭坛组成部落权力中心。
2. 蛇族冰窟：蓝黑冰洞与地热温泉交错，洞壁有蛇形刻痕，热雾在寒风中翻涌。
3. 飞龙断崖：数百个岩洞巢穴盘踞悬崖，天空常有飞龙剪影。
4. 瀚海冰潮：风暴中的黑色海面，浮冰相撞，远处有巨兽背脊般的浪影。
5. 万骨雪原：历代兽潮死者遗骸半埋在雪中，锈兵器露出地面。
6. 骨笛谷：北部萨满与巨兽立誓之地，风吹骨孔会形成低沉音阶。

冲突用途：兽潮压力、北境灾难真相、驭兽部落政治、巨兽冲锋、主角理解敌人也是寒灾受害者。

视觉锚点：暴雪、兽骨图腾、皮毛甲、冰蓝火光、飞龙剪影、冻裂大地、巨角猛犸、半兽人战鼓。

### 2.4 东北沼泽

东北沼泽位于中央大陆东北部，天气寒冷潮湿，却有肥沃黑土地和发达水系，形成沼泽、山地、平原、林地和沙地交错的奇特地貌。这里是虫族发源地，也是旧帝国流放贵族启蒙虫族的地方。

统治势力：虫族母巢、沼祖派巢群、书壳院旧档分院、泥巢民聚落。

关键场景：

1. 母巢黑台：虫族圣地，黑色土丘上建有环形宫殿，地下是巨大孵化腔。
2. 流放者旧堡：人族没落贵族被发配之地，石堡半陷入沼泽，墙上有人族纹章和虫族刻字并存。
3. 千渠城：虫族吸收人族工艺后建造的水道城市，桥梁低矮，甲壳船穿行，岸边立有虫甲灯。
4. 芦骨战场：虫族最早与人族流放军合作之地，芦苇中残留古战车轮。
5. 黑土祭田：肥沃黑土地，虫族以奴隶耕作，泥地里插满记录产量的骨牌。

冲突用途：揭示虫族如何从流放贵族那里学得文明、追查旧盟约、潜入母巢、虫族内部传统派与清明派冲突。

视觉锚点：黑泥、冷雾、芦苇、半沉石堡、发光虫卵、甲壳船、骨牌田、虫族书记官用人族文字记账。

### 2.5 东部瀛洲

东部瀛洲位于中央大陆东侧，是封闭岛国群，隔结界海与中央大陆相望。结界海由旧帝国云裔人族布下，只有旧帝国龙船体系才能稳定航行；虫族焚毁船坞与星图后，航路断绝，结界海逐渐失控。

统治势力：瀛洲兽族诸藩、岛主幕府、神社院、浪客商团。瀛洲兽族不是北境蛮兽，而是拥有独立礼法、血脉图腾、山城武士和海上禁忌的岛国文明。

关键场景：

1. 龙船渡：旧时接待人族龙船的港口，如今船坞荒废，只剩兽族守军和破碎龙骨泊位。
2. 白狐京：兽族贵族都城，屋檐弯曲，街道悬挂兽纹灯笼。
3. 赤岳火山：瀛洲圣山，火山灰覆盖神社与温泉村。
4. 结界海西岸：海面有不稳定透明光幕，普通船只接近会迷航。
5. 海族礁城遗影：低潮时可见海底宫墙轮廓，暗示海族势力深入此域。

冲突用途：瀛洲锁国、围剿岛链人族、开海派寻找失落航路、主角未来恢复航权。

视觉锚点：海雾结界、破败龙船泊位、兽纹灯笼、火山红光、竹林雨、潮水下的宫殿影子。

### 2.6 东海岛链

中央大陆东海岸与东部瀛洲之间，并非只有结界海空白带，而是散布着大量岛链：潮骨群岛、碎帆屿、雾灯礁、白浪残港、黑盐岛和龙骨浅滩。

虫族焚帆令后，一批旧帝国港民、龙船学徒、灯塔守人、渔猎氏族和流亡军户逃入岛链。他们与中央大陆断绝，逐渐形成东海岛链人族。岛链人族最初是守港遗民，后来因无国可归、无粮可种、无史可认，在千年封锁中变成海盗、走私者、雾航向导和沉船打捞者。

他们劫虫族海税船，也劫瀛洲商船；他们憎恨虫族焚毁航权，也怨恨瀛洲兽族长期把他们视为海上污秽。

关键场景：

1. 潮骨群岛：岛礁如白骨露出海面，是海盗聚会与沉船分赃地。
2. 碎帆屿：残破桅杆插满海岸，岛民用旧船帆搭屋。
3. 雾灯礁：旧帝国潮星灯塔残基，夜里会发出不稳定月白光。
4. 黑盐岛：盐田、走私港和虫族海税船常出没的地方。
5. 龙骨浅滩：传说沉着最后一艘完整龙船的骨架。

冲突用途：找回航权、遭遇海盗人族、瀛洲围剿、海族谈判、恢复结界海真实用法。

视觉锚点：白雾、碎帆、贝骨饰、旧船钉、潮灯残光、盐绳发辫、龙骨残片、海盗小船穿雾。

### 2.7 南部丛林地

南部丛林地是多雨密林，巨树、藤桥、瀑布、雾谷、树上城和湿热河谷交错。精灵主要居住于此，少量中央大陆南部边缘的人族在精灵庇护下生活，保存旧法术和文明传承。

统治势力：树冠议会、古树祭司、林誓守卫、雨庭药师。精灵很少接纳外族，也很少走出丛林，但他们与人族遗民有古老庇护盟约。

关键场景：

1. 苍冠树城：建在巨树冠层中的精灵主城，藤桥连接平台，树干中有发光符文。
2. 隐歌谷：庇护南方人族幸存者的山谷，夜间用歌谣、火圈和木牌仪式传承历史。
3. 万雨瀑布：巨大瀑布后藏有精灵祭坛。
4. 月苔林：夜晚苔藓发光，适合秘密会面和追踪。
5. 断根边界：精灵与虫族势力的缓冲地，树木被砍伐后留下巨大树桩。

冲突用途：逃犯晏南枝的来源、旧帝国血脉保护线、精灵是否介入护送、坐骑与森林追逐。

视觉锚点：巨树城、藤桥、雨幕、发光苔藓、树冠弓手、月角鹿、藤背豹、瀑布祭坛、夜间歌谣火圈。

### 2.8 西部高原地

西部高原地由连绵高山、峡谷、矿脉、地下城、冰川湖和风蚀石林组成。这里是矮人王国、矿炉氏族和高原种族的生活地。

统治势力：山王议盟、祖炉家族、契约厅、灰烬行会。矮人不承认虫族是大陆之主，但重视贸易、矿脉安全和契约收益。

关键场景：

1. 赤砧王城：矮人主城，建在山腹内，巨型祖炉照亮穹顶。
2. 千锤矿道：纵横地下的矿道系统，轨车、吊桥、矿灯密布。
3. 高原风门关：控制西部通道的山口要塞，石门巨大。
4. 星铁陨坑：远古陨石坑，产出稀有金属，湖水呈银蓝色。
5. 断王石厅：古代矮人内战遗址，王座裂开，战斧插在地面。
6. 禁枪炉：矮人山城最深处的小型军械工坊，炉口常年封印，只在王命、旧盟契约或矿道危机中开启，能少量制造火门枪、短管火铳和黑火炮。

冲突用途：获取旧帝国工程图、边墙修复技术、符文兵器、基础火枪、矮人内斗、虫族贸易制裁。

视觉锚点：山腹王城、熔炉火光、巨锤雕像、矿车轨道、星铁蓝光、厚重石门、矮人符文铠甲、被油布包裹的短管火铳。

### 2.9 跨区域通道

1. 北境军道：从中央大陆粮仓带通往绝境山脉边墙的军事道路。
2. 旧帝国驿道：贯穿南北的旧人族道路，主角父亲曾为其测绘，残阳坳是其中一处驿站遗址。
3. 结界海航路：旧帝国龙船体系才能稳定通过的海路，现因焚帆令和千年失修变成迷航带。
4. 南缘密林小道：中央大陆逃亡者通往精灵庇护地的秘密路线。
5. 西部矿商道：矮人商队通向中央大陆的高原道路。

## 3. 种族与文明

### 3.1 人族：失翼的旧文明后裔

外形：普通人族与现实人类接近，但旧帝国血脉觉醒者在施法时会显出淡金、银白或赤曜色灵翼纹路，纹路沿肩胛、脊柱、手腕和眼角展开。边墙人族多有冻疮、粗糙手掌和灰白麻衣；南缘遗民常佩木牌、红线祖牌和歌纹刺青；岛链人族则带盐痕、贝骨饰和旧船铜件。

独特能力：灵翼经络、文字法、星图法、歌谣记忆、工程法阵、驿道测绘、结界维护、龙船制造、边墙武脉。人族的强项不是单体肉身，而是把语言、建筑、道路、船舶、仪式、武备和群体记忆编织成文明级力量。少数人族擅长法术，大多数人族依靠工艺、组织、武器、训练和身体强化弥补与外族的力量差距。

主要亚群：

1. 边墙人族：居住于绝境山脉堡垒、矿村、烽燧和冰谷农田，擅长耐寒、弓弩、陷阱、古歌记忆、低阶符咒残片和边墙武脉。
2. 南缘遗民：受南部精灵庇护，保存较多人族古语、祭仪和法术谱系。
3. 中央奴民：生活在虫族治下城市、庄园、工坊和军营，表面祭虫皇，私下供人族祖牌。
4. 混血人族：虫族曾推动下层兽族与人族女性通婚，部分后代形成混血群体。
5. 云裔人族遗脉：被后世误称为仙族的旧帝国高阶法统。
6. 东海岛链人族：旧港遗民后代，千年中变成海盗、走私者和雾航向导。

内部派系：

1. 复翼者：寻找解除灵翼封印的方法。
2. 灰墙军：认为生存高于复仇，愿继续守墙换粮。
3. 旧书会：暗中收集残卷、童谣、纹章和墓志。
4. 新血派：接受混血现实，主张建立多族共同体。
5. 雾航派：岛链人族中的航路恢复者，想找回真正龙船。

视觉锚点：旧帝国日月双印、断翼符号、磨旧铜饰、灰白麻布、边墙冻疮、红线祖牌、盐绳发辫、旧船铜扣。

### 3.2 云裔人族遗脉：被误称为仙族的旧帝国高阶法统

云裔人族不是独立种族，而是昭明帝国人族中的高阶法统，旧称“天工山门”。他们掌握星象法、结界法、边墙工程、龙船制造、远航术和大型阵基维护。虫族篡夺后，焚毁云裔谱系、龙船船坞、航海星图、结界海阵图和东海港籍，把这支人族从正史中切割出来。

外形：身形清瘦，骨相接近普通人族，但因长期修习星象吐纳，眼瞳在夜间会映出星点；高阶者发色可因法阵反噬变白，皮肤有淡金阵纹残影。服饰多为旧帝国白、青、金三色，佩星盘、结绳和龙骨量尺。

独特能力：

1. 结界法：建造、维护和重启大型区域法阵。
2. 边墙工程：把山脉、城墙、烽燧和地脉连成防御系统。
3. 龙船制造：用龙骨木、星潮铜和月绳造出能穿越结界海的船。
4. 星潮测算：通过星象、潮汐和风向计算航路。
5. 短暂御风：借灵翼经络和器具滑行或短距离凌空。

现状：主脉在焚帆令中覆灭，少量遗脉藏入山门、岛链、矮人矿城和南林人族村落。后世“仙族”只是虫族伪史和民间误称。

### 3.3 虫族：夺取文明的沼泽王朝

虫族发源于东北沼泽，原本以母巢、猎群和繁殖季为核心。被流放人族贵族启蒙后，迅速吸收文字、军制、法术和宫廷制度。夺取中央大陆后，虫族一面毁灭人族记忆，一面模仿人族礼法，自称文明正统。

外形：虫族可穿人族官袍、使用人族姓名，但本相带有外骨骼、复眼纹、节肢关节、甲壳脊线或薄翼膜。高阶虫族擅长以蜡膜、香料和礼服遮掩虫相；军府虫族则保留黑绿甲壳和刃肢。

独特能力：

1. 巢群感应：同巢虫族可通过气味、触须和微振共享警戒信息。
2. 气味标记：给区域、奴民、猎物和罪犯留下可追踪气味。
3. 快速繁育：战时能快速扩张兵源，但会加重粮食压力。
4. 外骨骼防御：甲胄虫族能抵御普通刀箭。
5. 夜视与攀爬：螳刃、蜂巢等亚种适合暗杀、搜查和城市追捕。
6. 虫蜡筑巢：快速改造建筑、封存档案和覆盖旧碑。
7. 群体遗忘术：蜂巢虫族能把政令、恐惧和记忆压制做成类似“社会气味”的群体法术。

主要亚种：

1. 甲胄虫族：主战阶层，外骨骼厚重，适合正面冲锋和守城。
2. 螳刃虫族：轻捷刺客与军官阶层，擅长攀爬、夜视和突袭。
3. 蜂巢虫族：行政、记录、繁育和法术阶层，负责气味标记、信息共享和遗忘法术。
4. 蛾翼虫族：少数具有飞行与幻术能力的祭司、艺人和外交者。
5. 蚁工虫族：底层劳作者，负重强，常在矿区、粮仓和城建中服役。
6. 蜉蝣虫族：寿命短但繁殖快，多用于消耗战与侦察。

弱点：高度依赖巢母、粮仓、湿热巢室和等级秩序；离开群体后容易焦躁、失向或陷入本能饥饿。

### 3.4 东部瀛洲兽族：海雾之外的岛国武风文明

瀛洲兽族长期生活在结界海与海族大洋之间，形成氏族武士、神社、山城、海崖渔猎和血脉图腾并存的岛国文明。他们对中央大陆既好奇又戒备，不承认肃明国的大陆正统。

外形：瀛洲兽族保留明确兽征，但文明程度高于北境部落。狐族有狐耳、尾影和金褐眼；犬狼族肩背强健、犬齿明显；猿族臂长、掌骨粗，善攀岩；鹤羽族有羽冠、轻骨和白灰羽披。贵族会用礼服、面具和家纹约束兽性。

独特能力：

1. 血脉图腾：通过氏族神社强化某一兽性能力。
2. 家名誓约：越忠于家名，能力越稳定；背离家名则能力反噬。
3. 神社结印：在神社、刀具、护符和鸟居中储存法力。
4. 山城武艺：适合狭窄山路、城门、桥梁和屋顶战。
5. 短距幻术：狐族擅长气味遮掩、影尾幻象和梦兆诱导。
6. 追踪与信使：犬狼族追踪，鹤羽族高空通信，猿族机关攀城。

主要亚族：狐族、犬狼族、猿族、鹤羽族、狸工族、虎纹武家、海獭船匠族。

与岛链人族关系：瀛洲锁国派视岛链人族为海盗和结界污染源；开海派则暗中雇佣岛链向导寻找失落航路。

### 3.5 北部诸族：驭巨兽的寒灾部落联盟

北部诸族生活在极寒环境中，肉体强大，文明形式接近迁徙部落、兽群盟誓和萨满记忆。千年雪灾使他们不断南下，引发与虫族和边墙人族的五百年战争。

共同能力：北部诸族擅长控制体形庞大的野兽。他们不是简单奴役野兽，而是以幼崽共养、血盐喂养、骨笛音阶、梦境契约和战利品共享建立盟约。巨兽不会无条件服从；若部落违背迁徙誓言，巨兽会反噬或离群。

主要部族与种族：

1. 黑牙半兽人：高大、獠牙外露、肩背覆厚毛，擅长近战、战鼓和猛犸冲阵。能与巨角猛犸建立血誓。
2. 白鬃狼裔：灰白毛发、竖瞳、嗅觉极强，擅长雪地奔袭、群猎和夜间侦察。常骑霜狼、雪鬃座狼。
3. 冰鳞蛇族：皮肤有蓝黑鳞片，体温低，能在冰窟和地热裂缝中生存。擅长毒、缠锁、冰雾潜行，能驱使冰蟒和地热蛇群。
4. 铁角牛族：体格宽厚，额生短角，耐寒耐饿，擅长搬运、盾阵和破城。常与披甲牦牛、岩背犀同行。
5. 雪羽鹰族：骨轻、臂生羽纹，能滑翔、驯鹰和高空侦察。掌控雪鹰、裂风隼和幼飞龙。
6. 霜蹄鹿族：身形修长，角枝复杂，擅长草药、迁徙路线和梦境占卜。常与巨角雪鹿、林冠麋群结盟。
7. 灰皮巨魔族：身躯庞大、再生力强、语言缓慢但记忆深，负责拖动攻城木、冰桥和巨石。能安抚岩背巨兽。
8. 赤鬃狮裔：鬃毛如火，擅长王庭礼仪和决斗，统合多个猎群。
9. 雪獾掘地族：身材矮厚，爪强，擅长冰下地道和突袭粮仓。
10. 冰湖龟民：行动缓慢，背壳坚硬，擅长守护冰湖和渡河。

北部巨兽与坐骑：

1. 巨角猛犸：冲阵与搬运主力。
2. 霜狼：白鬃狼裔常用坐骑。
3. 岩背犀：破城与撞门巨兽。
4. 披甲牦牛：铁角牛族运输兽。
5. 冰原巨熊：黑牙萨满的护卫兽。
6. 雪原长牙虎：赤鬃狮裔的对手与盟兽。
7. 六足驮鹿：霜蹄鹿族迁徙坐骑。
8. 地热冰蟒：蛇族洞穴守卫。
9. 裂风飞龙：雪羽鹰族驯育的空中战兽。
10. 山脊巨鹰：侦察与传信。
11. 冰湖甲龟：冰湖渡河和移动盾墙。
12. 冻原雷蜥：能以背鳍储存静电，战场冲击力强。

视觉锚点：骨甲、兽皮、冰霜纹身、巨角战盔、极光祭坛、血红战旗、冻裂皮肤、骨笛、巨兽骨架营地。

### 3.6 南部精灵：记忆守林者与林间骑乘文明

南部精灵热爱和平，但这种和平建立在严格边界、长寿记忆和对外族历史悲剧的冷静判断上。他们庇护部分人族遗民，却不愿轻易为人族复国卷入大陆战争。

外形：精灵身形修长，耳尖，肤色从树皮浅褐、雨雾青白到夜林深灰不等；眼瞳常呈叶绿、琥珀、月银或花紫。服饰以藤甲、叶纹布、树脂轻甲和羽木饰为主。

独特能力：

1. 草木沟通：读取植物生长、伤痕与入侵痕迹。
2. 生命术：治愈伤口、催发生长、延缓毒性。
3. 活体弓术：弓与使用者共生，能调整弓力与箭路。
4. 毒草药理：雨精灵能制毒、解毒和布置麻痹雾。
5. 水雾幻术：在雨林雾气中遮蔽路径和身形。
6. 梦境巡林：夜精灵能通过树根梦境感知禁林异动。
7. 盟约见证：花冠精灵能以歌舞让誓言产生法术约束。

精灵坐骑与伙伴：

1. 月角鹿：森精灵常用坐骑，安静、跳跃力强，能在树根与溪石间高速穿行。
2. 藤背豹：夜精灵追踪伙伴，背部有藤纹，擅长伏击和无声攀树。
3. 雨翼鹤：雨精灵信使坐骑，可短距载人飞越瀑布与雾谷。
4. 花冠麋：花冠精灵仪式坐骑，鹿角会随季节开花，能稳定歌舞法阵。
5. 树冠巨蜥：林誓守卫重型坐骑，能贴附巨树树干，适合树城防御。
6. 青羽迅鸟：丛林奔跑鸟类，不能长飞，但速度极快，用于巡逻和传令。
7. 梦苔龟：古树祭司缓慢坐骑，龟壳生发光苔藓，可承载小型祭坛。
8. 银喉猿：会模仿人声和鸟鸣，常被用作警戒伙伴。
9. 叶翼蝠：夜间小型侦察伙伴，能穿越树洞和藤缝。

历史伤口：人族帝国鼎盛时曾砍伐南林边缘，建城、采矿、修路。精灵后来庇护人族，不代表忘记这段侵害。

### 3.7 西部矮人：祖炉、符文与山城契约文明

矮人不是单一王国，而是多个山城、矿炉氏族和地下商路联盟。矮人重契约、工艺和祖炉记忆，长期在大陆战争中维持中立贸易，却暗中保存大量人族旧帝国机械、法器和建筑图纸。

外形：体格矮壮，骨骼厚重，胡须、发辫和金属饰物代表家族、职业与契约等级。长期矿炉生活使其皮肤偏赤褐、灰黑或铜色，眼睛适应暗处火光。

独特能力：

1. 符文锻造：把契约、记忆和力学结构刻入金属。
2. 矿脉听觉：通过敲击和耳贴石壁判断矿脉与裂缝。
3. 祖炉记忆：氏族炉火中保存先祖工艺片段。
4. 契约誓印：违约会在身体或器物上产生可见裂纹。
5. 重甲作战：矮人可在狭窄矿道中形成不可移动的铁墙。
6. 地下工程：修复旧帝国边墙深层结构。
7. 基础火枪：仍能制造火门枪、短管火铳和小型黑火炮，但产量低、射速慢、维护难。

主要亚族：炉山矮人、深岩矮人、云梯矮人、灰烬矮人。

弱点：重契约导致政治反应迟缓；若契约被虫族利用，矮人会在道义和祖训之间撕裂。矮人保留基础火枪制作能力，却不敢公开大规模供给人族，因为这会触发肃明国的贸易制裁和军事清剿。

### 3.8 海族：潮汐王庭与深海盟约

海族统治瀛洲南西大洋，也影响结界海外围航线。他们不是单一水下种族，而是潮汐王庭、珊瑚城邦、深渊祭司和海兽盟约组成的海洋文明。

外形：形态差异极大，包括鲛人、鳞民、珊瑚祭司、深渊触须族和海兽骑士。浅海族上半身近人，皮肤有鳞光；深海族眼大、肤色幽蓝或灰白，常佩珊瑚骨冠和贝甲。

独特能力：

1. 潮汐术：改变局部海流、浪高和船只方向。
2. 水压感知：在深水中感知远处震动。
3. 洋流记忆：以歌声记录海流变化。
4. 珊瑚生长：建造礁城、治愈伤口或制造毒刺。
5. 海兽盟约：与巨鲸、海蛇、礁龟、鲨群建立契约。
6. 深海歌声：影响梦境、恐惧和方向感。

与东海断航关系：海族知道结界海原本可控，也知道虫族不会维护旧阵。但他们不会主动帮助人族，因为昭明旧帝国和虫族都曾伤害海洋。

### 3.9 边缘族群

1. 墙下混裔：由人族、兽族、虫族底层、北地俘虏长期混居形成的边墙边缘群体。
2. 沼泽亡书人：人族流放贵族与虫族早期学徒的后代，生活在东北沼泽废墟，掌握虫族夺权真相。
3. 风砂民：生活在东北沼泽边缘沙地与西部高原之间的小型游牧商路族群，传播货物、谣言和情报。
4. 清明院识别者：被虫族训练出来的人族或混血儿童，能识别旧帝国暗语、血纹和禁歌。

### 3.10 各势力内部派系与代表人物

这一节用于恢复各族内部政治裂缝。后续剧情不应把任何种族写成单一意志，而应让主角在每个地区都面对“同族不同立场、敌族也有分裂”的局面。

昭明遗民与人族派系：

1. 复翼者：主张解除锁翼咒、恢复昭明法统，愿意利用晏南枝的血脉号召旧臣。代表人物：晏南枝早期的南方师长顾怀章。
2. 灰墙军：边墙人族军户中的保守派，认为守墙换粮比复国更现实，不愿为远方宗室牺牲。代表人物：老墙师薛临墙。
3. 旧书会：收集残卷、墓志、童谣、驿令和军械铭牌，不急于起兵，只想保存真相。代表人物：亡书人孟归藏。
4. 新血派：接受混血、失籍、海盗、奴民和边民都属于新故土，反对纯血复国。代表人物：墙下混裔少女陆青砾。
5. 雾航派：岛链人族中的航路恢复者，目标是找回龙船、潮星灯塔和结界海真实航路。代表人物：岛链船主秦雾灯。

肃明国虫族派系：

1. 清明院：白翳所属派系，主张清明籍、血纹筛查、记忆审讯和系统性清除。代表人物：白翳。
2. 甲军府：军方强硬派，认为清明院太慢，主张用军队和边境战争直接压服各族。代表人物：甲军将领厉螳。
3. 书壳院：档案与伪史机构，负责改写旧史、封存禁书、训练识别者。代表人物：书壳院掌籍闻蜕。
4. 巢母议会：虫族繁育与血统核心，重视族群延续，不完全信任过度人族化的官僚。代表人物：玄巢母。
5. 沼祖派：东北沼泽传统派，认为肃明国忘记虫族从泥地生存的本源，反对穿人袍、学人礼。代表人物：泥巢祭老乌涎。
6. 泥巢民：底层蚁工、蜉蝣和沼泽虫民，名义上是征服者同族，实际仍被粮仓、矿井和巢室制度压迫。代表人物：蚁工青沼。

东部瀛洲兽族派系：

1. 锁国派：认为结界海是神罚边界，岛链人族是海污，不应重开航路。代表人物：白狐京摄政狐斋。
2. 开海派：相信白浪葬不是神罚，而是航路失修和海族深潮所致，愿寻找旧龙船技术。代表人物：鹤羽信使羽见。
3. 武家扩张派：主张趁肃明国和人族内乱，向岛链与西岸扩大影响。代表人物：虎纹武家少主虎堂胤。
4. 浪客商团：以雇佣、走私和情报为生，表面中立，实际谁能开海就支持谁。代表人物：狸工商头狸久。
5. 神社守旧派：掌握神社结印与血脉图腾，担心开海会破坏家名誓约和岛内秩序。代表人物：神社院主犬守。

北部诸族派系：

1. 南迁战派：由黑牙、铁角等部族组成，认为只有冲破边墙才能让族群活下去。代表人物：黑牙王赫鲁。
2. 寻灾源派：以霜蹄鹿族萨满和雪羽鹰族侦察者为主，怀疑千年雪灾背后有旧阵失衡。代表人物：霜蹄萨满鹿弥。
3. 王庭统合派：赤鬃狮裔试图建立北部共同王庭，统一兽群盟约后与中陆谈判。代表人物：赤鬃王女阿烬。
4. 掠粮派：雪獾掘地族、部分狼裔和流亡猎群组成，只关心抢粮过冬，容易被肃明国挑动。代表人物：雪獾头领裂爪。
5. 巨兽守誓派：冰湖龟民与灰皮巨魔族中较古老的部族，认为巨兽盟约不能被滥用于侵略。代表人物：冰湖龟民长者寒壳。

南部精灵派系：

1. 闭林派：反对继续庇护人族，认为昭明帝国旧债未偿，肃明国清剿也不该引入森林。代表人物：树冠议员榕洛。
2. 护人派：坚持古老庇护盟约，认为人族遗民若被清除，森林最终也会被肃明国吞噬。代表人物：雨庭药师澜枝。
3. 古树祭司：维持森林长记忆，愿见证盟约，但不轻易参战。代表人物：古树祭司眠苔。
4. 林誓守卫：军事派，接受有限合作，但要求外族不得携带战争进入树城。代表人物：林誓守卫青弦。
5. 花冠调停派：以誓言、婚盟、歌舞和仪式维持各派平衡，适合在剧情中做谈判桥梁。代表人物：花冠舞者明萝。

西部矮人派系：

1. 中立商派：继续与肃明国贸易，认为契约和矿脉安全高于人族旧债。代表人物：契约厅长锻衡。
2. 旧盟派：认为矮人曾间接加速昭明灭亡，必须偿还古债，暗中帮助人族。代表人物：祖炉长女赤砧珑。
3. 闭山派：拒绝再让高原卷入大陆战争，宁愿封锁矿道和禁枪炉。代表人物：深岩王叔石晦。
4. 禁枪炉匠：保存基础火枪和小型火炮工艺，担心技术一旦外流会制造新帝国。代表人物：灰烬枪匠炉七。
5. 灰烬行会：以军火、旧图和矿道通行证牟利，可能同时卖给肃明国和反清明势力。代表人物：行会主灰贩。

海族派系：

1. 潮汐王庭：主张维持海陆边界，不愿为陆地政权付出代价。代表人物：潮王女澜珂。
2. 珊瑚城邦：痛恨昭明旧港和肃明海税造成的海域污染，要求陆地先偿还海债。代表人物：珊瑚祭司赤珊。
3. 深渊祭司：认为结界海失控是陆地人自作自受，反对修复航路。代表人物：深渊歌者无潮。
4. 海兽骑士：重视盟约与勇气，若沈维桑能救下海兽或履行海誓，可能成为盟友。代表人物：老骑士鲸铠。
5. 开潮派：少数年轻海族，认为重开结界海可重新平衡海陆秩序。代表人物：海兽骑士泠刃。

东海岛链人族派系：

1. 雾航派：掌握残缺航路和灯塔禁歌，想从海盗重新变回守灯者。代表人物：秦雾灯。
2. 黑盐劫掠派：以劫船、勒索和火油战生存，不相信大陆人会接纳岛链。代表人物：黑盐头领鲍沉。
3. 守灯遗老：旧港灯塔守人后代，保存潮星灯碎片和港籍残页。代表人物：老灯师周潮骨。
4. 龙骨潜民：潜入沉船和礁城打捞旧物，与海族关系复杂。代表人物：潜民少女阿贝。
5. 归岸派：希望离开海盗身份，回到中央东岸重建港口，但被肃明国东海禁岸和瀛洲围剿同时压迫。代表人物：港民后裔林归岸。

边缘族群派系：

1. 墙下混裔：不被昭明复国派完全接纳，也不被肃明国信任，是新故土理念最重要的现实基础。代表人物：陆青砾。
2. 沼泽亡书人：掌握虫族启蒙和篡夺真相，既恨肃明国，也不信任昭明宗室。代表人物：孟归藏。
3. 风砂民：中立情报商，能把南部歌谣、矮人旧图和北境兽讯带到主角手中。代表人物：风砂商沙衡。
4. 清明院识别者：包括沈照眠这样的被训练儿童，是肃明国暴力制度最直接的人格化伤口。代表人物：沈照眠。

## 4. 背景故事

### 4.1 人族黄金时代

昭明帝国以“日月照耀皆为故土”为最高宣言，建立白石城、星图法阵、旧驿道、龙船船坞、潮星灯塔、结界海航标和绝境边墙。人族与精灵、矮人、瀛洲兽族、海族均有过贸易、盟约或冲突。

这个时代的强大来自法术与科技的合流。少数灵翼法师可以维持星图通讯、结界校准、军阵防护和战场治疗，但真正支撑帝国扩张与防御的是道路、粮仓、军械学院、枪炮工坊、蒸汽牵引、矿道轨车、边墙工程和常备军纪律。普通人族无法凭肉身与虫族甲胄、北境巨兽、瀛洲武家和海族海兽正面对抗，因此火枪、大炮、弩炮、蒸汽机和城防工程曾是昭明帝国维持平衡的关键。

昭明军制的核心不是“法师统治战场”，而是“法师把普通军队组织成更大的战争系统”。烽燧法阵传讯，星潮师校准炮火，工程法师加固城门，医师稳定伤兵，枪炮兵、长枪兵、驿道运输队和蒸汽工匠承担真正的大规模战争消耗。

昭明并非纯洁无瑕。帝国扩张时曾砍伐南林、压迫小族、修建大港、捕猎海兽、盗取珊瑚灵核，也让不少外族记住了日月旗的阴影。

### 4.2 东北流放与虫族启蒙

东北沼泽原本是人族帝国发配罪臣、失败贵族和异端学者的地方。流放贵族为了生存，与当地虫族巢群交易文字、医术、军阵和基础法术。虫族从此不再只是沼泽猎群，而逐渐拥有记录、等级、战术和政治野心。

虫族最早学习人族文明不是为了篡夺，而是为了逃离泥沼、饥饿、寒潮和瘟疫。这种“从泥地爬向日光”的集体记忆，后来变成他们夺取中央大陆后拒绝退让的心理根源。

### 4.3 裂翼之乱

1200 年前，人族帝国内斗。皇族、贵族长老、地方军府、云裔法统和法术书院互相争权。部分贵族长老秘密与虫族结盟，借助虫族繁育快、忍耐强、不惧死亡的兵力推翻皇族。

虫族进入中央大陆后，被白石古城、金河粮仓、书院、船坞和宫殿震撼。他们第一次意识到，自己未必只能做战争工具，也可以成为大陆主人。

### 4.4 篡夺之夜

当人族各派两败俱伤，高阶法师因内战受伤无法完整施法时，虫族发动背叛。虫族屠杀皇族、贵族、史官、法师和书院学徒，焚毁书籍，毁碑改档，夺取宫城。

虫族没有立刻以虫族之名统治大陆，而是对外宣称自己继承文明正统。他们改用人族姓名、官制、礼仪、服饰和建筑，同时将真正的人族降为奴民、军户、罪民、混血群体和失籍海民。

### 4.5 灭械令与战争文明断代

篡夺之夜后，肃明国很快意识到：只要人族仍掌握火枪、大炮、军用蒸汽机、火药配方、测距术和军械学院，数量庞大的人族奴民就仍有机会重新弥补与外族的肉身差距。因此虫族在焚书、锁翼之外，又推行“灭械令”。

灭械令分三步执行。第一步是焚毁军工图纸、炮铸模具、枪管量具、火药配方和旧帝国军械教材。第二步是拆毁战争工坊、炮台、军用蒸汽牵引车和边境军械库。第三步是将枪炮匠、蒸汽机师、测距官、炮兵教官和军械学院学徒迁徙、阉割传承或处决。

虫族没有完全毁灭蒸汽文明，因为他们仍需要生产、采矿、排水、粮仓、运输和城市工程。于是后世中央大陆形成一种畸形断代：磨坊、水泵、矿井升降机和粮仓轨车仍然存在，但能够改变战争平衡的火器、炮兵、军用蒸汽和精密测量体系几乎消失。

灭械令的结果，是战争形式在数百年内持续退化。火枪和大炮从旧帝国常备军武器，变成传说、违禁品和矮人山城少量保存的危险旧艺。肃明国可以用冷兵器、虫甲、巢群、监察和清明籍控制人族；北境诸族与瀛洲兽族也在肉身和坐骑优势上重新占据上风。

### 4.6 焚帆令与东海断航

虫族夺权后的第三年，甲军府攻破东海白石港，书壳院焚毁龙骨船坞、潮星灯塔、云裔星图库和港籍总册。数千名船匠、灯塔守人、星潮师和港民被处决或迁徙。幸存者逃入东海岛链，成为后来的岛链人族。

虫族随后宣布东海为“逆潮禁地”，禁止真正人族靠近。中央大陆人族从此失去海洋、瀛洲、岛链和远航记忆。千年后，东海只在禁歌中留下四个意象：白帆、潮灯、龙骨、归岸。

### 4.7 遗忘与锁翼

虫族书壳院与蜂巢术士共同施展群体遗忘法术，压制人族对旧史、旧文字、旧法术和旧航权的认知。虫族又用锁翼咒切断人族灵翼经络，让人族后代难以觉醒高阶法术。

这场灭史并不完美。南部人族在精灵庇护下把史书拆成歌谣、祭礼、纹身、织物图案和儿童游戏。岛链人族把航路藏在禁歌、潮汐谚语和船骨上。边墙人族虽然忘记含义，却仍在送葬、祈雪、守夜和成人礼中重复旧动作。

### 4.8 肃明国

虫族建立肃明国，对外宣称中央大陆必须“肃尽前明”，清除昭明帝国留下的日月痕迹后，才可能进入清明。虫族上层越发人族化，沼泽传统派对此不满。虫族底层泥巢民也没有获得征服红利，仍在沼泽、矿地和粮仓中劳作。

为了削弱人族身份认同，肃明国推行迁徙、禁书、禁私学、混血政策、边境军户制度、东海禁岸和清明籍筛查。中央奴民逐渐忘记自己祖先的城市，边墙人族逐渐只记得守墙、寒冷和军令，东海人族则被污名化为海匪。

### 4.9 千年雪灾与五百年血战

若干年后，北部极寒迎来千年雪灾。草场被冻死，冰海暴涨，兽群迁徙路线崩溃，北部诸族与巨兽群大规模南下。

虫族政权借战争之名，把大量人族迁往绝境山脉。人族成为守墙军、矿奴、前哨、诱饵和尸墙。长达五百年的血战既真实抵御了北境灾难，也被虫族利用为消耗人族、塑造自身救世主叙事的工具。

因为灭械令已经断绝大规模火器和炮兵体系，边墙人族只能用长枪、弩、盾墙、陷坑、滚石、床弩和残缺法阵面对巨兽与北境战士。五百年的冷兵器战争迫使边墙军户把搬石、守夜、挨冻、负重、站桩、盘架、呼吸和皮肉锻打结合起来，逐渐形成边墙武脉。它不是旧帝国法术复兴，而是普通人族在被剥夺枪炮之后，为了活下去发明的身体代偿。后来少数老墙师从旧帝国残卷中重新读出“天人合一”之义，才把这套透支身体的外家功夫，推向可以修复自身的内家道路。

### 4.10 南部隐歌

中央大陆南部边缘的一支人族在精灵庇护下幸存。他们不能公开写史，于是将人族历史变成奇怪的歌谣和仪式。每首歌都是一段残缺档案，每个舞步都是法阵路径，每个儿童游戏都对应旧帝国地图。

精灵庇护人族并非完全无私。人族旧帝国曾伤害南林，精灵内部始终存在闭林派和护人派之争。人族遗民必须接受精灵监管，不能轻易把战争引进森林。

### 4.11 云裔法统覆灭与旧帝国断航

云裔人族曾协助修筑边墙、布置结界海、制造龙船。旧帝国内斗时，云裔法统分裂：一部分试图维持帝国秩序，一部分反对皇族扩张，一部分闭守山门。虫族篡夺后，云裔主脉在焚帆令与山门清剿中覆灭。

他们的失败不是外族旁观，而是人族内部高阶法统在内斗、虫族屠杀、技术焚毁和道德失败中崩解。后世所谓“仙族传说”，只是虫族伪史为掩盖人族曾拥有高阶航权和结界文明而制造的误称。

### 4.12 矮人的契约债

虫族夺权时，部分矮人王国因契约向虫族供应攻城器械，间接加速人族灭亡。另一些矮人冒死藏匿人族书匠、云裔量尺、法器图纸和边墙工程图。

矮人内部至今为此争吵。中立商派继续与虫族贸易，旧盟派认为必须帮助人族偿还古债，闭山派则拒绝再让高原卷入大陆血战。矮人山城仍保留基础火枪与小型火炮工艺，却把它视为会引来清剿的禁契技艺，只在矿道防御、王城内战和极端交易中少量使用。

### 4.13 瀛洲白浪葬与岛链围剿

瀛洲兽族传说中，曾有大规模船队试图穿越结界海，与中央大陆恢复旧日往来。船队在白色海雾中消失，只留下破碎船板和羽饰，此事被称为白浪葬。

锁国派把白浪葬解释为神罚，开海派则认为这是结界海失控与海族深潮共同造成的灾难。千年中，瀛洲兽族长期围剿岛链人族海盗；岛链人族则认为瀛洲忘恩负义，享受旧帝国航路庇护，却在旧港人族落难后把他们称作海污。

### 4.14 海族旧债

人族帝国鼎盛时曾修建大港、捕猎海兽、盗取珊瑚灵核。虫族继承这些港口后，又加重海税和河口污染。海族对陆地文明的怨恨跨越政权，不会因为人族受害就自动站在人族一边。

若主角未来寻求海族帮助，必须面对昭明帝国也曾欠下海债的事实。

### 4.15 当前时代的松动

近两百年，北境战争趋于低烈度，但兽潮周期仍存在。肃明国开始推行“清明籍”计划，试图彻底筛查旧帝国血脉、禁歌、旧驿图和不受登记的村落。

与此同时，南部人族歌谣开始沿商路秘密传播，矮人发现边墙深层结构并非虫族所建，瀛洲开海派开始活动，岛链人族发现龙骨浅滩仍有完整船阵残骸，云裔遗脉传说重新出现。

## 5. 主角与核心人物

### 5.1 主角：沈维桑

身份：残阳坳猎户兼引路人，旧帝国驿道测绘官后代。

年龄：17 岁。

姓名来源：“维桑”取自《诗经·小雅·小弁》的“维桑与梓”，桑梓后世指故乡。这个名字带有故土难离、敬畏本根的含义，也与他从只守小家到重新理解故土的角色弧相扣。

外形：身形瘦硬，常穿灰褐猎衣、旧皮护腕和磨损短靴。左肩有一道旧伤，随身带一把猎弓、一柄短刀和半枚日月驿令。火场之后，他衣角长期保留焦痕。

性格：沉默、谨慎、嘴硬，习惯先观察再行动，不轻信宏大口号。他讨厌英雄叙事，也不认为自己有资格谈复国或正义。残阳坳的生存经验让他先判断风向、水源、退路和人心，比起“应该怎样”，他更在意“怎样能把人活着带出去”。

他的善良不是单纯柔软，而是一条被乱世磨得很窄的底线。他可以骗监察队、偷粮、设伏、烧桥、假意服软，也能在救不了所有人时先保住还能带走的人；但他不把无辜者当诱饵，不交出弱者换安全，不允许任何旗帜把活人牺牲说成理所当然。每一次冷硬选择都会留下心理债，他不一定说出口，却会记住名字、地点和没能带走的人。

他的矛盾在于：他越厌恶正统、血脉和旗帜，越容易把自己缩回“只守家人和承诺”的小圈子；可他又无法真正对陌生人的苦难视而不见。看见孩子、伤者、混裔、军户和被白册抹掉名字的人，他会先冷脸计算代价，最后还是把人往自己身后拉。一旦答应护送某人，他会把这件事看得比自己的命更重，但他也会逼自己承认，乱世里的善良若没有判断、隐忍、欺骗和撤退能力，只会变成新的牺牲。

能力：

1. 山路与废驿识别：能从风向、石碑、旧井、驿墙判断方向。
2. 猎弓与陷阱：擅长小范围伏击、诱敌和脱身。
3. 旧驿暗语：母亲教过他部分驿站暗号，他起初不知道其历史意义。
4. 边墙武脉半成品：父亲曾教过他筑架、负重行走、残缺明劲和短爆发步法。他能硬撑伤痛和严寒，但强行发明劲会留下筋膜撕裂、胸肺淤痛和旧伤反噬。
5. 灵翼残纹：情绪极端或接触日月旧物时，眼角和肩胛会浮现金赤色纹路，主要带来方向感、旧驿共鸣和短暂感知，不直接提供压倒性杀伤。
6. 承诺驱动：他的强项不是天赋压倒一切，而是在绝境中不断履行承诺。

家庭：

1. 父亲沈季衡：旧帝国末期驿道测绘官后代，表面死于山洪，实际因藏匿旧帝国密图被虫族监察使处死。
2. 母亲罗青禾：村中医者，懂草药，也懂旧帝国驿站暗语。屠村时临死托付沈维桑护送晏南枝。
3. 妹妹沈照眠：年幼，崇拜兄长。屠村事件中失踪，被虫族带入清明院，后续可能成为识别旧帝国暗语与血纹的工具。

角色弧：沈维桑最初只想保护小家，不相信昭明帝国，也不相信肃明国。屠村后，他被迫承认逃避不会带来安全。他的成长不是从普通人变成王者，而是从“只认眼前几个人”变成“愿意为一片互相承认的故土承担代价”。

### 5.2 关键逃犯：晏南枝

身份：来自南方旧帝国残脉地区的旧帝国皇室旁支血脉，身上携带“日月血牒”与寻找“月下盟书”的线索。

年龄：18 岁。

外形：黑发，常以南方红线束发；眼神冷静，衣物外层是逃亡者粗布，内层藏有细密日月纹。随身带一枚月白玉片和一卷被拆成多段的血牒。

性格：克制、聪明、受过训练，有贵族式的忍耐和骄傲。她相信血脉仍有政治价值，也相信旧帝国旗帜可以重新凝聚人心。她与沈维桑的冲突在于：她需要“故土”的大叙事，沈维桑只相信具体的人命。

能力：

1. 礼法与谈判：能识别旧帝国宗室、军府和城邦礼制。
2. 密探训练：懂伪装、暗号、追踪反制和毒物辨识。
3. 日月血牒共鸣：接近旧帝国遗迹时能唤起残缺记录。
4. 南方旧歌：掌握部分隐歌谷传承，能引出旧帝国地图线索。

角色弧：从相信自己必须成为旧帝国复兴的象征，转向理解旧帝国若要重生，不能只依靠血统与日月旗，而必须承认那些在帝国崩塌后仍然活着、受苦、背叛、妥协的人。

### 5.3 妹妹：沈照眠

身份：沈维桑妹妹，屠村后被虫族带走。

性格：聪明、敏感、记忆力强，小时候喜欢学母亲的草药名和哥哥的山路记号。

后续线：她被送入清明院，训练成识别旧帝国血脉、禁歌和驿道暗语的工具。中段可能以追捕者身份回归，亲手识破沈维桑和晏南枝的伪装。

角色弧：从“主角要找回的家人”变成“清明秩序制造出来的伤口”。她最终要选择继续执行清明籍，还是拒绝成为屠村的工具。

### 5.4 主要反派：白翳

身份：肃明国中枢监察使，清明籍计划执行者。

外形：常穿无尘白袍，外覆薄翅般的半透明虫膜披肩，面甲上没有表情，只在眼部露出暗色复眼纹。

性格：冷静、洁癖、真诚相信“肃尽前明，还天下清明”。他不是单纯残暴者，而是认为旧帝国日月旗造成数百年战争与族群压迫，只有彻底清除旧血脉、旧符号、旧记忆，大陆才能获得稳定。

能力：

1. 血纹筛查：能通过虫蜡针识别旧帝国血脉反应。
2. 记忆审讯：借蜂巢术诱导嫌疑人遗忘或供出禁歌。
3. 气味追索：给逃犯留下跨城可追踪的清明香。
4. 清除令：有权将村落定为旧污窝藏点并执行灭村。

角色功能：白翳与沈维桑形成镜像。二者都见过乱世，都厌恶无意义牺牲；白翳选择用系统性清除换取秩序，沈维桑最终选择用共同盟约承担混杂与不确定。

## 6. 故事开端

故事以肃明国的虫族监察队进入残阳坳开始。

监察队接到密令：一名来自南方的旧帝国血脉逃犯已越过封锁线，可能藏在残阳坳一带。虫族监察使白翳以“清除旧污、还村清明”为名，封锁村道、搜查地窖、查验血纹、焚毁旧帝国物件。村民试图沉默自保，但旧驿站暗格暴露，证明村中有人曾协助逃犯。

搜查升级为屠杀。监察队为了掩盖误判与逼出逃犯，将残阳坳定为“旧污窝藏点”，屠村焚寨。沈维桑在火场中救出晏南枝，发现她正是监察队追捕的人。母亲罗青禾临死前要求他护送晏南枝离开，并留下父亲旧物：一枚刻着日月双印的驿令，以及一张残缺大陆驿道图。

火场逃亡的第一场战斗必须体现现世武力退化：虫族监察队用刃肢、短矛、弩和清明香逼杀村民，村中没有火枪，也没有能一招翻盘的法师。沈维桑靠猎弓、烟尘、陷阱、短刀和半吊子的边墙武脉救人，第一次灵翼残纹只帮他看见旧驿活路，而不是直接击败监察队。

沈维桑最初护送晏南枝不是为了复国，而是为了完成母亲遗言，并查清父亲死亡与妹妹失踪的真相。二人由此踏上穿越大陆的旅程。
