# 第01集《清明香入村》Evolution Notes

## 状态摘要

- 本集评分：89/100，Decision：pass。
- 不建议再次重写第01集；主要风险已从剧本结构转移到分镜呈现边界。
- 本轮失败模式多数已修复，保留风险集中在序幕时长控制、月白反光边界与药屋院落空间调度。
- 本文仅作为后续可审查提案，不修改 writer-room skill、rubric、template 或 agent task card。

## Project-Specific Lessons

1. 第01集可使用30秒以内的时代背景蒙太奇，但必须只服务当前年景，不给临时战场、征粮和南方密室人物建立支线。
2. 验血钩子有效做法是只保留针刺、血纹、白册旧污疑似三拍。后续同项目开场应继续限制信息密度，让观众先抓住一个核心恐惧。
3. 沈维桑的场内目标必须可见：护妹妹、读懂母亲警告、压住冲动、隐藏井下线索。第02集应让他的主动行动更早出现，而不是只靠反应推进。
4. 罗青禾这类守密角色适合用可解释动作传递风险，例如护药柜、遮手、敲水罐、按草药袋；不应过早确认她具体知道旧驿秘密。
5. 旧驿、日月纹、方向痕等证据必须在升级节点第一次清楚露出。前段只给行为异常，不给观众可直接破案的镜头。
6. 月白玉片在第01集只能作为未命名窄线反光存在。第02集如要揭示，必须明确这是第一次完整识别，而不是重复第01集已经展示过的物件。
7. 清明香、虫蜡针、血纹、虫蜡爬痕等效果已经偏多。导演分镜应统一视觉规则，避免每个道具都发展成独立特效系统。
8. 药屋、后院、水缸、旧井、短矛封锁线的空间关系是第01集最大保留风险。没有空间图时，观众可能不理解沈维桑为何能看见井沿反光、为何不能靠近。

## Reusable Process Lessons

1. Hook gate：首5秒建议限制为最多三个视觉信息点和一个可读文本点，且只能有一个主记忆点。
2. Motivation gate：每场都检查主角是否有可拍的即时目标、风险判断、选择动作和可见代价。
3. Reveal ledger：跨集悬念物件需要记录三个层级：可暗示、可清楚露出、可命名。任何提前命名都应触发 continuity warning。
4. Guardian opacity check：守密角色可以暴露紧张和保护动作，但不能让镜头或旁白提前确认秘密知识范围。
5. Term density check：结尾威胁优先让观众听懂危险，世界观制度词控制在一到两个核心词内。
6. Restraint cost check：主角选择不行动时，必须让观众看见他放弃了什么，例如妹妹受逼退、证据将暴露、武器被压回。
7. Director-room preflight：进入分镜前，应先完成空间图、视线图、封锁线和关键物件可见性边界。

## 第02集优先控制

1. 如果第02集揭示月白玉片，应把完整露出设计成剧情升级点，并避免开场直接命名解释。
2. 沈维桑需要更早采取主动行动，让保护妹妹、追查井下人、规避白翳监视之间形成明确冲突。
3. 旧驿线索继续按层级推进：先异常，再证据，再命名，再解释。
4. 白翳的压迫仍应保持低声登记式，不转成咆哮型反派。

## Director-Room 优先控制

1. 锁定药屋门口、后院、水缸、旧井、短矛封锁线的相对位置。
2. 明确沈维桑、沈照眠、罗青禾、白翳各自能看见什么，不能看见什么。
3. 月白反光只用窄线反射，不拍完整玉片、不拍佩戴者、不拍手部归属。
4. 清明香爬向井沿、虫蜡封条吃痕、血纹爬页应统一运动速度、颜色和质感。
5. 序幕三镜头按北境战场、征粮队、南方密室分配，合计不得超过30秒。
