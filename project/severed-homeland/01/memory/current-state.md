## 已修复的关键问题

1. 开场背景不足：最终稿新增28秒时代背景蒙太奇，用北境战场、征粮队和南方密室交代帝国历1226年的乱世背景，并在0:28切回沈维桑出场。
2. 首段验血钩子信息过挤：最终稿将验血段压成针刺、血纹、白册三拍，并让“旧污疑似”定格半秒。
3. 沈维桑即时动机不够早：最终稿新增他护住沈照眠、读懂罗青禾手势、在旧井前克制喊人的动作链。
4. 药屋过早露出旧驿刻痕：最终稿只保留罗青禾在旧木边停顿和遮手，不提前展示痕迹。
5. 罗青禾旧驿知情程度过明：最终稿降低她主动遮掩的明确性，只通过药名、手势和水罐提示暗示。
