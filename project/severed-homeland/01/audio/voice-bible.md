# 第01集 Voice Bible

## 制作边界

- 本文件锁定第01集中文对白、旁白与功能角色声线。
- 精确对白来自 `01/script/final-script.md` 与 `01/shots/shot-list.json`，不得让视频模型生成画面内文字或口播字幕。
- 每个角色必须使用不同音色、语速、气口和情绪边界；同类肃明角色也要按层级区分。
- 导出短文件名以 `audio/dialogue/d001.wav` 递增，实际文件在录音或 TTS 完成后写入。

## 全局声音风格

- 普通话为主，允许轻微地域质感，但不能变成现代方言喜剧。
- 表演要克制、低压、可听懂；不要广播腔、动漫腔、游戏 NPC 腔。
- 旁白、对白、命令、童声和官吏念册要有明确声线层级。
- 句尾尽量收住，不做舞台剧式拖腔。

## MiniMax / Qwen3-TTS 双路线

- 逐句提示词真源：`01/audio/provider-prompts.json`。
- MiniMax 路线：优先生成表演张力更强的版本，适合旁白、白翳、官吏和罗青禾这类低压声线。
- Qwen3-TTS 路线：并行生成更稳的普通话版本，适合作为咬字、节奏和角色区分的 A/B 对照。
- 两条路线都必须遵守本文件角色声线；不得因为模型默认音色而合并角色差异。
- 同一句导出两版 provider 文件，再按 QC 选定最终短文件名，详见 `01/audio/provider-workflow.md`。
- 配乐提示词也在 `01/audio/provider-prompts.json` 内；Qwen3-TTS 只负责对白、旁白、旧歌/童谣人声层，不作为纯配乐生成器。

## 角色声线

### VO_NARRATOR / 旁白

- 声线：中低音，成年女性或偏中性的成熟声线，冷静、压抑、历史感强。
- 语速：慢到中速，句间留白明显。
- 气口：短句后留 6 到 10 帧空气，给画面和音效压进来。
- 情绪边界：不煽情，不哭腔，不热血；像在宣判一段旧账。
- 混音位置：居中偏前，低频留给战场和制度环境声。

### XUE_LINQIANG / 薛临墙

- 声线：50 岁上下男性，低哑、粗粝、风雪里压出来的墙师声。
- 语速：短促，急但不乱。
- 气口：句前一小口冷气，句尾立刻收住。
- 情绪边界：不喊口号，不英雄化；像把所有人从恐惧里拽回门线。
- 区分点：全集中最“墙上”的人声，低、硬、干，带旧伤和缺粮感。

### GRAIN_TAX_CLERK / 粮税小吏

- 声线：低阶虫族官吏，偏中性男声，干薄、鼻腔窄、尾音像念账。
- 语速：均匀偏快，像读条目。
- 气口：很少自然停顿，逗号处也像算盘拨格。
- 情绪边界：不凶吼，不得意；恐怖来自把活人念成粮册。
- 区分点：比白册官吏更“粮仓”，声线带木秤、骨算盘和湿粮尘感。

### GRAIN_WOMAN / 金河妇人

- 声线：成年女性，饥饿、疲惫、护家，声音薄但不是哭喊。
- 语速：短促，只有一句，像从胸口挤出来。
- 气口：句前有吸气，句尾被押车声压掉。
- 情绪边界：不能撒泼，不能长哭；只保留家粮被夺的一瞬抗辩。

### GU_HUAIZHANG / 顾怀章

- 声线：年长男性旧臣，南方旧礼法气质，低稳、湿润、带压力。
- 语速：中慢，字句清楚。
- 气口：每句都像在把责任压到晏南枝手上。
- 情绪边界：不慈祥，不全然正确；带旧帝国合法性的负担和排除性。
- 区分点：比旁白更有人味，比官吏更温，但更沉重。

### YAN_NANZHI / 晏南枝

- 声线：18 岁女性，清冷、端正、受过旧礼法训练。
- 语速：中慢，短句干净。
- 气口：紧张时先停半拍再问，不能慌乱尖叫。
- 情绪边界：尊严先于恐惧；逃亡疲惫不能削弱气度。
- 区分点：年轻但不幼，清亮但不甜。

### BAICE_CLERK / 白册官吏

- 声线：低阶登记型虫族官吏，偏中性男声，干净、细硬、无温度。
- 语速：平直偏快，像清点、盖印、划名。
- 气口：没有情绪性叹息；顿点像白册翻页。
- 情绪边界：不怒、不笑、不怪叫；开口即让奴兵动手。
- 区分点：比粮税小吏更冷、更洁净、更制度化。

### SHEN_ZHAOMIAN / 沈照眠

- 声线：8 到 10 岁女童，轻、小、自然，带草药名背诵的熟练感。
- 语速：前半句自然快，意识到不对后声音断掉。
- 气口：尾音被白册纸声吞掉。
- 情绪边界：不能成人化表演，不能卖萌；脆弱来自她不知道自己说错了什么。

### LUO_QINGHE / 罗青禾

- 声线：37 岁女性，医者与母亲，稳、低、清醒。
- 语速：中速，危险时压低。
- 气口：句前很少慌张吸气，句尾像把人按住。
- 情绪边界：不哭、不喊；保护来自清醒判断。
- 区分点：温和外层下有硬骨头，和晏南枝的贵气、妇人的饥饿感都不同。

### SHEN_WEISANG / 沈维桑

- 声线：17 岁男性，山路猎户少年，偏低但未完全成年。
- 语速：平时短促，撒谎时半真半假地放松一点。
- 气口：压住弓弦时带轻咳或短呼吸。
- 情绪边界：不能少年热血吼叫；他在算代价。
- 区分点：年轻、耐走、山野，不是书生，也不是军人。

### GATE_SLAVE_SOLDIER / 封门奴兵

- 声线：混血奴兵，成年男性或中性，麻木、僵硬、服从命令。
- 语速：短、直、无解释。
- 气口：像照规矩吐出命令。
- 情绪边界：不能邪恶化；声音里有被制度磨平的空。

### BAI_YI / 白翳

- 声线：高阶清明院虫族，男性表达，中低音，极冷、极稳、无尘。
- 语速：慢于白册官吏，每句都像审定事实。
- 气口：几乎听不到疲劳，停顿像擦拭虫蜡针。
- 情绪边界：不嘶吼，不变态，不怪物化；可怕来自理性与清洁感。
- 区分点：全集中最干净、最冷、最有中枢压迫的声音。

### CROWD_AND_OFFSCREEN / 群众与画外声

- 金河求饶声：低、散、被车轮压掉。
- 残阳坳村民：压低骚动，不能盖住白册官吏。
- 旧歌画外声：南方湿暗里半句被捂住，像秘密被掐断。
- 儿童/老人反应：只做短吸气、闷哼和衣料摩擦，不额外加解释性台词。
