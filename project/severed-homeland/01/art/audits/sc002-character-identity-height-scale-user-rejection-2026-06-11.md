# SC002 角色一致性与身高差用户退回审计 / 2026-06-11

结论：`E01_C016B`、`E01_C017`、`E01_C018` 与 `E01_R005`-`E01_R009` 的当前候选不得继续作为视频生产参考。用户视觉 QC 已明确指出角色一致性仍未满足，包括身高差设定。

## 退回原因

- 全剧母卡与分集参考帧存在尺度冲突：根部 `C016` 曾写 170-185cm，而 Bible 要求基层虫吏 190-210cm；`C018`、`C019` 也存在字段与描述不一致。
- 旧候选仍把“候选通过、等待用户视觉 QC”写入多个 handoff 文件，容易被后续 ComfyUI 或导演提示词误用。
- 参考帧约束仍不够强：只写“同地面”不足以阻止模型用台阶、桌面、粮袋、推车、门槛、前景放大或低伏遮挡制造假身高差。

## 新硬尺度

- `C025` 普通人族基准：儿童 105-145cm，少年 145-165cm，普通成人 160-190cm，老人因佝偻显矮。
- `C017` 混血奴兵：170-200cm；SC002 代表值应读作 175-185cm，保留人族直立骨架、五指硬化人手、骨白腕绑和低位金钱尾辫。
- `C016` 基层虫吏：190-210cm；SC002 代表值应读作 195-205cm，只开册、下令、盖印，不扣腕、不搬粮、不持兵器。
- `C018` 纯虫族小兵：200-230cm；SC002 站立等效应读作 215-230cm，反关节腿、爪足、短矛或钩镰负责封路。
- `C019` 中阶重甲虫士兵：220-260cm；比 `C018` 更宽更重，只用于守门、盾墙或重车押运压力。

## 重生验收门槛

- 角色卡必须包含脚底/靴底/爪足同线的比例条，至少并列 `C025` 成人/儿童基准、`C017`、`C016`、`C018`、`C019`。
- 参考帧必须让可见脚底、爪足或地面接触点落在同一泥地或同一车道地平线。
- 身高差不得来自桌台、检查台、台阶、粮袋、车板、门槛、前景放大、镜头畸变、裁切或低伏遮挡。
- 只要 `C016`、`C017`、`C018`、`C025` 读成同身高阶层，或发生官吏/奴兵/纯虫族小兵职责互换，即失败。
- 下一轮 `E01_R005`-`E01_R009` 必须按 `4096x2304` 重新生成；当前 3840x2160 候选不能进入视频 handoff。
