# 第一季大纲评分

评分对象：第一季开发大纲包，而非最终剧本。

目标分：85。

总分：90。

决策：pass。

## 分项评分

| 维度 | 分数 | 说明 |
| --- | ---: | --- |
| 钩子 | 14/15 | 每集前5秒都有可见危险、道具反转或身份压迫。第04集的磨平炮架刻度是较强世界观钩子，但剧本阶段需强化人物危险。 |
| 动机 | 15/15 | 沈维桑的母亲遗言、妹妹失踪、父亲真相和护送承诺形成持续行动动机。 |
| 冲突 | 14/15 | 追捕、路线争执、纯血复国争议、边墙现实和清明籍压迫逐层升级。第07集陆青砾转向需要在剧本中补足触发。 |
| 节奏 | 13/15 | 12集季结构清晰，前3集立约，中段失史，后段边墙和盟书反转。第04-06集信息密度偏高，需用动作拆解。 |
| 对白潜力 | 12/15 | 当前是大纲，尚未写对白。角色间价值冲突足够强，后续需要避免设定口播。 |
| 可拍摄性 | 18/20 | 大多数场景可通过近景、道具、窄空间和局部动作完成。第08、10、12集有较高制作压力，但已给出替代表现策略。 |
| 连续性 | 4/5 | 道具、地理、角色弧线稳定。后续需严控灵翼残纹能力边界。 |

## 最高价值修改点

进入剧本阶段时，优先处理第04-06集的信息释放节奏。每集只让一个新设定成为主信息，其余内容藏在道具和行动里。

## 通过理由

第一季大纲已经满足短剧开发条件：每集有明确时长、钩子、出场人物、场景概要、故事梗概和结尾悬念；季终完成主题反转，并为南方隐歌谷和清明院妹妹线留下后续推进。
