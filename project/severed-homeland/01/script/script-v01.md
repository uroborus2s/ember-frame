# 第01集《清明香入村》Script v01

时长：约2分45秒  
形式：竖屏短剧

## 1. 外景｜残阳坳村口｜黄昏｜0:00-0:25

【0:00-0:05｜必须钩子】

特写：一个孩子发抖的手腕被按在白布上。  
白翳的虫蜡针垂直落下，针尖像活虫口器，刺破皮肤。

血没有滴落，而是在腕上爬成细细纹路。

清明白册自动翻页，空白处浮出四个冷白字：旧污疑似。

音效：针入肉的轻响。纸页无风自翻。

孩子母亲刚要哭，被虫族监察短矛拦住。

白翳擦净虫蜡针，语气平稳。

白翳：下一个。

竖屏上移：残阳坳村口被螳刃虫族封住。短矛交叉，村道尽头落日发红。

清明香从瓷盂里溢出，贴着地面爬过草鞋、猎靴、门槛。

沈维桑站在人群后，灰褐猎衣，背着猎弓。他先看风向，再看村口泥印。

沈照眠攥着小草药袋，躲在罗青禾身后。

白翳：按户验。无人离坳。

音效：虫甲摩擦。村民屏息。

## 2. 外景｜村口验血处｜黄昏｜0:25-0:45

虫族监察把白布摊开，村民一个个伸腕。

沈维桑被推上前。

虫蜡针悬在他腕上。

罗青禾的目光扫过他左肩，又极轻地摇头。

沈维桑压住呼吸，手腕不动。

针尖刺下。血纹只短短一线，被清明香压住，没有继续散开。

白册浮字：山籍猎户。

白翳抬眼，看沈维桑的旧皮护腕。

白翳：十七？

沈维桑：嗯。

白翳：猎户识路。

沈维桑：只认山路。

白翳没有追问，只把针收回。

白翳：药屋。

罗青禾手指一紧，草药篮里的青黛散出一点粉。

## 3. 内景｜罗家药屋｜黄昏｜0:45-1:20

竖屏窄构图：药柜高而旧，抽屉木牌写着白芷、南星、青黛、女贞。

白翳走进药屋，白袍未沾尘。虫族监察在门外守着。

罗青禾挡在柜前，声音很稳。

罗青禾：乡下药屋，只有伤寒药。

白翳：近日有南来的女子？

罗青禾打开一格药柜。

罗青禾：南星有毒，女贞苦寒。用错了，会死人。

白翳看着她的手。

罗青禾把南星抽屉推回去，指尖在柜底停了半息。那里有一道几乎磨平的旧驿刻痕。

她立刻把草药篮放低，遮住刻痕。

白翳：你答的不是我的话。

罗青禾：监察使问药，我只懂药。

沈维桑站在门边，视线掠过柜底。罗青禾不看他，只把一个空水罐递过去。

罗青禾：井边那味药潮，别碰。去后院取水。

沈维桑听懂了半句，接过水罐。

白翳闻到空气里的清明香回流，停步。

白翳：后院有井？

罗青禾：旧井，水涩。

白翳：涩水也记路。

音效：药柜木头轻响。清明白册在白翳袖中微微翻页。

## 4. 内景｜药屋门口｜黄昏｜1:20-1:45

沈照眠蹲在门槛边，把小草药袋里的药名牌按顺序摆好。

她太紧张，开始小声背。

沈照眠：青黛三钱向西，白芷两钱避井，月过旧驿，门听三息……

罗青禾猛地按住她的嘴。

屋内静住。

白翳缓缓回头，没有提高声音。

白翳：再背一遍。

沈照眠眼睛发红，摇头。

沈维桑挡到妹妹前面。

沈维桑：小孩乱背药歌。

白翳看向沈照眠的小草药袋。

白翳：药歌不会说旧驿。

他翻开清明白册，在空页边添下一个极小的记号：识句。

罗青禾的脸色终于变了。

白翳：孩子记性好。清明院会喜欢。

他说得像在登记粮数。

## 5. 外景｜罗家后院旧井｜暮色｜1:45-2:10

沈维桑拎着水罐来到旧井边。

竖屏俯拍：井口很窄，井绳磨得发黑。清明香的烟线从墙根慢慢爬来。

沈维桑先看井沿灰尘。灰尘上有一道新擦痕。

他把水罐放下，俯身听。

井下很深，先是水声。

然后，一声压得极低的呼吸。

沈维桑握住短刀。

沈维桑：谁？

没有回答。

井壁湿苔间，一角月白玉片反了一下残阳，像碎月一闪。

音效：井下呼吸骤停。远处药屋传来木板被撬开的闷响。

沈维桑回头。

## 6. 内景｜罗家药屋｜暮色｜2:10-2:33

虫族监察掀开药柜前的地板。

地板下露出暗格，暗格边缘刻着残缺日月纹和旧驿方向痕。

一名监察把虫蜡封条按上去。虫蜡沿刻痕爬开，像要把纹路吃掉。

罗青禾站在原地，没有辩解。

白翳蹲下，用干净帕子隔着手指摸过刻痕。

白翳：残阳坳，不只是村。

罗青禾：旧木头，什么痕都有。

白翳：旧木头不会藏在地板下。

沈照眠哭出声，又硬生生憋住。

沈维桑冲进门口，看见暗格，脚步停住。

罗青禾只看他一眼，目光飞快落向后院旧井。

别靠近。别说。

沈维桑把短刀藏回袖下。

## 7. 外景｜药屋外与旧井｜暮色入夜｜2:33-2:45

村民被赶到药屋外。清明香已经盖过门槛，正朝旧井井沿爬去。

白翳站在台阶上，合上清明白册。

白翳：残阳坳全村复查。

他环视村民，声音仍然平。

白翳：隐人、隐路、隐名者，按旧污论。

虫族监察短矛落地，封住药屋与后院。

沈维桑看向旧井。

井下又传来一声极轻的呼吸。

清明香触到井沿的瞬间，井壁那角月白玉片再次反光。

黑场。

音效：白册啪地合死。
