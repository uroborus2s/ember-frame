# 第01集《清明香入村》Final Script

时长：约2分45秒  
形式：竖屏短剧

## 1. 外景｜残阳坳村口验血席｜黄昏｜0:00-0:25

【0:00-0:05｜钩子】

特写：一个孩子发抖的手腕被按在白布上。

虫蜡针落下，针尖像活虫口器，刺破皮肤。

血没有滴落，而是在腕上爬成细细纹路，顺着白布边缘沁到清明白册页角。

清明白册无风翻开，空页浮出四个冷白字：旧污疑似。

四字定格半秒。

音效：针入肉的轻响。纸页翻动。

【0:05-0:25】

孩子母亲猛地上前，被虫族监察短矛拦住。

白翳用白帕擦净虫蜡针，语气平稳。

白翳：下一个。

竖屏上移：残阳坳村口被螳刃虫族封住。短矛交叉，村道尽头落日发红。

清明香从瓷盂里溢出，贴着地面爬过草鞋、猎靴、门槛。

沈维桑站在人群后，灰褐猎衣，背着猎弓。他先看风向，再看村口泥印。

沈照眠攥着小草药袋，躲在罗青禾身后。

白翳：按户验。无人离坳。

音效：虫甲摩擦。村民屏息。

## 2. 外景｜村口验血处｜黄昏｜0:25-0:48

虫族监察把白布摊开，村民一个个伸腕。

沈维桑被推上前。

虫蜡针悬在他腕上。

罗青禾的目光扫过他左肩，又极轻地摇头。

沈维桑压住呼吸，手腕不动。

针尖刺下。血纹只短短一线，被清明香压住，没有继续散开。

白册浮字：山籍猎户。

白翳抬眼，看沈维桑的旧皮护腕。

白翳：十七？

沈维桑：嗯。

白翳：猎户识路。

沈维桑：只认山路。

白翳没有追问，只把针收回。

沈维桑退回人群，第一件事不是看白翳，而是把沈照眠的小草药袋塞回她怀里，把她往自己身后带半步。

沈照眠想说话，他用两根手指轻轻按住袋口。

白翳：药屋。

罗青禾手指一紧，草药篮里的青黛散出一点粉。

## 3. 内景｜罗家药屋｜黄昏｜0:48-1:22

竖屏窄构图：药柜高而旧，抽屉木牌写着白芷、南星、青黛、女贞。

白翳走进药屋，白袍未沾尘。虫族监察在门外守着。

罗青禾挡在柜前，声音很稳。

罗青禾：乡下药屋，只有伤寒药。

白翳：近日有南来的女子？

罗青禾打开一格药柜。

罗青禾：南星有毒，女贞苦寒。用错了，会死人。

白翳看着她的手。

罗青禾把南星抽屉推回去，指尖在柜底一块旧木边上停了半息，像怕他碰坏药柜。

她随手放低草药篮，挡住自己的手。镜头不露她挡住了什么。

白翳：你答的不是我的话。

罗青禾：监察使问药，我只懂药。

沈维桑站在门边，视线先落到妹妹的小草药袋，又落到母亲压在篮沿的手。

罗青禾不看旧井，只把一个空水罐递给他。

罗青禾：后院水缸。井边那股潮味，别碰。

沈维桑接罐时，罗青禾指尖在罐沿敲了两下，又按住沈照眠的草药袋。

沈维桑明白了：带开眼线，护住妹妹，别让井边的东西露出来。

白翳闻到空气里的清明香回流，停步。

白翳：后院有井？

罗青禾：废井，水涩。

白翳：涩水也记路。

音效：药柜木头轻响。清明白册在白翳袖中微微翻页。

## 4. 内景｜药屋门口｜黄昏｜1:22-1:48

沈照眠蹲在门槛边，把小草药袋里的药名牌按顺序摆好。

她太紧张，开始小声背。

沈照眠：青黛三钱向西，白芷两钱避井，月过旧驿，门听三息……

罗青禾猛地按住她的嘴。

屋内静住。

白翳缓缓回头，没有提高声音。

白翳：再背一遍。

沈照眠眼睛发红，摇头。

沈维桑挡到妹妹前面。

沈维桑：小孩乱背药歌。

白翳看向沈照眠的小草药袋。

白翳：药歌不会提旧驿。

他翻开清明白册，在空页边添下一个极小的记号：识句。

罗青禾的脸色终于变了。

白翳：孩子记性好。清明院会喜欢。

他说得像在登记粮数。

沈维桑的手背绷紧，但他没有抬头，只把沈照眠又往身后压了半步。

## 5. 外景｜罗家后院旧井｜暮色｜1:48-2:12

沈维桑拎着水罐来到后院。

水缸在墙边，旧井在旁。井口很窄，井绳磨得发黑。清明香的烟线从墙根慢慢爬来。

沈维桑按母亲的话先走向水缸，没有碰井绳。

一阵极轻的气声从井下擦过。

他停住。

竖屏俯拍：井沿灰尘上有一道新擦痕。

沈维桑先回头看药屋方向。门口，沈照眠被短矛影子逼得往里退。罗青禾站在白翳身侧，眼神不许他喊。

井下又传来一声压得极低的呼吸。

沈维桑握住短刀刀柄，又慢慢松开。

他没有叫人，只用靴尖拨起一点湿泥，压住正爬向井沿的清明香烟线。

沈维桑低声：别出声。

井下没有回答。

井壁湿苔间，夹着一角油纸。油纸包边缘透出一线月白反光，像碎月一闪。

音效：井下呼吸骤停。远处药屋传来木板被撬开的闷响。

沈维桑抬头。

## 6. 内景｜罗家药屋｜暮色｜2:12-2:35

虫族监察掀开药柜前的地板。

地板下露出暗格。暗格边缘第一次清楚显出残缺日月纹，旁边还有被刀尖刻出的旧驿方向痕。

一名监察把虫蜡封条按上去。虫蜡沿刻痕爬开，像要把纹路吃掉。

罗青禾站在原地，没有辩解。

白翳蹲下，用干净帕子隔着手指摸过刻痕。

白翳：残阳坳，不只是村。

罗青禾：旧木头，什么痕都有。

白翳：旧木头不会藏在地板下。

沈照眠哭出声，又硬生生憋住。短矛逼得她退到门框上，小草药袋掉在地上。

沈维桑冲到门口，看见暗格，脚步停住。

他的手已经碰到刀柄。

罗青禾只看他一眼，目光飞快落向后院旧井。

别靠近。别说。护住照眠。

沈维桑喉结动了一下，把短刀藏回袖下，弯腰捡起妹妹的小草药袋。

## 7. 外景｜药屋外与旧井｜暮色入夜｜2:35-2:45

村民被赶到药屋外。清明香已经盖过门槛，正朝旧井井沿爬去。

白翳站在台阶上，合上清明白册。

白翳：残阳坳全村复查。

他环视村民，声音仍然平。

白翳：谁替旧路遮一寸，按旧污论。

虫族监察短矛落地，封住药屋与后院。

沈维桑把沈照眠护在身后，看向旧井。

井下又传来一声极轻的呼吸。

清明香触到井沿的瞬间，井壁那线月白反光再次亮起。

黑场。

音效：白册啪地合死。

## 重写说明

本稿将前5秒压成针刺、血纹、白册三拍，把白翳擦针和“下一个”移到5秒后；药屋前段只保留罗青禾的停顿和遮手，不提前暴露旧驿刻痕；沈维桑的行动改为先护妹妹、读懂母亲对旧井的警告，再在井下呼吸和药屋危机之间选择暂时隐瞒；第01集只写未命名的月白反光，结尾保留“按旧污论”的冷静威胁。
