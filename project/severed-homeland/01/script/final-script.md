# 第01集：白册进村

时长目标：4分30秒
格式：AI竖屏短剧详细剧本
本集功能：用边墙战争、征粮暴政和南方密谋建立世界压力，再把压力压到残阳坳沈家。

## 场景表

1. 锁喉关外墙：边墙战争背景片段。
2. 金河粮仓带：征粮与白册背景片段。
3. 南方红线密室：晏南枝北上背景片段。
4. 残阳坳村口：白册进村。
5. 罗青禾药屋：晏南枝藏身。
6. 旧井检查桌：白翳抵达。

## 成片剧本

### SC01 锁喉关外墙 / 暴雪 / 5秒钩子

暴雪斜砸黑石墙。骨钟被破门猛犸的铁包巨角震响，钟声像骨头裂开。

墙上，一个年轻灰墙军户跪倒，手里的长矛断成两截。墙外黑牙斧兵砍向拒马，雪狼骑从风雪里绕向烽燧。

骨钟裂开一道细缝，铜锈里露出旧日月刻痕。年轻军户伸手去扶钟，掌心立刻被冻血黏住。

旁白：北墙五百年，血从未干。

薛临墙的手按住墙缝，掌心全是冻裂血。他没有喊冲锋，只低声命令。

薛临墙：补枪。别看兽，看门。

### SC02 金河粮仓带 / 清晨

麦金色粮袋堆成墙。虫族粮税小吏戴高账吏帽，白册夹板悬在胸前。

奴兵掀开一家人的粮缸，只倒出半斗杂粮。老人想护住祖牌小盒，被缚网奴兵按到泥里。孩子被迫在虫蜡封条上按手印。

粮税小吏：查清明籍。金河不清，粮也得清；族谱对不上，粮就不算你家的。

妇人：这是过冬粮。

粮税小吏抬手，押车奴兵把粮袋拖走。

旁白：金河不清，白册先清户。官笔一点，良户成逃籍；粮归官仓，人归押车，祖牌还在，家门没了。

### SC03 南方红线密室 / 雨夜

雨敲油布。顾怀章摊开旧驿残图，红线从南缘一路钉到北境。晏南枝戴着遮面斗篷，指间握着月白玉片。

顾怀章：你不是逃。你是日月还在人间的证据。

晏南枝：证据能让北边的人开门吗？

顾怀章把半卷血牒推给她。

顾怀章：找到月下盟书。旧臣看见它，就会记起该跪向谁。

晏南枝没有立刻接。雨声里，有人低声唱旧歌，唱到一半被手捂住。

旁白：南方遗民仍在旧歌里找路，却不知道路尽头等着的，不一定是皇座。

### SC04 残阳坳村口 / 白天

画面切回肃明历1226年，残阳坳。

村口摆起检查桌。白册官吏念名，奴兵扣腕排队，纯虫族小兵封住旧井和药屋外圈。村民穿旧麻短袄，粮袋、药篮和柴刀都被堆到一旁。

白册官吏：残阳坳，原旧驿地名一处。按清明籍，逐户复核。

沈维桑背弓站在人群边，左肩微低。他先看旧井，再看后山出口。

三处出口已经各插一支白蜡旗。旗上虫蜡未干，说明封村不是临时起意。

沈照眠抱着草药袋，听见官吏念错一个古旧地名，下意识接了半句禁歌。

沈照眠：日月重明，雪夜留门。

村口忽然静下。白册官吏抬头，复眼转向她。

罗青禾一步挡到女儿身前，手里还捏着药剪。

罗青禾：孩子背药名，背串了。

白册官吏：药名里有日月？

沈维桑挤出笑，半真半假地插话。

沈维桑把沈照眠的草药袋勾到自己脚边，指腹按住弓弦，硬把那半句歌压成一声咳。

沈维桑：那是后山旧药路，我让她背来认草。官爷要验旧话，就验我这张猎户嘴。

白册官吏翻开白册。纸面浮出淡白虫蜡纹。

白册官吏：沈家。祖上这一笔说不清，先挂疑名。

他把一炷清明香插进检查桌裂缝。

白册官吏：香烧完还说不清，全家按疑户押册。

### SC05 罗青禾药屋 / 同时

药屋内，晏南枝藏在药柜暗格里。她透过木缝看见虫族小兵的爪足在门外停住。

罗青禾进屋抓药，声音压低。

罗青禾：别动。白册闻得到慌。

晏南枝：你知道我是谁？

罗青禾：知道你会害死这里的人。

晏南枝的手按住月白玉片，脸色一白。

罗青禾打开药柜底板，露出一道旧驿暗号。暗号被草药灰遮了多年，仍能看出日月残纹。

晏南枝：这是昭明驿记。

罗青禾：所以我更知道，皇旗救不了门外的人。

罗青禾：旗一露，残阳坳就不是村，是反寨；白册一写，良户也成逆户。

门外，封门奴兵开始敲门。

封门奴兵：开药柜。

### SC06 残阳坳旧井旁 / 午后

村民被赶到旧井边。虫蜡封门牌挂上药屋门框。沈维桑的视线落在药屋窗缝，又迅速转向后山风口。

白册官吏把药柜底板的旧驿暗号摆到检查桌上。

白册官吏：查出旧驿暗号，按窝户论。先封全村户口，谁家都不许改供。

人群炸开。奴兵扣住几个少年，虫族小兵用钩镰堵死退路。

沈照眠被吓得去摸草药袋。沈维桑伸手要拉她，白册官吏的蜡笔先落到纸上。

白册官吏：沈照眠，听旧歌有反应。先记作疑童。

远处传来清明香铃声。白袍高瘦的白翳走进村口，风里没有一点尘沾到他的衣角。

白翳停在检查桌前，擦了擦虫蜡针。

白翳：一块暗号藏不住这么多年。有人开过门，也有人带过路。

他看向沈家三人，白册上缓缓浮出“沈家旧姓旁注”。

白翳：残阳坳按逆案查办。封村，清户，搜旧井。

沈维桑的手按上猎弓。罗青禾隔着人群看他，极轻地摇头。

白翳翻到新的一页。

白翳：先从孩子开始。

白翳：大人能改供，孩子改不了童谣。祖上留的暗号，歌一接，根脚就露了。

沈照眠被奴兵拉出半步。沈维桑眼神冷下，风声里只剩骨钟般的白册翻页声。

他的拇指扣住弓弦，又缓缓松开。罗青禾的眼神告诉他：这一箭若出，全村先死。

切黑。
