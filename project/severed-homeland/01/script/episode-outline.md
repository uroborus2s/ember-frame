# 第01集 Outline：清明香入村

1. 时代背景：帝国历1226年，北方十三部落联盟大举入寇，战场、征粮队和南方人族密谋在28秒内交代民不聊生与起事暗流。目的：让残阳坳搜查成为乱世切面。情绪：压抑、饥荒、暗火。视觉：雪地战场、征粮木牌、旧驿图、断裂日月纹。
2. 主角入场与钩子：沈维桑在人群后观察风向和泥印，随后虫蜡针刺破孩子手腕，白册浮出“旧污疑似”。目的：从宏观乱世切到主角所在村庄。情绪：村民屏息。视觉：猎弓、白袍、虫蜡、白册。
3. 封村：螳刃虫族封住村道，清明香沿地面蔓延。目的：切断退路。情绪：沈维桑警觉。视觉：烟线、短矛、村口残阳。
4. 药屋试探：白翳询问罗青禾是否见过南方女子，罗青禾用草药名掩护暗语。目的：展示母亲知道旧驿秘密。情绪：压抑对峙。视觉：药柜、草药篮。
5. 妹妹失言：沈照眠背出旧驿方向歌，白翳停步回头。目的：把清明院识别线埋下。情绪：沈维桑恐惧。视觉：小草药袋、白翳复眼纹。
6. 旧井异响：沈维桑去井边取水，听见井下呼吸。目的：引出逃犯。情绪：怀疑。视觉：井口阴影、月白反光。
7. 暗格暴露：监察队撬开药屋地板，露出旧驿刻痕。目的：搜查升级。情绪：村民绝望。视觉：日月残纹、虫蜡封条。
8. 结尾转折：白翳宣布残阳坳接受清明复查，任何隐瞒者按旧污论处；沈维桑看向旧井。目的：逼入第02集火场。视觉：白册合上，清明香覆盖井沿。
