# 第01集《白册进村》摄影方案 / Camera Plan

## 全局摄影锁 / Global Camera Lock

中文：16:9 横屏宽屏以“上下层级”和“窄门槛压力”为核心。镜头不追求漂亮空景，必须服务制度压迫、人物克制和地理可读。运动速度保持真实，不用漂浮慢动作；手持只用于人群压力和门槛推搡，白册与白翳镜头尽量稳定、干净、冷。

English: The 16:9 frame is built around vertical hierarchy and threshold pressure. Shots should not chase beauty plates; they must serve institutional pressure, restraint and readable geography. Motion stays real-time, with no floaty slow motion. Handheld is reserved for crowd pressure and threshold contact; register and Bai Yi shots stay stable, clean and cold.

## 场景摄影策略 / Scene Camera Strategy

### SC001 锁喉关外墙 / Suohou Gate Outer Wall

- 镜头：`SC001-SH001` 到 `SC001-SH003`
- 景别：斜侧攻城远景到城头中景偏近、局部中近景、薛临墙中近景。
- 光线：暴雪冷白为主，加入云层裂隙冷天光、雪雾体积层、远处火线暖反打和黑石大暗面；火罐与冻血暗红只做局部信息，禁止圣光化。
- 运镜：开场从攻城方斜后侧低机位起，不做正面居中冲城。兽潮从左前向右后斜推城门，镜头顺旗杆、烟柱、巨角和黑墙外立面起吊，落到城头中景偏近的骨钟、黑石女墙和守军肩背；墙外兽潮仍在斜下方背景可见。骨钟横摆和暴雪雪雾可作为隐藏剪切点，禁止穿墙、180 度调头和游戏感飞镜。
- 连续性：骨钟、断矛、旧日月刻痕必须同场建立，后续旧帝国残痕才有重量。

### SC002 金河粮仓带 / Jinhe Grain Depot

- 镜头：`SC002-SH001` 到 `SC002-SH004`
- 景别：横屏远景建立粮袋墙，中近景读制度流程，门槛特写读家庭被写没。
- 光线：清晨麦金冷雾、湿木褐、虫蜡白；不要拍成灰色灾民营。
- 运镜：粮袋和押车保持横向运动，家庭门槛保持纵深不动，形成“粮走了，家留空”的反差。
- 连续性：粮税小吏高账吏帽、白册夹板、骨算盘和粮牌必须明显；奴兵扣腕和押车，不承担开册职责。

### SC003 南方红线密室 / Southern Red-Line Room

- 镜头：`SC003-SH001` 到 `SC003-SH004`
- 景别：桌面特写、双人中近景、血牒手部插入、红线过渡。
- 光线：雨夜油灯、湿暖低照；月白玉片只做手心冷反光。
- 运镜：沿红线向北推，保持旧驿路线的方向感；不要给晏南枝正面华丽揭示。
- 连续性：顾怀章在灯侧，晏南枝在阴影侧，红线始终位于两人之间。

### SC004 残阳坳村口 / Canyangao Gate

- 镜头：`SC004-SH001` 到 `SC004-SH009`
- 景别：先用宽镜锁定检查桌、旧井、药屋、后山出口，再进入禁歌与圆谎近景。
- 光线：土褐日光与虫蜡冷白对冲；白蜡旗、白册、清明香要从生活色里跳出来。
- 运镜：沈维桑的 POV 用三点眼线：旧井、药屋、后山出口；白册官吏镜头稳定，村民镜头可轻微手持。
- 连续性：三支白蜡旗必须在 `SC004-SH001` 和 `SC004-SH009` 可见，证明封村早已布好。

### SC005 罗青禾药屋 / Luo Medicine House

- 镜头：`SC005-SH001` 到 `SC005-SH005`
- 景别：暗格 POV、抓药中景、柜内外分割近景、旧驿暗号插入、门影压力。
- 光线：药屋内草药青暗光，门缝虫蜡冷白；晏南枝藏身不可过亮。
- 运镜：多数保持静，紧张来自木板、呼吸和门影；旧驿暗号只用手擦灰揭示。
- 连续性：罗青禾始终站在门与暗格之间，既是医者动作线，也是保护线。

### SC006 旧井检查桌 / Old Well Check Table

- 镜头：`SC006-SH001` 到 `SC006-SH010`
- 景别：井边宽镜、桌面罪证、层级群像、白翳入场全身、中近景审讯、弓弦结尾特写。
- 光线：午后硬光逐渐被虫蜡白压住；白翳入场后画面变干净、低噪、低动作。
- 运镜：白翳移动用直线后退镜；白册与虫蜡针用静态插入；沈维桑扣弦用极近景。
- 连续性：井口、检查桌、药屋封门牌要在 `SC006-SH001`、`SC006-SH002`、`SC006-SH008` 三次确认。
