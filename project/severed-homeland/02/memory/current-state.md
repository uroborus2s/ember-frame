## 剧情终态

残阳坳搜查升级为制度化屠村。第01集结尾的「旧污疑似」、药屋暗格、旧井月白反光和全村复查，在第02集开场转化为清明白册上的「旧污窝藏」虫蜡封条。白翳以「还村清明」下令，火把落向村屋，清明院的清除逻辑从怀疑进入执行。

沈维桑在火场中先保护沈照眠，再按罗青禾手势打开旧井暗门，救出晏南枝。晏南枝出场先复现第01集的一线月白反光，随后露出裂月形玉片；她怀中油纸只短暂露出血色细纹和残缺日月印，不解释身世。

罗青禾在药屋烟火中断后，把半枚日月驿令交给沈维桑，并用核心遗言锁定护送契约：「别替旗送命。送她去月下，不是送她去复国。」沈维桑并非认同复国叙事，而是承接母亲临终托付。

沈照眠被清明院白衣童役识别，随后由肃明奴兵扣上虫蜡环、套押车绳并推上虫族木车带走。沈维桑射箭救妹失败，箭被螳刃截断，确认他不是轻易放弃，而是在奴兵押车和螳刃追兵压迫下被迫撤离。

谷口突围依赖猎户经验、滚木陷阱、烟尘、兽道和半成品明劲。灵翼残纹只在沈维桑左肩旧伤被划开时短闪痛感，不负责找路、不击败敌人。半成品明劲只震松滚木绳结一指，代价为鼻血、左肩撕裂、左臂失力。

结尾，沈维桑在虫族车辙旁捡到沈照眠烧焦发绳，确认寻妹线成立。半枚日月驿令被血灰蹭亮后只微热指向北坡山腹黑石缝。没有石门亮起；这是第03集石门开启的延迟揭示控制点，不是本集失败。
