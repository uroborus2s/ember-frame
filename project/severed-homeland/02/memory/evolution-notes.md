# 第02集 Evolution Notes：火里的逃犯

## 可保留经验

- 短剧钩子应直接落在危险、道具反应或价值冲突上。
- 历史信息优先用可见物承载，例如白册、血牒、旧驿暗语、骨笛、阵基、虫蜡档案墙和盟书残页。
- 每集都应让沈维桑的“先救活人”与晏南枝的“日月旗如何重释”发生一次具体摩擦。

## 不自动修改 skill

本文件只记录项目经验，不自动改写 writer-room skill、agent task card、rubric 或模板。

## 95分目标精修经验

- 95 分目标不需要重写主线，优先把抽象主题变成可见道具、时限、身体代价和角色选择。
- 本集有效增强点：押车入外署时限、车轮节奏、识别署风险和护送契约的还债属性被压实。
