# 第02集 Evolution Notes：火里的逃犯

## 可保留经验

- 短剧钩子应直接落在危险、道具反应或价值冲突上。
- 历史信息优先用可见物承载，例如白册、血牒、旧驿暗语、骨笛、阵基、虫蜡档案墙和盟书残页。
- 每集都应让沈维桑的“先救活人”与晏南枝的“日月旗如何重释”发生一次具体摩擦。

## 不自动修改 skill

本文件只记录项目经验，不自动改写 writer-room skill、agent task card、rubric 或模板。

## 95分目标精修经验

- 95 分目标不需要重写主线，优先把抽象主题变成可见道具、时限、身体代价和角色选择。
- 本集有效增强点：押车入外署时限、车轮节奏、识别署风险和护送契约的还债属性被压实。

## 词顺口润色经验

- 设定名词可以保留，但关键台词必须马上接入族谱、户籍、粮牌、账本、旧案、讨债等中国式历史语义。
- 谜语式说明应改成角色能在冲突现场说出口的短句，优先让官吏、逃亡者、册外人和守边者各自用自己的身份说话。
- 本集有效增强点：旧明血纹、清污、洗名验歌等说明已落到血脉日月纹、乱根、改名套旧歌语义。

## 隐语典故润色经验

- 月饼藏信、河不清、白莲香会等联想可以转译为本世界的月饼、金河、白墙、清明香和白册，而不是直接讲历史典故。
- 隐语必须服务场景动作：递信、验名、查粮、渡河、读盟书、救活人。
- 本集有效增强点：在罗青禾交驿令时加入“月饼能藏信，血牒也能藏祸”，把月饼藏信的隐语落到护送与血牒代价。
