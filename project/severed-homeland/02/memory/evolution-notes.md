# 第02集 Evolution Notes

> 说明：以下内容只是学习演进提案，用于未来 writer-room 运行时参考。未经用户明确批准，不得修改 `writer-room` skill 本体、角色卡、评分规则或模板文件。

## 本集可复用经验

1. 制度暴力要先给可拍凭证，再给动作结果：白册、虫蜡封条、「旧污窝藏」比旁白解释更快建立屠村动机。
2. 跨集道具揭示适合采用「上一集视觉特征复现 → 局部露出 → 完整确认」的递进节奏，避免新信息突兀。
3. 重大身份与世界观信息应分层释放：本集只让血色日月残印一闪，不让角色说透血牒、身世和复国背景。
4. 遗言类台词要短、硬、能约束后续剧情。本集「送她去月下，不是送她去复国」明确锁定沈维桑的私人承诺动机。
5. 主角被迫撤离前必须有可见失败动作。本集射箭救妹失败后再撤，能支撑后续寻妹执念。
6. 低魔动作要有明确物理目标、有限效果和即时代价。半成品明劲只震松湿绳一指，同时造成鼻血、左肩撕裂和左臂失力。
7. 能力伏笔可制造痛感、危险或短暂共鸣，但不能替代角色经验、观察、找路或战斗胜利。
8. 下一集接口只需给方向、压力和未完成问题，不应提前消费下一集的奇观事件。

## 建议加入未来角色卡的检查项

1. `script-doctor-agent`：检查每个重大暴力升级前是否存在制度凭证、物证或执行口令，避免暴力显得随机。
2. `rewrite-agent`：每场最多完整揭示一个核心信息，其余设定必须通过抢夺、遮掩、反应、道具边角或追捕行为保留悬念。
3. `continuity-agent`：跨集道具首次清晰露出时，必须先复现上一集已建立的视觉记忆点。
4. `dialogue-agent`：遗言、契约、誓言类台词必须短句化，并检查是否能约束角色后续选择。
5. `script-doctor-agent`：主角放弃亲人、目标或原则前，必须至少有一个可拍的失败行动。
6. `rewrite-agent`：低魔能力不得直接完成找路、杀敌、开门或解决路线选择；必须依赖环境、经验和代价。
7. `continuity-agent`：制度设定可先展示基层执行者与动作结果，正式名词和完整机制留到对应集数揭示。
8. `script-evaluator-agent`：通过稿仍需记录「刻意延迟揭示」与「真实残余风险」的区别，避免后续误修。

## 建议加入评分/连续性门禁的检查项

1. Hook 门禁：前5秒危机是否包含可视化因果，而不只是灾难动作。
2. 信息密度门禁：单场是否同时解释超过一个核心身份、道具或世界观概念。
3. 跨集承接门禁：关键道具是否先复现旧视觉特征，再扩展新含义。
4. 动机门禁：沈维桑的护送动机是否来自私人承诺与罗青禾托付，而非复国效忠。
5. 低魔门禁：灵翼残纹和明劲是否都有物理边界、有限效果与持续代价。
6. 失败动作门禁：主角撤离、放弃或转向前是否已经尝试并失败。
7. 可拍性门禁：火场、虫族追兵、滚木陷阱是否能用近景、声响、局部肢体、烟尘和动作结果完成。
8. 下一集接口门禁：结尾是否只提供方向和压力，不提前完成下一集应发生的新事件。

## 不要改动的规则

1. 不要把第02集「没有石门亮起」当作需要修复的问题；这是第03集石门点亮的接口控制点。
2. 不要让第02集末尾的黑石缝被理解为已经开启的石门；只能保留微热、冷风和方向提示。
3. 不要让灵翼残纹替沈维桑找路、杀敌或解决路线选择。
4. 不要让半成品明劲变成击飞敌人的强力觉醒。
5. 不要让沈照眠在第03集快速脱困；她应继续进入清明院训练线。
6. 不要让晏南枝一次性解释完整身世、血牒、月下和复国背景。
7. 不要把白翳写成咆哮型或私人复仇型反派；他应保持登记、白册、清明香和制度执行气质。
8. 不要自动修改 `writer-room` skill 本体。所有演进建议必须先作为提案记录，待用户批准后才可执行。

## 下一次 writer-room 启动提示

启动第03集时，请优先继承第02集的通过稿状态：沈维桑左肩撕裂、鼻血后虚弱、左臂失力，不能满弓恢复；他同时背负护送晏南枝去月下与寻找沈照眠两条目标。晏南枝与沈维桑只是临时同盟，仍存在不信任和路线分歧。

第03集开场建议从北坡山腹黑石缝与半枚日月驿令真正触发石门点亮开始。注意：第02集没有石门亮起，这是正确的延迟揭示，不是缺陷。路线冲突应升级为「追北线车辙/旧驿线索」与「南下避清明香保血牒」之间的战略选择，不要重复一句南北分歧。

继续控制低魔边界：灵翼残纹只能提供痛感、短暂共鸣或危险提示；明劲只能产生有限物理助力，并且必须持续消耗沈维桑。血牒、月下、复国和晏南枝身世仍应通过物证、追捕反应和短句分段释放。
