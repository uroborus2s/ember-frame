# 第02集《火烧残阳坳》Storyboard Plan

## 使用说明

本 storyboard plan 按 `02/shots/shot-list.json` 的 45 个 shot ID 编排。每个 panel 只描述可画、可拍、可生成的画面，不改剧情，不新增台词。所有构图按 9:16 竖屏处理，优先使用近景、极近景和可读的局部动作。

## 全集连续性锁

- 北坡、北偏车辙、木车离开方向、黑石缝统一为画面右侧或右上方。
- 南侧黑林统一为画面左侧。
- 残阳坳火场、旧井、药屋作为身后或下方背景方向。
- 清明香是贴地淡灰冷白烟线，不是追踪界面。
- 裂月玉片、油纸血牒、灵翼残纹、半枚驿令都只短暂可见，不扩大成解释性特效。
- 沈维桑谷口后必须保留鼻血、左肩撕裂、左臂失力；晏南枝短刀崩缺后不得恢复；沈照眠被带走后不再回到主角身边。

## SC001 制度落火：白册先于火焰

### SC001-SH001

- 构图/层次：极近俯拍，前景是清明白册页角压住画面下半部，中景是虫蜡封条和药屋暗格木屑，背景虚化到冷白纸纤维。
- 人物/动作：无人正脸；机位像登记桌上的冷眼，蜡内细足在浅景深边缘蠕动。
- 道具/光线：白册文字“旧污疑似”旁浮出“旧污窝藏”，虫蜡封条居中偏右，冷白光压住火场暖色。
- 转场：微推到“旧污窝藏”，以纸页低响接白翳指尖。
- 连续性/参考：`PROP-QINGMING-WHITE-REGISTER`、`PROP-INSECT-WAX-SEAL`；制度命令先于火焰出现。

### SC001-SH002

- 人物/动作：白翳垂直站位，动作干净平稳，指尖停顿后合上白册，视线藏在暗色复眼纹里。

- 转场：册页合死声硬切到火把脱手。
- 连续性/参考：`CHAR-BAIYI-02`；白翳是制度化执行者，不演成狂热屠夫。
