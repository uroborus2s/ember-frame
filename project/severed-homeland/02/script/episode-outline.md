# 第02集 Outline：火烧残阳坳

1. 钩子：白翳下令清明复查，火把落下。目的：搜查升级为屠杀。情绪：恐惧骤然爆发。视觉：白袍不沾灰、火光吞屋。
2. 井下救人：沈维桑打开井下暗门，晏南枝用短刀抵住他。目的：主角与逃犯相遇。情绪：互不信任。视觉：月白玉片、血牒油纸。
3. 药屋诀别：罗青禾挡住追兵，把日月驿令交给儿子。目的：建立护送契约。情绪：私人悲痛。视觉：药柜燃烧、半枚驿令。
4. 妹妹失踪：沈照眠被白衣童役识别，奴兵扣腕押车，螳刃追兵斩断沈维桑救援箭线。目的：形成主角个人目标，并建立肃明抓捕任务中的奴兵/虫族分工。情绪：沈维桑失控。视觉：小草药袋掉落、虫蜡环、奴兵旧麻军袍、螳刃切箭。
5. 谷口突围：沈维桑用猎弓射断绳扣，引落滚木陷阱，强行爆发明劲救晏南枝。目的：展示低魔战斗。情绪：求生。视觉：烟、短刀、刃肢影子。
6. 母亲遗言回响：晏南枝想独自逃跑，沈维桑按住伤口说“我答应了”。目的：护送关系成立。情绪：冷硬承诺。视觉：焦痕衣角。
7. 结尾转折：车辙旁留下沈照眠发绳，白翳远处闻到清明香残味。目的：妹妹线和追捕线并行。视觉：发绳、白册、火场远景。
