# 第02集《火烧残阳坳》Final Script

时长：约3分钟  
形式：竖屏短剧

## 1. 外景｜残阳坳药屋前｜夜色初起｜0:00-0:18

【0:00-0:05｜前5秒钩子】

极近景：清明白册页角，上一集写下的「旧污疑似」旁，多出一枚虫蜡封条。封条压着药屋暗格木屑，蜡里细足蠕动，冷白字浮起：旧污窝藏。

白翳的指尖停在封条上，白袍未沾灰。

白翳：还村清明。

他合册。册页合死的轻响，变成火把脱手声。

两名螳刃追兵抬起刃肢。几名旧麻军袍的奴兵同时松手，火把从竖屏上沿坠下，砸进两侧村屋屋顶。

【0:05-0:18】

火光炸开。饭碗滚进火里，门闩被短矛挑断，窗纸卷黑。村民呼喊不见全景，只见手、脚、烟和倒下的木梁。

沈维桑把沈照眠按到水缸后，先看风向，再看药屋后院。

沈照眠：哥……

沈维桑：低头，捂住袋子。

清明香贴地爬向旧井。白翳站台阶上，声音仍平。

白翳：旧污不留屋。

## 2. 外景｜罗家后院旧井｜夜｜0:18-0:43

罗青禾隔着烟看见沈维桑，只用两根手指点向井，又点向谷口。

沈维桑把沈照眠从水缸后推到相邻的柴垛和水沟口，抓起她的小草药袋按回她怀里。

沈维桑：数到三十，没听见我，就钻水沟。

沈照眠摇头。他没等她回答，冲到旧井边，掀开井绳下的湿苔石。

暗门缝里先闪出一线月白反光，和昨夜井沿那一下完全相同。

下一瞬，一只握短刀的手探出。晏南枝从井壁里跌出，刀尖抵在沈维桑喉侧。火光掠过她胸前，才看清那线反光来自一枚裂月形玉片，边缘被灰尘盖住，仍冷得像井水。

晏南枝：谁让你开门？

沈维桑看她刀尖，又看追兵影子已经压上井沿。

沈维桑：想死就关回去。

她没有收刀，只把怀里的油纸往衣内压紧。油纸边角被火照亮一瞬，血色细纹只露出残缺日月印，随即被她手背盖住。

井口上方，螳刃短矛敲到石沿。

## 3. 内景｜井下暗道口/药屋后门｜夜｜0:43-1:08

沈维桑一把按灭爬到井边的清明香烟线，拉晏南枝钻出暗门。她踉跄，短刀仍横在两人之间。

沈维桑低声：白翳找的是你怀里那张纸。

晏南枝手指收紧红线，没解释，只反手把油纸压得更深。

沈维桑捡起湿苔石复位，故意把井绳踢进火星里。烟一涨，井口被灰挡住。

药屋方向传来药柜倒地声。

沈维桑：走后门。

晏南枝盯他。

晏南枝：你认路？

沈维桑没有答。他抓一把灰抹在自己和她衣角上，贴着烟最厚的墙根跑。

## 4. 内景｜罗家药屋｜夜｜1:08-1:43

药屋里全是蓝黑烟。青黛粉被罗青禾砸进火堆，虫族追兵被呛得后退。药柜抽屉燃烧，白芷木牌裂开。

罗青禾用身体顶住半扇门，肩侧已被螳刃划开。

罗青禾：后门，快！

沈维桑冲进来，第一眼找妹妹。

沈维桑：照眠还在外面！

罗青禾抓住他的手，把半枚日月驿令塞进掌心。铜玉混色，断面日月纹只剩半圈，烫得像刚从火里取出。

沈维桑：她到底是谁？

罗青禾看一眼晏南枝，又看回沈维桑。她的声音被烟磨哑，却字字清楚。

罗青禾：别替旗送命。送她去月下，不是送她去复国。

沈维桑手指收紧，半枚驿令把掌心硌出血。

沈维桑：我不懂月下。

罗青禾：路会懂你。

门缝被刃肢刺穿。罗青禾猛地推倒药柜，柜体砸住追兵前腿，也挡住沈维桑回身的路。

沈维桑：娘！

罗青禾隔着火看他，只做最后一个口型：走。

## 5. 外景｜药屋侧巷/虫族木车旁｜夜｜1:43-2:10

沈维桑拖着晏南枝冲出侧门。烟后，沈照眠从柴垛旁的水沟口被呛出来，眼里全是火光。

沈照眠：哥！

一名清明院白衣童役从木车旁钻出，个子不高，袖口挂着白册小签。两名奴兵守在车边，旧麻军袍外缝虫蜡白绑带，耳后细甲鳞被火光照亮，手里握着押车绳和叉矛。

白衣童役盯住沈照眠腕上昨夜留下的「识句」虫蜡痕，又拾起她散落的药名牌。

白衣童役：这一个，带院里。

虫蜡环啪地扣住沈照眠手腕。她的小草药袋掉进泥里，药牌散开。

沈维桑搭箭，箭离弦直奔扣环。

一名奴兵用押车绳套住沈照眠肩侧，另一名奴兵把她往木车边拖。螳刃追兵侧身一切，箭杆在半空断成两截。前半截钉进木车边板，离沈照眠手臂只差一寸。她吓得缩肩，被奴兵推上车。

沈维桑再摸箭，背后只剩空羽。他要冲过去，另一名追兵刃肢横扫，晏南枝以短刀硬挡，刀口崩缺，整个人被震到墙上。

晏南枝：烟要散了！

车轮压过碎瓦，向北侧山道拖出深辙。沈照眠的哭声被火和车轴声截断。

沈维桑捡起泥里的小草药袋，塞进怀里，眼睛死死盯着车辙。

沈维桑：谷口。

## 6. 外景｜残阳坳谷口猎人陷阱｜夜｜2:10-2:42

谷口窄，烧焦草堆下藏着旧猎人的滚木绳扣。沈维桑伏地摸泥，闻到兽骚味，先看见北坡石根下被兽蹄磨亮的窄缝，缝口还挂着自己早年设陷留下的旧麻绳。

晏南枝靠在石边，短刀横胸，刀口缺了一牙。

晏南枝：你不是武者。

沈维桑：半个都不算。

螳刃影子切开烟尘。沈维桑一箭射出，不射人，射断草堆后的绳扣。滚木轰然坠下，压住一名追兵腿节，把第二名撞进石壁。

第三名从侧面跃出，刃肢划开沈维桑左肩旧伤。痛意炸开，他眼角和肩胛金赤一闪，像伤口里有火针扎过，立刻熄灭。

晏南枝的短刀刺进追兵腿节，却被滚木和甲壳卡死。追兵抬起另一只刃肢。

沈维桑扑过去，掌根抵住滚木尾端卡死的绳结，脚踩石槽，短促一吐劲。半成品明劲只在掌根炸开一声闷响，不击飞敌人，只把湿绳震松一指。

滚木偏开半寸，腿节缝隙张开，晏南枝拔出短刀，反手割断追兵筋膜。

沈维桑鼻血滴到灰里，左肩伤口撕开，左臂立刻垂下。

白翳的声音从远处烟后传来。

白翳：他在谷口。

沈维桑咬住痛，指向石根窄缝。

沈维桑：钻兽道。

## 7. 外景｜谷口外坡道｜夜｜2:42-2:53

两人从窄缝滚出。晏南枝第一反应往南侧黑林看，那里风大，清明香烟味被吹散。

晏南枝：往南，先离开清明香。

沈维桑靠在石上，按住左肩，视线却钉在谷下车辙。两道车印绕过火场，偏向北坡。

沈维桑：车往北偏了。

晏南枝：你要回去？

沈维桑看她，又看怀里的小草药袋。

沈维桑：我答应了送你，不等于我不找她。

远处螳刃刮过石声逼近。晏南枝沉默半息，把短刀收低。

晏南枝：先活过这一段。

沈维桑：同一句话还你。

## 8. 外景｜残阳坳外山坡/车辙旁｜夜｜2:53-3:00

沈维桑回望残阳坳。火光舔过旧井，药屋梁木塌下，罗青禾的青布头巾在药屋门槛燃起，白册纸灰在热风里翻飞。

虫族木车车辙旁，一截烧焦发绳被压进湿灰，发绳上还系着沈照眠的小草药结。

沈维桑蹲下捡起发绳。血从掌心淌过半枚驿令，混着灰一蹭，断面日月纹微微亮净，又很快暗下，只剩一丝热意朝北坡山腹的黑石缝牵去。

没有石门亮起。只有石缝里吹出一点冷风。

晏南枝看见他掌心，又看南边黑林。

远处火场边，白翳停步，闻到被烟冲淡的清明香残味。他翻开白册，在「旧污窝藏」下落笔。

黑场。

音效：木车远去，发绳在掌心被攥紧。

## 重写说明

本稿将前5秒改为白册、虫蜡封条和「旧污窝藏」近景后再落火把，明确屠村是清明院制度命令执行；晏南枝出场先复现第01集的一线月白反光，再完整露出裂月形玉片，并把油纸内的血色日月残印控制为一闪，不让角色解释身份。罗青禾遗言改为「送她去月下，不是送她去复国」，沈照眠由清明院白衣童役识别，奴兵扣腕押车，螳刃追兵切断救援箭线，形成“奴兵抓人、虫族断路”的抓捕分工。谷口突围降级灵翼残纹为痛感伏笔，逃路来自猎户兽道和旧陷阱经验；半成品明劲只震松滚木绳结，代价是左肩撕裂、鼻血和左臂失力。结尾保留发绳车辙，并用半枚驿令被血灰蹭亮、微热指向北坡石缝，衔接第03集路线争执和石门开启，但不提前点亮石门。
