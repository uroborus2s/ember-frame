# 第02集：火里的逃犯

时长目标：4分20秒
格式：AI横屏宽屏剧详细剧本
本集功能：完成屠村火场、晏南枝出逃、沈照眠被押走和罗青禾遗言，确立护送契约。

## 场景表

1. 药屋地窖：月白玉片暴露。
2. 村口火线：白翳定罪。
3. 药屋火场：沈维桑救出晏南枝。
4. 押车路口：沈照眠被带走。
5. 后山坡：罗青禾交付驿令。

## 成片剧本

### SC01 罗青禾药屋地窖 / 傍晚 / 5秒钩子

地窖门被钩棍撬开。火光灌入暗格，晏南枝怀里的月白玉片亮出日月纹。

封门奴兵愣住，立刻吹响气味牌。

封门奴兵：昭明旧脉的日月纹！

地窖上方传来押车铜环声，沈照眠的声音被火噼啪声压碎。

沈照眠：哥！

晏南枝抬手遮住玉片，手指仍在发抖。罗青禾从药柜旁扑来，一把药粉撒进奴兵眼里。

罗青禾：跑。

### SC02 残阳坳村口 / 火起

白翳站在井旁，白册官吏把虫蜡封条贴上村门。奴兵推来押车，车上已经有几个被绑的村民。

村民：我们只是山民！

白翳：山民两个字，遮不住逆案根脚。清明籍清户，不问你们喊什么名。

他话音落下，虫族小兵点燃药屋侧墙。火先咬住干草，再卷上木梁。

沈维桑被两个奴兵按住。他看到沈照眠被推向木车，眼神变了。

沈维桑：照眠！

沈照眠：哥！

奴兵的扣环锁住她的腕。沈维桑肩膀猛地一沉，用左肩旧伤硬撞开一人，短刀割断绳结。

白翳没有追。他只是抬起虫蜡针。

白翳：他会去火里。旧血总以为自己能同时救所有人。

他把一枚白蜡转运签丢给押车奴兵。

白翳：第二辆，天亮前入外署。

### SC03 药屋火场 / 连续

浓烟压低。沈维桑用湿布捂住口鼻，从后窗撞进药屋。他先踢翻水缸压住火舌，再顺着药柜暗号摸到地窖口。

晏南枝被烟呛得站不稳，仍把血牒包往怀里塞。

沈维桑：血牒救不了你现在这一口气。要死，出去再挑死法。

晏南枝：这是血牒。

沈维桑：我知道。外面的人会为了它杀你。

他把她从暗格拖出。火梁坠下，沈维桑用猎弓撑住半瞬，左肩疼得发僵。晏南枝第一次看清他的伤。

晏南枝：你撑不住。

沈维桑：看路，不看我。

两人从药屋后墙滚出。沈维桑点燃早埋的烟草包，浓烟沿风口扑向村口。

### SC04 押车路口 / 夜色

沈维桑和晏南枝贴着柴垛潜行。押车已经动了。沈照眠被缚网绑在第二辆车上，烧焦发绳垂在脸侧。

沈维桑冲出去半步，被罗青禾从阴影里一把拽住。

沈维桑：娘，她在车上。

罗青禾嘴角有血，青布头巾烧开半边，却仍稳稳按住他的手。

罗青禾：你现在去，只会多一辆车。

沈维桑：那我看着？

罗青禾看向押车远去，眼里疼到发空。

罗青禾：记住车轮声。记住清明香。活着的人才有路追。

她抓起一把湿泥，在沈维桑掌心划出三道短痕一道长痕。

罗青禾：三短一长，是外署急车。过金河前，还有一次换马。

晏南枝低声说。

晏南枝：清明院会把她送去识别署。

晏南枝：外署先换她的名，再让她背旧歌。孩子一开口，家里的根脚就露在白册上。

沈维桑猛地看她。

沈维桑：你早知道他们抓孩子？

晏南枝沉默。这个沉默比承认更重。

### SC05 后山坡 / 火后

三人退到后山坡。残阳坳在身后烧成红黑一片。罗青禾跪坐在草丛里，手里握着半枚日月驿令。

她把驿令塞进沈维桑掌心。驿令冰冷，边缘有旧血和草药灰。

罗青禾：送她去月下。

沈维桑：我不送皇族。

罗青禾：不是送她去复国。

沈维桑看着她。

罗青禾：你不是替她守旧旗，是替残阳坳守一条活路。先拿到路，再去把照眠从白册上换回来。

晏南枝脸色微变。

沈维桑：那照眠呢？

罗青禾抬手摸他的脸，指腹全是药草染色。

罗青禾：找她。可你要先有一条路。

罗青禾：月饼能藏信，血牒也能藏祸。别看皮上的花，看里面夹的命。

她把半枚驿令按进他掌心，指向北坡一条被草盖住的旧水沟。

罗青禾：从这里走，别从山脊走。白翳会等你上高处。

远处，白翳站在火边，看向后山坡方向。他没有看见三人，只闻到一缕被烟故意带偏的清明香。

白翳：猎户会藏风。那就查风走过的旧路。

后山坡上，罗青禾最后看向火场，又看向晏南枝。

罗青禾：别让他为你的旗死。

她的手落下。沈维桑握紧驿令，没有哭出声。

晏南枝摘下月白玉片，低声说。

晏南枝：到月下，我先查押车名册。

沈维桑抬眼。

沈维桑：这不是许诺，是还债。路我带，你听我的。

火光吞没残阳坳。两人转身入山。

切黑。
