# 第02集《火烧残阳坳》Script v01

时长：约3分钟  
形式：竖屏短剧

## 1. 外景｜残阳坳村屋间｜夜色初起｜0:00-0:18

【0:00-0:05｜前5秒钩子】

白翳站在药屋台阶上，白袍干净，脚边清明香贴地游走。

他抬手，声音轻得像登记名字。

白翳：还村清明。

话音落下。

虫族火把同时脱手，越过竖屏上沿，砸向两侧村屋。

火光炸开。土墙被照红，窗纸瞬间卷成黑边。

【0:05-0:18】

近景：一只饭碗滚进火里。门闩被短矛挑断。有人伸手去抱孩子，画外传来螳刃擦墙的尖响。

沈维桑猛地把沈照眠按到水缸后。

沈照眠：哥……

沈维桑：低头。

烟从地面压过来。白翳合上清明白册。

白翳：旧污不留屋。

## 2. 外景｜罗家后院旧井｜夜｜0:18-0:45

沈维桑拖着沈照眠往后院退。药屋方向传来药柜被撞翻的闷响。

罗青禾隔着烟看他，只做一个手势：井。

沈维桑咬住牙，把沈照眠推向柴垛后。

沈维桑：在这儿等我。

沈照眠抓住他的衣角。

沈照眠：娘呢？

沈维桑没答。他冲到旧井边，掀开井绳下的湿苔石。井壁里露出一道窄暗门。

里面一只手先伸出来，手上握着短刀。

刀尖抵住沈维桑喉侧。

晏南枝从暗门里露面，灰尘满脸，黑发用红线束着。她胸前挂着完整月白玉片，冷光贴着火色闪了一下。

晏南枝：谁让你开门？

沈维桑看她刀尖，又看她怀里的油纸。

沈维桑：想死就关回去。

晏南枝没松刀。

晏南枝：你姓沈？

沈维桑：现在问姓，太慢了。

画外，虫族短矛敲上井沿。

## 3. 内景｜井下暗道口｜夜｜0:45-1:10

沈维桑拉晏南枝出暗门。她踉跄半步，仍把刀横在两人之间。

油纸散开一角。镜头第一次清楚露出日月血牒：血色细纹在纸上压成残缺日月印。

沈维桑盯了一眼。

沈维桑：白翳找的是你。

晏南枝：也找开门的人。

沈维桑：那就都别让他找到。

她看向他左肩旧伤。

晏南枝：你会旧路？

沈维桑：我会活路。

井口上方，火把照进来。虫族影子爬过井壁。

沈维桑一把按灭井边清明香烟线，拽她往药屋后门跑。

晏南枝短刀没收，只压低声音。

晏南枝：别碰我的东西。

沈维桑：先别丢命。

## 4. 内景｜罗家药屋｜夜｜1:10-1:45

药屋里全是烟。药柜燃烧，青黛粉被火舌卷起，蓝黑烟雾呛得虫族追兵后退。

罗青禾把一只药臼砸向火堆，草药粉爆开，烟更浓。

罗青禾：从后门走！

沈维桑冲进来。

沈维桑：娘！照眠还在外面！

罗青禾抓住他的手，把半枚日月驿令塞进他掌心。铜玉混色，断面像被刀劈开，日月纹只剩一半。

沈维桑：这是什么？

罗青禾：路。

她看一眼晏南枝，目光不是敬畏，是托命。

罗青禾：别问她值不值。先让她活。

晏南枝收紧月白玉片，第一次没有反驳。

门外螳刃刺入，穿过罗青禾肩侧。她闷哼，仍把药柜推倒，挡住门缝。

沈维桑要冲过去。

罗青禾按住他手背。

罗青禾：你答应过，护住能护的。

沈维桑眼红。

沈维桑：我没答应这个。

罗青禾看着他，声音低到几乎被火吞掉。

罗青禾：送她去月下。

她的手滑落。半枚驿令在沈维桑掌心硌出血印。

## 5. 外景｜药屋侧巷｜夜｜1:45-2:10

沈维桑拖着晏南枝冲出侧门。烟后，沈照眠从柴垛边探头。

沈照眠：哥！

沈维桑转身。

一名白衣识别者从烟里出现，伸手按住沈照眠的腕。虫蜡环啪地扣上。

沈照眠挣扎，小草药袋掉在泥里，药名牌散开。

沈维桑：照眠！

他刚冲出一步，螳刃追兵从侧面劈来。晏南枝用短刀挡住，刀口崩出缺。

晏南枝：你过去，她也死！

远处车轮压过碎瓦。沈照眠被推向虫族木车，哭声被烟呛断。

沈维桑捡起泥里的小草药袋，手指发抖。

白衣识别者回头看他，像已经记住他的脸。

沈维桑搭箭，箭尖对准识别者。

晏南枝按下他的弓臂。

晏南枝：烟散了。

三名螳刃追兵逼近。

沈维桑把草药袋塞进怀里，声音哑。

沈维桑：走谷口。

## 6. 外景｜残阳坳谷口猎人陷阱｜夜｜2:10-2:42

谷口狭窄。滚木绳扣藏在烧焦草堆后。烟尘从村里追出来。

沈维桑伏地摸泥，判断风向。他把半枚驿令咬在齿间，双手搭弓。

晏南枝靠在石边，短刀横胸。

晏南枝：你不是武者？

沈维桑：半个都不算。

螳刃影子切开烟。沈维桑一箭射出，没有射人，射断绳扣。

滚木砸落，挡住两名追兵。第三名从侧面跃出，刃肢划开沈维桑左肩旧伤。

沈维桑痛到跪地。

晏南枝短刀刺进追兵腿节，刀被卡住。

沈维桑按住肩，眼角和肩胛忽然闪过一线金赤残纹。地面旧驿浅痕亮了半息，指向塌石后的窄缝。

他闷哼，鼻血滴到灰里。

晏南枝：你会发光？

沈维桑：会疼。

白翳的声音从远处烟后传来。

白翳：停。

金赤残纹瞬间灭掉。沈维桑知道暴露了。

他强行提气，一拳砸在滚木尾端。明劲半成品只震开半尺，却足够让晏南枝拔出短刀。

沈维桑左臂垂下。

沈维桑：钻过去。

晏南枝看他伤口。

晏南枝：你撑不住。

沈维桑：所以快。

## 7. 外景｜谷口外坡道｜夜｜2:42-2:52

两人跌出窄缝。晏南枝第一反应是独自往暗林跑。

沈维桑一把按住流血的左肩，靠在石上。

沈维桑：站住。

晏南枝回头，刀仍在手。

晏南枝：你娘死了。你妹妹没了。还送我？

沈维桑看着火光，眼底没有英雄气，只剩硬撑。

沈维桑：我答应了。

晏南枝沉默半秒，把短刀收低。

晏南枝：我不信你。

沈维桑：路上再说。

## 8. 外景｜残阳坳外山坡/车辙旁｜夜｜2:52-3:00

沈维桑回望残阳坳。火场只见屋梁塌落的近声、火光舔过井沿、清明香被热浪卷碎。

他低头，看见虫族木车车辙旁压着一截烧焦发绳。发绳上还挂着沈照眠常用的小草药结。

沈维桑捡起来，攥到掌心。

晏南枝看见，没有说话。

远处火场边，白翳停步。他闻到空气里一丝清明香残味，目光转向谷外黑林。

白翳翻开白册，落笔。

白翳：沈维桑。旧路有应。

螳刃追兵压低身形，转向山路。

黑场。

音效：车轮远去，发绳焦灰在掌心碎裂。
