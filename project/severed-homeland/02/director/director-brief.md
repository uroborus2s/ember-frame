# 第02集《火烧残阳坳》导演简报

## 生产承诺

本集成片必须让观众在三分钟内感到：清明制度不是抽象压迫，而是能把一个有秩序的村庄、一个母亲、一对兄妹和一个尚未命名的护送契约同时烧断的执行机器。

本集不改剧情，不新增人物线。核心不是“主角觉醒打赢追兵”，而是沈维桑在救妹失败、母亲断后、晏南枝暴露、左肩撕裂后，被迫带着半枚驿令、小草药袋和未完成的承诺逃出残阳坳。

## 导演风格

- 形式：9:16 竖屏短剧，写实电影质感，低魔东方史诗奇幻。
- 摄影基调：极近景开钩，中近景逃亡，手部和道具特写承接信息，烟火遮挡压缩空间。
- 镜头重心：白册、虫蜡封条、火把、旧井、裂月玉片、油纸血牒、半枚驿令、小草药袋、虫蜡环、断箭、滚木绳结、烧焦发绳。
- 表演策略：沈维桑先判断路线和风向，情绪被动作压住；晏南枝冷、紧、藏；罗青禾用最少动作完成托付；白翳始终平稳干净，不演成狂热屠夫。
- 视觉克制：不拍大规模屠村全景，不拍完整虫族怪物潮，不把灵翼残纹拍成战斗开挂，不提前点亮石门。

## 情绪曲线

1. `0:00-0:18` 制度落火：白册封条和「旧污窝藏」先于火焰出现，观众必须明白屠村来自清明院命令。
2. `0:18-0:43` 秘密出井：沈维桑在妹妹和井下人之间做第一轮危险选择，晏南枝以刀和月白反光进入。
3. `0:43-1:08` 临时同逃：两人还不信任，但烟、井绳、后门路线把他们压进同一逃亡动线。
4. `1:08-1:43` 母亲托付：药屋是本集情感核心，罗青禾把护送定义为“去月下”，不是复国效忠。
5. `1:43-2:10` 救妹失败：沈照眠被白衣童役识别，奴兵扣腕押车，螳刃切断沈维桑的箭，护送承诺和寻妹目标同时成立。
6. `2:10-2:42` 谷口低魔突围：胜负来自猎户陷阱、地形和有限明劲代价，不来自神力爆发。
7. `2:42-2:53` 路线分歧：南逃避香与北追车辙形成第03集接口，情绪保持短句、克制、互相试探。
8. `2:53-3:00` 余烬钩子：烧焦发绳、血灰驿令和北坡黑石缝给出方向感，但必须明确没有石门亮起。

## 镜头语言

- 开场：极近景拍白册页角、虫蜡封条、木屑和蜡内细足；火把从竖屏上沿坠落，制造竖屏压顶感。
- 火场：用饭碗、门闩、窗纸、木梁、手脚、烟和声音表现村庄被烧，不使用大范围航拍式灾难全景。
- 旧井：先建立水缸、柴垛、水沟口、井沿和药屋后门的方位；裂月玉片只从井壁湿光中短暂露出。
- 药屋：蓝黑烟、药柜抽屉、青黛粉、白芷木牌和青布头巾承担空间记忆。罗青禾递驿令时用掌心特写，不用长台词解释设定。
- 木车：空间必须连续清楚：水沟口 -> 木车旁白衣童役和两名奴兵 -> 虫蜡环扣腕/押车绳拖人 -> 螳刃断箭钉板 -> 车辙北偏。
- 谷口：拆成摸泥、兽蹄磨痕、旧麻绳、射断绳扣、滚木偏移、腿节缝隙、左肩伤口和鼻血。动作要碎而可读。
- 结尾：火场回望只给旧井、药屋梁木、青布头巾、白册纸灰和车辙；半枚驿令只微热一瞬，黑石缝只吹冷风。

## 连续性优先级

1. 沈维桑：17岁少年猎户，灰褐猎衣、草药青内襦、旧皮护腕、猎弓、短刀；本集后衣角可焦黑但衣服仍完整。左肩撕裂、鼻血、左臂失力必须带入下游镜头。
2. 晏南枝：雾紫或冷灰流亡遮蔽装，红线束发，旧帝国标识和血牒全部隐蔽。裂月形玉片和油纸血牒只能短暂暴露，不做身份说明镜头。
3. 罗青禾：37岁村中医者，青布头巾、草药绿围裙、药柜和半枚驿令是核心锚点。临终仍保持医者和母亲的尊严，不拍成脏乱受害奇观。
4. 沈照眠：儿童比例、小草药袋、药名牌、烧焦发绳、虫蜡识别环。被带走时恐惧要清楚，但不延长成折磨展示。
5. 白翳：无尘白袍、高筒官帽、白色面甲、暗色复眼、清明白册、虫蜡针和清明香；火场中白袍不沾灰是反派视觉压迫点。
6. 虫族追兵与奴兵：少量非人形螳刃个体负责外圈封锁、切箭和追击，用短矛、刃肢、甲壳腿、影子和动作结果表现；肃明奴兵负责扣腕、押车和封门，保留旧麻军袍、虫蜡白绑带、耳后细甲鳞和人族直立轮廓。二者不要混成同一种敌人。
7. 残阳坳空间：药屋前 -> 后院水缸/柴垛/水沟口 -> 旧井 -> 药屋内 -> 侧巷木车 -> 谷口陷阱 -> 外坡车辙。shot list 必须先锁方位。
8. 半枚日月驿令：药屋交付，结尾被血灰蹭亮并微热指向北坡黑石缝；不得提前开启石门。
9. 灵翼残纹：只在左肩被划开后眼角和肩胛金赤短闪，表现痛感和伏笔，不提供找路或战斗胜利。
10. 明劲：只震松湿绳一指，代价立刻可见；不能拍成冲击波、击飞或完整武学爆发。

## 下游生产风险

- 火场风险：本集火烟镜头多，scene breakdown 应拆成可控局部空间，使用背光、烟层、燃烧道具和音效替代大规模村庄焚烧。
- 空间风险：沈照眠从水缸后到柴垛/水沟口，再到木车旁，是最容易断裂的调度链，storyboard 必须优先画清。
- 虫族/奴兵风险：螳刃追兵需要稳定非人形轮廓，但镜头可多用局部；奴兵必须稳定为混血执行者，不能画成普通人类兵，也不能画成完整虫怪。
- 动作风险：滚木陷阱和明劲不要连续长镜头硬拍，应拆成因果明确的特写组。
- 信息密度风险：裂月玉片、血牒、驿令、识句虫蜡痕、发绳都在本集出现，prompt draft 必须保证每个镜头只承担一个主信息。
- 第03集接口风险：结尾只能给黑石缝、冷风和微热方向感，不能表现石门启动、光纹铺路或旧驿机关开启。

## 推荐下游顺序

1. 先锁第02集关键角色/道具参考：火后沈维桑、流亡晏南枝、罗青禾火场状态、沈照眠虫蜡环、白翳火场白袍、清明院白衣童役、肃明奴兵、半枚日月驿令、小草药袋、烧焦发绳。
2. scene-breakdown-agent 先按八个剧本场景保留原时码，不合并药屋托付、木车掳人和谷口突围。
3. visual-continuity-agent 优先建立残阳坳火场方位图、人物服装损耗规则、低魔效果规则和第03集伤势延续项。
4. shot-planner-agent 将火场、井口、药屋、木车、谷口和结尾拆成短镜头，避免单镜头承载多重设定。
5. prompt-room 后续再把 shot prompt draft 转成 ComfyUI-ready 提示词；本阶段只保留导演意图、可拍动作和视觉锚点。
