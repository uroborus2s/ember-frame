# 第02集《火烧残阳坳》摄影机方案

## 总体摄影规则

- 画幅：9:16 竖屏短剧，所有构图优先服务“压顶、逼近、无处横逃”的竖向压迫。
- 默认尺度：极近景、近景、中近景为主；中远景只用于谷口、外坡和火场回望的空间确认，不拍大规模屠村全景。
- 默认镜头语言：制度道具极稳，逃亡人物轻微手持，动作因果用短促特写组拆清。白翳镜头稳定、洁净、低运动；沈维桑镜头跟随视线、手和地面痕迹。
- 默认镜头运动：制度线用静止、慢推、微距推近；逃亡线用贴墙跟拍、短促横移、烟中遮挡；谷口动作允许手持撞击，但每次撞击后必须回到可读空间。
- 镜头质感：写实电影感，低魔克制。文字、纹路、虫蜡环、断箭、绳扣、发绳、驿令等信息道具必须读得清，不靠炫光解释。
- 景深：人物对峙用轻压缩中长焦感压近背景烟火；道具特写用浅景深但关键字和纹路必须对焦；谷口陷阱用较深景深保证绳扣、滚木、石槽关系可读。
- 表演摄影：沈维桑先看风向、退路、车辙和地痕，再反应情绪；晏南枝站姿挺直但动作紧，刀始终控制距离；罗青禾镜头尊严克制；沈照眠恐惧清楚但不延长折磨。
- 禁忌：不拍完整虫族怪物潮，不拍石门亮起，不拍灵翼残纹提供战斗胜利，不拍明劲冲击波，不把火场拍成奇观灾难片。

## 光线与色彩规则

- 制度白：白翳、清明白册、虫蜡封条、白衣童役使用虫蜡白、骨白、珍珠灰和冷金小点。白色必须洁净、冷、干，不被火场染脏。
- 火场暖：村屋、饭碗、窗纸、梁木、侧巷和外坡火光为低饱和橙红，主要从侧后方或画面上沿压入，不做明亮英雄火光。
- 药屋蓝黑烟：罗家药屋内部以蓝黑烟、药柜暗木、草药绿围裙、青布头巾和火堆暗红组成；青黛粉入火后烟色更重，但不遮掉驿令交付。
- 井口月白：旧井与晏南枝入场只给一线月白冷反光，裂月玉片边缘短促可见，不扩大成发光身份展示。
- 低魔金赤：沈维桑灵翼残纹只在左肩被划开后眼角和肩胛短闪，像伤口火针，立刻灭；半枚驿令结尾只被血灰蹭亮一瞬，随后暗下。
- 清明香：贴地淡灰冷白烟线，低、细、慢；可以被泥灰按灭，可以被风吹散，不表现为可视化追踪界面。
- 结尾色阶：远处火场暗红、湿灰低黑、驿令一瞬铜玉微亮、黑石缝冷风偏青灰。最后黑场保留木车声和发绳攥紧声。

## 空间与屏幕方向规则

残阳坳第02集空间必须连续：

```text
药屋前 / 白翳台阶
        |
后院水缸 -- 柴垛 / 水沟口 -- 旧井 / 暗门 -- 药屋后门
        |
药屋内 / 药柜断后
        |
侧巷木车 / 北偏车辙
        |
谷口猎人陷阱 / 北坡石根兽道
        |
谷口外坡道 / 南侧黑林 vs 北偏车辙 / 北坡黑石缝
```

- 建立后统一屏幕方向：北坡、北偏车辙、木车离开方向、黑石缝固定为画面右侧或右上；南侧黑林固定为画面左侧。
- 残阳坳火场、旧井和药屋作为身后或下方背景，逃亡者从火场压迫中向外坡移动。
- 白翳与追兵从村庄火场方向压来，不能和木车北偏方向混淆。
- 沈照眠调度链必须连贯：水缸后 -> 柴垛/水沟口 -> 侧巷木车旁 -> 木车北偏离开。
- 药屋内轴线：后门入画，药柜在中线，半扇门和追兵在门缝方向，罗青禾用药柜挡住沈维桑回身路。
- 谷口轴线：绳扣、滚木、石壁、石槽、兽道窄缝先建立，再发生射绳、滚木、肩伤、明劲、钻兽道。
- 第7、8场反切不得左右互换南北；路线分歧必须在画面中同时成立。

## 镜头组摄影设计

### SC001 制度落火：白册先于火焰

- `SC001-SH001`：极近景俯拍白册页角与虫蜡封条，机位像登记桌上的冷眼；微推到「旧污窝藏」，虫蜡细足只在浅景深边缘蠕动。
- `SC001-SH002`：从白册边缘略低角度看白翳指尖，先聚焦虫蜡封条，再拉到白色面甲和无尘白袍；火场暖光不污染白袍。
- `SC001-SH003`：低角度仰拍火把从竖屏上沿坠落，跟随快速下摇，制造压顶感；螳刃只露刃肢和手部。
- `SC001-SH004`：局部灾难蒙太奇，地面饭碗、门闩、窗纸、手脚、倒梁硬切；手持轻震但不乱，避免村庄大全景。
- `SC001-SH005`：水缸后眼平中近景，沈维桑把妹妹压低后短横移到地面清明香；远处白翳在台阶上保持稳定竖直。

### SC002 秘密出井：水缸、柴垛、旧井三点锁位

- `SC002-SH001`：隔烟中近景，罗青禾手势和沈维桑视线对接；镜头不追情绪，只追两个方向点：旧井、谷口。
- `SC002-SH002`：略高肩后机位，小横移从水缸揭示相邻柴垛和水沟口；小草药袋按回怀里的手部特写必须清楚。
- `SC002-SH003`：井沿低机位近景，井绳、湿苔石、手、暗门缝依次入焦；月白反光只是一线，不推成完整机关。
- `SC002-SH004`：喉侧高度紧双人镜，晏南枝跌出后刀口占前景，沈维桑侧脸被压住；裂月玉片只被火光扫亮边缘。
- `SC002-SH005`：沈维桑肩后反打，刀尖、井沿追兵影子、晏南枝压油纸三层同框；焦点从刀尖拉到短矛影。
- `SC002-SH006`：胸前微距锁定油纸边角，血色细纹和残缺日月印一闪即遮；短矛敲石声作为剪点。

### SC003 临时同逃：物理遮蔽，不靠神秘能力

- `SC003-SH001`：贴地低近景，清明香烟线爬到井边，沈维桑手掌猛按入泥灰；镜头跟手，不表现法术反制。
- `SC003-SH002`：被暗门和墙根压缩的紧双人镜，短刀横在两人之间；微手持漂移表现同逃但未信任。
- `SC003-SH003`：手部近景，晏南枝收紧红线并压深油纸；静止构图，禁止身份解释式推镜。
- `SC003-SH004`：井沿低角度特写组，湿苔石复位、井绳进火星、烟灰上涌三切完成入口遮蔽。
- `SC003-SH005`：墙根跟拍，机位低于人物腰线，沿厚烟贴墙横移到药屋后门；沈维桑抹灰动作必须可见但不丑化衣装。

### SC004 母亲托付：药屋是情感核心

- `SC004-SH001`：从后门向内竖屏中景慢推，蓝黑烟、青黛粉、药柜和罗青禾同时建立；追兵只用后退甲壳局部。
- `SC004-SH002`：白芷木牌裂开到罗青禾肩伤的拉焦，强调药屋身份被火破坏；肩伤不做猎奇特写。
- `SC004-SH003`：药柜残骸上方手持急扫，沈维桑第一眼找妹妹的位置，再落到晏南枝守门阴影。
- `SC004-SH004`：掌心俯拍极近景，罗青禾把半枚驿令塞入沈维桑手中；铜玉断面、半圈日月纹、掌心出血同框。
- `SC004-SH005`：烟中压缩反打，罗青禾看晏南枝再看沈维桑；对白清楚，镜头不抬成复国宣言。
- `SC004-SH006`：地面低机位拍刃肢穿门和药柜倒下，柜体同时砸追兵前腿并封住沈维桑回身路。
- `SC004-SH007`：隔火紧反应双人镜，先推罗青禾口型“走”，再切沈维桑被火和倒柜阻隔；不展示尸体结果。

### SC005 救妹失败：木车、扣腕、断箭、北偏车辙

- `SC005-SH001`：侧门到水沟口横向跟拍，必须让药屋侧门、柴垛水沟口、木车方向在同一轴线上可读。
- `SC005-SH002`：木车轮边略低中近景，车板先遮挡再揭示白衣童役；白衣干净冷静，像制度延伸。
- `SC005-SH003`：临床式高角度特写组，手腕识句痕、药名牌、童役手依次硬切；不扩展童役个人情绪。
- `SC005-SH004`：俯拍虫蜡环扣腕和小草药袋落泥，撞击瞬间锁镜；沈照眠只给短促吸气和儿童手腕比例。
- `SC005-SH005`：沈维桑弓臂肩后机位追箭，箭被刃肢切断后硬切木车板；断箭离手臂一寸必须清楚，扣环未断。
- `SC005-SH006`：中近景动作压缩，妹妹、车板、追兵在一条线上解释无干净射线；晏南枝挡刃肢，刀口崩缺入画。
- `SC005-SH007`：地面低机位跟车轮压碎瓦，慢仰到沈维桑捡小草药袋和盯北偏车辙；车辙方向固定右上。

### SC006 谷口突围：陷阱因果胜过神力

- `SC006-SH001`：竖屏中景建立谷口窄道，低角度慢靠近烧焦草堆下的绳扣和湿麻绳；空间先于动作。
- `SC006-SH002`：贴地横爬镜头跟沈维桑指尖，泥、兽骚味、兽蹄磨痕、北坡窄缝、旧麻绳依次被看见。
- `SC006-SH003`：石边中近景静持，晏南枝崩缺短刀横胸，沈维桑仍看绳扣；对话压低，追兵影子只压进烟尘。
- `SC006-SH004`：弓臂肩后动作镜，箭道指向草堆后绳扣而非追兵身体；断绳特写接滚木坠落。
- `SC006-SH005`：滚木低机位撞击，先给腿节被压和第二名撞石，再手持甩到第三名突袭左肩；冲击后立刻可读伤口。
- `SC006-SH006`：极近景断裂感剪辑，眼角和肩胛金赤短闪如痛针，立刻灭；无光路、无反击效果。
- `SC006-SH007`：绳结、腿节、掌根、石槽低近景连切；明劲只用掌根闷响和湿绳松一指表现，不出现冲击波。
- `SC006-SH008`：从鼻血滴灰上仰到沈维桑垂下左臂和指向石根窄缝；白翳声音来自烟后远处，维持追捕方向。

### SC007 路线分歧：南避香，北追车辙

- `SC007-SH001`：兽道外机位看两人滚出，手持落定后向画面左侧南黑林小摇；风声吹散清明香，晏南枝先看左。
- `SC007-SH002`：沈维桑肩后，从怀中小草药袋拉焦到右上方北偏车辙；左肩血和左臂失力必须保留在前景。
- `SC007-SH003`：南林与北辙之间的静态紧双人镜，晏南枝收低短刀但不收掉戒备；刮石声逼近后切结尾。

### SC008 余烬钩子：发绳、驿令、黑石缝、白册

- `SC008-SH001`：外坡压缩长焦回望火场，旧井、药屋梁木、青布头巾燃起、白册纸灰分层进入；只用2秒，避免扩展灾难。
- `SC008-SH002`：湿灰车辙俯拍极近景，烧焦发绳和小草药结先静止，沈维桑手入画拾起；木车远去声延续北偏方向。
- `SC008-SH003`：掌心微距看血灰蹭亮半枚驿令，硬切右上方北坡黑石缝；石缝只冷风，无石门亮、无光纹铺路。
- `SC008-SH004`：远处火场边静态中景，白翳闻残味、翻白册、落笔；低角度但不英雄化，切黑后只留笔尖、木车和发绳音效。

## 资产与参考需求

- 角色参考优先级：火后沈维桑、流亡晏南枝、罗青禾药屋火场状态、沈照眠扣腕状态、白翳无尘白袍、清明院白衣童役、少量非人形螳刃追兵。
- 场景参考优先级：药屋前白册台阶、后院水缸/柴垛/水沟口/旧井、药屋蓝黑烟内景、侧巷木车、谷口滚木陷阱、外坡北辙与南林、北坡黑石缝。
- 道具参考优先级：清明白册、虫蜡封条、清明香、裂月形玉片、油纸血牒、半枚日月驿令、小草药袋、药名牌、虫蜡环、断箭、滚木绳扣、烧焦发绳、青布头巾。
- 低魔参考：清明香为贴地烟线，裂月玉片为冷反光，灵翼残纹为痛感短闪，驿令为一瞬血灰微亮，明劲为掌根闷响和湿绳微松。

## 风险备注

- 火烟风险：烟必须压缩空间，但不能吞掉关键道具。每个镜头只让一个主信息最清楚。
- 箭数风险：木车旁不要明确拍空箭囊；谷口仍需一箭射断绳扣。用遮挡和短切保留道具连续。
- 空间风险：沈照眠从水缸到柴垛/水沟口再到木车，是本集最易断裂调度链，Storyboard 必须优先画清。
- 低魔风险：灵翼残纹、半枚驿令、清明香和明劲都必须克制，不能变成战斗或导航特效。
- 虫族风险：螳刃追兵保持少量非人形轮廓，用甲壳腿、刃肢、短矛、影子和动作结果表现，不做怪物潮。
- 情感风险：罗青禾断后用药柜、口型、青布头巾和梁木表现，不拍尸体或受害奇观。
- 第03集接口风险：结尾只给北坡黑石缝冷风和驿令热意方向感，不提前点亮石门。
