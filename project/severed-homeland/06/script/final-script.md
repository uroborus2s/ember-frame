# 第06集：白墙验名

时长目标：4分40秒
格式：AI横屏宽屏剧详细剧本
本集功能：展示清明院外署的白册、虫蜡针和错册机制；让沈照眠以识别者身份现身但不团圆。

## 场景表

1. 外署白墙：虫蜡针验名钩子。
2. 登记厅队列：沈维桑和晏南枝混入待验人群。
3. 蜂巢记忆室门口：制造错册救孩子。
4. 后巷逃路：沈照眠出现。

## 成片剧本

### SC01 清明院外署白墙 / 白日 / 5秒钩子

无尘白墙前，虫蜡针刺入一名孩子手腕。白册自动浮出一个旧姓，孩子的母亲当场跪下。

白册官吏：针下应出旧姓。带进记忆室，翻祖账。

孩子：娘，我没说话。

奴兵扣住孩子。母亲扑上去，被纯虫族小兵用矛柄挡回。

白翳站在白墙阴影里，声音平静。

白翳：根脚藏在血里，像族谱压在箱底。清明院只是掀箱。

白翳：白墙要白得像莲，人跪下才信。可根扎在哪儿，白册知道。

母亲把孩子的袖子往下拽，想盖住针孔。

白翳：盖住针孔，盖不住根脚。

母亲：他只记得我叫娘。

白翳：白册会替他记祖上那一笔。

### SC02 外署登记厅 / 稍后

沈维桑和晏南枝混在待验人群里。沈维桑换了船工湿牌，晏南枝脸上抹着河泥，但姿态仍太直。

沈维桑压低声音。

沈维桑：别站得像等人开宫门。

晏南枝压低下颌：我已经站得很低了。

沈维桑：你低头也像在等人让路。

晏南枝瞪他一眼，终于学着逃荒妇人把肩缩下。

白册官吏查到他们前一排。一名少年被虫蜡针刺出旧驿半音，押车奴兵立刻上前。

晏南枝的手握紧短匕。

沈维桑：别动。

晏南枝：你每次都说别动。

沈维桑：因为你每次都要让人多死。

白翳忽然抬头。他闻到一丝被压过的旧驿香灰味。

白翳：封出口。有人带着灰烬书院的缺页。

### SC03 蜂巢记忆室门口 / 连续

人群骚乱。沈维桑把孟归藏给的转运签塞到登记桌下，又用半句旧驿暗语引诱白册官吏写错路线。

白册官吏低头核对。

白册官吏：转运签缺角。来源，灰烬书院。

沈维桑把湿粮牌压在白册页角，潮痕把“外署”二字洇开。

他低声念出半句旧驿暗语，诱得官吏补上“金河粮仓”。

沈维桑：金河船工送错的孩子，按粮仓错册走，不归外署收。

白册官吏迟疑。错册墨迹在白墙上浮动，旧姓被新墨盖住，只剩一条错路。

晏南枝扶住少年母亲，低声说。

晏南枝：低头，哭，不要谢。

母亲立刻抱住少年痛哭。奴兵被哭声和错册拖住半拍。

白翳走近，指尖擦过错册墨迹。

白翳：半句真话，又半条假路。

沈维桑拉起晏南枝冲向后巷。

### SC04 外署后巷 / 黄昏

后巷窄而白，墙面干净得没有影子。虫族小兵从两头堵来。

沈维桑用短刀割断晾布绳，白布落下挡住钩镰。他拉着晏南枝翻过矮墙，落到蜂巢记忆室外的廊口。

廊口尽头，一个白衣小女孩站在那里。腕上虫蜡识别环泛白，烧焦发绳被干净白袍压在领口。

沈维桑的呼吸停住。

沈维桑：照眠。

沈照眠抬头，眼神像隔着白册看他。

沈维桑没有喊第二声，只从袖里摸出一根干草乌。

沈照眠的眼睛颤了一下。

沈照眠：草乌，半夏，白芷，青禾。

沈维桑眼眶瞬间红了。

沈照眠：错一味，会死人。

沈维桑：是我教你的。

沈照眠没有喊哥哥。她像背条目一样继续。

沈照眠：听见旧驿暗语有反应。对象，成年男子，人族，左肩旧伤。

晏南枝拉住沈维桑。

晏南枝：她在识别你。

沈维桑往前一步。沈照眠的眼神短暂失焦，手指摸向领口烧焦发绳。

沈照眠很轻很轻地说：哥......

白翳的声音从她身后响起。

白翳：识别完成。

沈照眠的手放下，眼神重新变白。

白翳：沈维桑，旧驿余户。晏南枝，昭明血牒在身。封巷。

沈维桑被晏南枝强行拖走。他回头看沈照眠，沈照眠站在白墙下，没有追，也没有哭。

白翳俯身看她。

白翳：你认得他。

沈照眠机械地回答。

沈照眠：白册认得。

白翳露出极淡的满意。

切黑。
