# 第06集 Outline：白册里的妹妹

1. 钩子：白册出现沈照眠条目。目的：妹妹线实锤。情绪：刺痛。视觉：朱砂圈名。
2. 外署潜入：沈维桑翻墙进入白墙院，晏南枝跟随。目的：主动冒险。情绪：急切。视觉：白墙、清明香。
3. 记忆室：儿童被要求听禁歌、认暗语、看血纹。目的：制度暴力具象化。情绪：寒意。视觉：蜂巢孔、白衣儿童。
4. 档案争夺：沈维桑找到照眠转送记录，晏南枝找到月下盟书边墙线索。目的：双线推进。情绪：目标冲突。
5. 白翳设局：白翳播放残阳坳旧暗语，诱发沈维桑残纹。目的：反派智压。情绪：暴露危机。视觉：复眼纹、虫蜡针。
6. 焚物坑逃脱：晏南枝点燃废白册制造烟雾，沈维桑救下一个识别者儿童。目的：动作和道德选择。视觉：白纸飞灰。
7. 结尾转折：白翳写下活捉令，清明香路线指向边墙。目的：追捕升级。视觉：白册合拢。
