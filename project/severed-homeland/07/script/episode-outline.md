# 第07集 Outline：墙下无姓人

1. 钩子：陆青砾撕日月旗布质问晏南枝。目的：冲击复国叙事。情绪：尖锐。视觉：碎旗、窑洞火光。
2. 求牌：沈维桑用旧驿暗语找通行牌，被黑市压价。目的：进入边墙前置阻碍。情绪：焦躁。视觉：粮牌、木牌。
3. 出卖：陆青砾认出晏南枝月白玉片，把二人引到废粮仓。目的：现实生存压力。情绪：背叛。视觉：虫族巡吏脚步。
4. 回头：虫族巡吏要连混裔儿童一起抓，沈维桑冒险救人。陆青砾割断标记绳。目的：建立临时盟友。情绪：选择。视觉：窄刀、绳断。
5. 价值争吵：陆青砾问晏南枝复国后混裔算不算旧民，晏南枝答不上。目的：主题推进。情绪：沉默比争辩更重。
6. 入口交易：陆青砾给出锁喉关下旧驿入口，但要求同行。目的：导向第08集。视觉：黑市木牌背面的旧路图。
7. 结尾转折：远处边墙骨钟响起，北境兽潮预警。目的：进入雪夜守墙。视觉：骨钟、雪线。
