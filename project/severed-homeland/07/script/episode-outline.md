# 第07集 Episode Outline：墙下无姓人

## 结构节拍

### 1. 5秒钩子

陆青砾把沈维桑的旧驿暗号卖给两拨人，再回头问他愿出多少钱买自己的命。

### 2. 30秒危机

墙下集市同时出现清明院悬赏、灰墙军粮牌和混裔黑市追债。

### 3. 中段升级

晏南枝试图用皇血号令旧人族，被陆青砾质问日月旗里是否有混裔无姓人的姓。

### 4. 结尾悬念

陆青砾交出旧驿入口线索：入口在锁喉关粮门下，由薛临墙掌握。

## 分场方向

- 主要角色：沈维桑、晏南枝、陆青砾、白翳探子、黑市商
- 主要场景：墙下集市、粮牌交换点、灰布棚窄巷、窑洞暗门、雪线出口
- 动作要求：所有重大信息必须落在可见动作、道具、空间选择或对白冲突上。
- 节奏要求：开场立刻进入危险或问题，中段用追捕、交易、审讯、救援或空间坍塌升级，结尾必须保留下一集推动力。

## 角色推进

本集推进由“眼前求生”转向“故土定义被不断扩展”。沈维桑每次选择都优先保护活人和路线；晏南枝每次遭遇都削弱单一血统复国逻辑；白翳和清明院的压力必须持续把旧名、旧路和亲情变成可追捕证据。

## 连续性锚点

- 半枚日月驿令、血牒残段、清明香、白册、虫蜡针、旧驿暗语和月下多族印记均按出现顺序递进。
- 沈照眠第一季不得真正团圆，只能以识别者伤口和追捕线推动后续。
- 月下盟书不得写成皇族复国诏书。
