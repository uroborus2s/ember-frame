# 第07集：墙下无姓人

时长目标：4分25秒
格式：AI竖屏短剧详细剧本
本集功能：引入陆青砾，用混裔无姓人的现实正面挑战晏南枝的血统复国逻辑。

## 场景表

1. 墙下集市窄巷：陆青砾卖暗号。
2. 粮牌交换点：三方追债。
3. 窑洞暗门：日月旗是否有无姓人的姓。
4. 锁喉关粮门线索：入口代价。

## 成片剧本

### SC01 墙下集市窄巷 / 傍晚 / 5秒钩子

陆青砾把一张写着沈维桑旧驿暗号的碎陶片卖给白翳探子，又转身把同一句暗号卖给灰墙军粮牌贩。

她靠墙站，笑得像随时准备跑。

陆青砾：同一句话，两边都听得懂，说明它值两份钱。

陆青砾：月饼里夹过字，碎陶里也能夹命。

沈维桑从她身后按住碎陶护符。

沈维桑：把我的命卖了几份？

陆青砾回头，不慌不忙。

陆青砾：还没卖完。你要买回去吗？

### SC02 粮牌交换点 / 连续

墙下集市窄得像被黑石墙挤出来的缝。无籍者、军户家眷、混裔跑腿和黑市商在灰布棚下交换粮牌。

白翳探子亮出悬赏木牌。

白翳探子：旧驿猎户和南缘女眷，交人免查。

另一边，黑市商堵住陆青砾。

黑市商：你欠的不是钱，是进墙名额。

灰墙军粮牌贩也拔刀。

粮牌贩：薛老墙师要旧驿入口，不要惹清明院。

陆青砾看向沈维桑。

陆青砾：看见了吗？墙下没有敌我，只有谁今天饿。

晏南枝把月白玉片藏在袖内，仍忍不住开口。

晏南枝：你们若还认日月旗下那份旧盟，就不该把同族卖给清明院。

集市忽然安静。几个混裔孩子看向她，耳后细鳞痕在火光里发亮。

陆青砾笑意消失。

陆青砾：同族？日月旗下的户籍里，有过我们的姓吗？

晏南枝低头，看见混裔孩子把耳后鳞痕藏进破帽里。她把袖中的月白玉片又按回去，没有再举。

晏南枝：我刚才那句，欠你们。

陆青砾：欠话不抵粮。

### SC03 窑洞暗门 / 入夜

追兵逼近。陆青砾带二人钻入窑洞暗门，却把门闩握在自己手里。

陆青砾：我可以卖你们，也可以放你们。看价。

沈维桑：你要什么？

陆青砾指向晏南枝袖中的血牒残段。

陆青砾：她的正统。撕一点旧姓，换我们这些没户籍的人过一夜。

晏南枝冷下脸。

晏南枝：血牒不能给黑市当粮牌。

陆青砾：那我们的命就能给复国当垫脚石？

外面，白翳探子点燃清明香。香味沿门缝钻入。

沈维桑没有碰中间血纹，只沿油纸旧裂口撕下一角。

沈维桑：不是拿她的命换。只是撕一点旧姓的边，换这些黑户今晚不被查。

晏南枝：你疯了？

晏南枝盯着缺口，最后亲手把缺角压进陆青砾掌心。

晏南枝：只换今晚。谁敢拿它卖人，我亲自讨回来。

晏南枝：那月下石台若少这一角打不开呢？

沈维桑看她。

沈维桑：那就说明你的月下只认文书，不认活人。

晏南枝被击中，沉默。

陆青砾把血牒边角递给一个混裔少年，让他去换进墙名额。然后她打开暗门另一侧。

陆青砾：旧驿入口在锁喉关粮门下，薛临墙守着。想进去，先让他信你们不是又一面骗粮的旗。

### SC04 墙下雪线出口 / 夜

沈维桑、晏南枝和陆青砾从窄巷钻出，北边骨钟遥遥响起。

晏南枝忽然停下，对陆青砾说。

晏南枝：我不知道日月旗里有没有你的姓。

陆青砾挑眉。

晏南枝：但我知道，如果没有，我不能假装它照过天下。

陆青砾看她半晌。

陆青砾：漂亮话不值粮。

沈维桑：她已经少了一角血牒。

陆青砾掂着那一角血牒。

陆青砾：一角纸换十几个孩子一夜。明天呢？

晏南枝：明天我先去问粮门，不先问姓氏。

陆青砾又笑了，退到墙影里。

陆青砾：那我跟到锁喉关，听你怎么问。

远处，白翳探子捡到窑洞外残留的血牒纤维。

探子：大人，他们往粮门去了。

白翳接过纤维。

白翳：旧血开始缺口了。很好，缺口处最容易露真相。

切黑。
