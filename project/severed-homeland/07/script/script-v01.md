## 成片剧本

### SC01 墙下集市窄巷 / 傍晚 / 5秒钩子

陆青砾把一张写着沈维桑旧驿暗号的碎陶片卖给白翳探子，又转身把同一句暗号卖给灰墙军粮牌贩。

她靠墙站，笑得像随时准备跑。

陆青砾：同一句话，两边都听得懂，说明它值两份钱。

沈维桑从她身后按住碎陶护符。

沈维桑：把我的命卖了几份？

陆青砾回头，不慌不忙。

陆青砾：还没卖完。你要买回去吗？

### SC02 粮牌交换点 / 连续

墙下集市窄得像被黑石墙挤出来的缝。无籍者、军户家眷、混裔跑腿和黑市商在灰布棚下交换粮牌。

白翳探子亮出悬赏木牌。

白翳探子：旧驿猎户和南缘女眷，交人免查。

另一边，黑市商堵住陆青砾。

黑市商：你欠的不是钱，是进墙名额。

灰墙军粮牌贩也拔刀。

粮牌贩：薛老墙师要旧驿入口，不要惹清明院。

陆青砾看向沈维桑。

陆青砾：看见了吗？墙下没有敌我，只有谁今天饿。

晏南枝把月白玉片藏在袖内，仍忍不住开口。

晏南枝：旧人族若还认日月旗，就不该把同族卖给虫族。

集市忽然安静。几个混裔孩子看向她，耳后细鳞痕在火光里发亮。

陆青砾笑意消失。

陆青砾：同族？日月旗里有我们的姓吗？

晏南枝说不出话。

### SC03 窑洞暗门 / 入夜

追兵逼近。陆青砾带二人钻入窑洞暗门，却把门闩握在自己手里。

陆青砾：我可以卖你们，也可以放你们。看价。

沈维桑：你要什么？

陆青砾指向晏南枝袖中的血牒残段。

陆青砾：她的正统。拿一点换我的无姓人过夜。

晏南枝冷下脸。

晏南枝：血牒不能给黑市当粮牌。

陆青砾：那我们的命就能给复国当垫脚石？

外面，白翳探子点燃清明香。香味沿门缝钻入。

沈维桑夺过晏南枝手中的血牒油纸，撕下一段旧边角，塞给陆青砾。

晏南枝：你疯了？

沈维桑：完整血牒能救你的人。这个角能救门外十几个孩子今晚不被验名。

晏南枝：那月下石台若少这一角打不开呢？

沈维桑看她。

沈维桑：那就说明你的月下只认纸，不认人。

晏南枝被击中，沉默。

陆青砾把血牒边角递给一个混裔少年，让他去换进墙名额。然后她打开暗门另一侧。

陆青砾：旧驿入口在锁喉关粮门下。薛临墙看门。想进去，要先让他相信你们不是又一面来骗粮的旗。

### SC04 墙下雪线出口 / 夜

沈维桑、晏南枝和陆青砾从窄巷钻出，北边骨钟遥遥响起。

晏南枝忽然停下，对陆青砾说。

晏南枝：我不知道日月旗里有没有你的姓。

陆青砾挑眉。

晏南枝：但我知道，如果没有，我不能假装它照过天下。

陆青砾看她半晌。

陆青砾：漂亮话不值粮。

沈维桑：她已经少了一角血牒。

陆青砾又笑了，退到墙影里。

陆青砾：那我就跟到锁喉关，看你们还能少什么。

远处，白翳探子捡到窑洞外残留的血牒纤维。

探子：大人，他们往粮门去了。

白翳接过纤维。

白翳：旧血开始破损。很好，破损处最容易看见真形。

切黑。
